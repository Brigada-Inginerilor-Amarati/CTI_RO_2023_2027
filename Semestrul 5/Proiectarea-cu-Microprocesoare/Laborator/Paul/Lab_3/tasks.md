# Lab_3

1. Care este valoarea codului ASCII, exprimat în hexa și în zecimal, pentru caracterul
’a’? Dar pentru caracterul ’A’?

'A' = 41h = 65
'a' = 61h = 97

2. Explicați în cuvinte principiul prin care pot să transform o litera mică oarecare în litera
mare echivalentă. Cum pot să fac transformarea inversă, din literă mare în literă mică?

Diferenta intre litere mari si mici este de 32, deci putem adauga 32 la litera mica pentru a o transforma in litera mare.
Sau stochez diferenta dintre 'a' si 'A' intr-un registru.

3. Care este caracterul special care se scrie la finalul unui șir de caractere, cu rol de
terminator de șir?

'$' = 24h = 36

4. Ce înseamnă caracterele speciale CR și LF? Care este rolul acestora?

CR = Carriage Return
LF = Line Feed

CR = resetare cursor la inceputul liniei
LF = newline
