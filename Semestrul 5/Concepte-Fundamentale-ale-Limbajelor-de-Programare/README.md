# Concepte Fundamentale ale Limbajelor de Programare

This repository contains materials for the "Concepte Fundamentale ale Limbajelor de Programare" (Fundamental Concepts of Programming Languages) course. It includes lecture notes, laboratory exercises, and project implementations.

## Structure

*   **Curs**: Lecture slides and theoretical materials (PDFs).
*   **Laborator**: Weekly laboratory exercises and source code partitions.
*   **Proiect**: Course project implementations.
*   **Docs**: Additional documentation.
*   **Examen**: Exam-related resources.

## Topics Studied

*   **Basic C#**: Syntax, arrays, control flow.
*   **OOP**: Inheritance, interfaces, abstract classes, polymorphism.
*   **Advanced OOP**: Generics, collections.
*   **Delegates & Events**: Event-driven programming.
*   **Async Programming**: Task-based operations, async/await.
*   **Data Handling**: File I/O, data structures.
*   **LINQ**: Language Integrated Query (procedural and declarative).

## Technologies

*   **Language**: C#
*   **Framework**: .NET

### CLI Building & Running

The projects are standard .NET projects and can be managed via the CLI.

**Build:**
```bash
dotnet build <path/to/project.csproj>
```

**Run:**
```bash
dotnet run --project <path/to/project.csproj>
```

Example:
```bash
cd Laborator/Paul/Laborator_01
dotnet run
```
