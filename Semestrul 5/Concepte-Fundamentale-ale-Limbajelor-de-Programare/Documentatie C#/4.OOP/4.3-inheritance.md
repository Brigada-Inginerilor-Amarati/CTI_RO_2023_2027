# 4.3 Inheritance

## Description

**Inheritance** is a fundamental OOP concept that allows a class to inherit properties and methods from another class. It enables code reuse and establishes a hierarchical relationship between classes.

Key terminology:
- **Base class** (Parent/Super class): The class being inherited from
- **Derived class** (Child/Sub class): The class that inherits from the base class

In C#, a class can inherit from only one base class (single inheritance), but can implement multiple interfaces.

## Key Concepts

### Inheritance Syntax
```csharp
class DerivedClass : BaseClass
{
    // Additional members
}
```

### Access Modifiers in Inheritance
- `protected`: Accessible in the class and derived classes
- `private`: Not accessible in derived classes
- `public`: Accessible everywhere

### Method Overriding
- `virtual`: Allows a method to be overridden in derived classes
- `override`: Overrides a virtual method from the base class
- `base`: Keyword to access base class members

## Code Examples

### Example 1: Basic Inheritance

```csharp
using System;

// Base class
public class Animal
{
    public string Name { get; set; }
    public int Age { get; set; }

    public Animal(string name, int age)
    {
        Name = name;
        Age = age;
    }

    public virtual void MakeSound()
    {
        Console.WriteLine($"{Name} makes a sound");
    }

    public void DisplayInfo()
    {
        Console.WriteLine($"Name: {Name}, Age: {Age}");
    }
}

// Derived class
public class Dog : Animal
{
    public string Breed { get; set; }

    public Dog(string name, int age, string breed) : base(name, age)
    {
        Breed = breed;
    }

    public override void MakeSound()
    {
        Console.WriteLine($"{Name} barks: Woof! Woof!");
    }

    public void Fetch()
    {
        Console.WriteLine($"{Name} is fetching the ball");
    }
}

// Another derived class
public class Cat : Animal
{
    public bool IsIndoor { get; set; }

    public Cat(string name, int age, bool isIndoor) : base(name, age)
    {
        IsIndoor = isIndoor;
    }

    public override void MakeSound()
    {
        Console.WriteLine($"{Name} meows: Meow!");
    }

    public void Purr()
    {
        Console.WriteLine($"{Name} is purring");
    }
}

class Program
{
    static void Main()
    {
        Dog dog = new Dog("Buddy", 3, "Golden Retriever");
        dog.DisplayInfo();
        dog.MakeSound();
        dog.Fetch();

        Console.WriteLine();

        Cat cat = new Cat("Whiskers", 2, true);
        cat.DisplayInfo();
        cat.MakeSound();
        cat.Purr();
    }
}
```

**Output:**
```
Name: Buddy, Age: 3
Buddy barks: Woof! Woof!
Buddy is fetching the ball

Name: Whiskers, Age: 2
Whiskers meows: Meow!
Whiskers is purring
```

### Example 2: Using base Keyword

```csharp
using System;

public class Employee
{
    public string Name { get; set; }
    public int EmployeeId { get; set; }
    protected decimal BaseSalary { get; set; }

    public Employee(string name, int id, decimal baseSalary)
    {
        Name = name;
        EmployeeId = id;
        BaseSalary = baseSalary;
    }

    public virtual decimal CalculateSalary()
    {
        return BaseSalary;
    }

    public virtual void DisplayInfo()
    {
        Console.WriteLine($"Employee: {Name} (ID: {EmployeeId})");
        Console.WriteLine($"Salary: ${CalculateSalary():F2}");
    }
}

public class Manager : Employee
{
    public decimal Bonus { get; set; }
    public int TeamSize { get; set; }

    public Manager(string name, int id, decimal baseSalary, decimal bonus, int teamSize)
        : base(name, id, baseSalary)
    {
        Bonus = bonus;
        TeamSize = teamSize;
    }

    public override decimal CalculateSalary()
    {
        return base.CalculateSalary() + Bonus;
    }

    public override void DisplayInfo()
    {
        base.DisplayInfo();
        Console.WriteLine($"Team Size: {TeamSize}");
        Console.WriteLine($"Bonus: ${Bonus:F2}");
    }
}

public class Developer : Employee
{
    public string ProgrammingLanguage { get; set; }
    public int ProjectsCompleted { get; set; }

    public Developer(string name, int id, decimal baseSalary, string language)
        : base(name, id, baseSalary)
    {
        ProgrammingLanguage = language;
        ProjectsCompleted = 0;
    }

    public void CompleteProject()
    {
        ProjectsCompleted++;
        Console.WriteLine($"{Name} completed a project! Total: {ProjectsCompleted}");
    }

    public override void DisplayInfo()
    {
        base.DisplayInfo();
        Console.WriteLine($"Language: {ProgrammingLanguage}");
        Console.WriteLine($"Projects Completed: {ProjectsCompleted}");
    }
}

class Program
{
    static void Main()
    {
        Manager manager = new Manager("Alice Johnson", 101, 80000, 15000, 5);
        manager.DisplayInfo();

        Console.WriteLine();

        Developer dev = new Developer("Bob Smith", 201, 70000, "C#");
        dev.CompleteProject();
        dev.CompleteProject();
        dev.DisplayInfo();
    }
}
```

**Output:**
```
Employee: Alice Johnson (ID: 101)
Salary: $95000.00
Team Size: 5
Bonus: $15000.00

Bob Smith completed a project! Total: 1
Bob Smith completed a project! Total: 2
Employee: Bob Smith (ID: 201)
Salary: $70000.00
Language: C#
Projects Completed: 2
```

### Example 3: Multi-Level Inheritance

```csharp
using System;

// Base class
public class Vehicle
{
    public string Brand { get; set; }
    public int Year { get; set; }

    public Vehicle(string brand, int year)
    {
        Brand = brand;
        Year = year;
    }

    public virtual void Start()
    {
        Console.WriteLine($"{Brand} vehicle is starting...");
    }

    public void DisplayBasicInfo()
    {
        Console.WriteLine($"{Year} {Brand}");
    }
}

// Derived from Vehicle
public class MotorVehicle : Vehicle
{
    public int EngineSize { get; set; }
    public string FuelType { get; set; }

    public MotorVehicle(string brand, int year, int engineSize, string fuelType)
        : base(brand, year)
    {
        EngineSize = engineSize;
        FuelType = fuelType;
    }

    public override void Start()
    {
        base.Start();
        Console.WriteLine($"Engine: {EngineSize}cc, Fuel: {FuelType}");
    }

    public void Refuel()
    {
        Console.WriteLine($"Refueling with {FuelType}");
    }
}

// Derived from MotorVehicle
public class Car : MotorVehicle
{
    public int NumberOfDoors { get; set; }
    public string Type { get; set; }

    public Car(string brand, int year, int engineSize, string fuelType, int doors, string type)
        : base(brand, year, engineSize, fuelType)
    {
        NumberOfDoors = doors;
        Type = type;
    }

    public override void Start()
    {
        base.Start();
        Console.WriteLine($"Type: {Type}, Doors: {NumberOfDoors}");
    }

    public void OpenTrunk()
    {
        Console.WriteLine("Trunk is now open");
    }
}

// Another class derived from MotorVehicle
public class Motorcycle : MotorVehicle
{
    public bool HasSidecar { get; set; }

    public Motorcycle(string brand, int year, int engineSize, string fuelType, bool hasSidecar)
        : base(brand, year, engineSize, fuelType)
    {
        HasSidecar = hasSidecar;
    }

    public override void Start()
    {
        base.Start();
        Console.WriteLine($"Sidecar: {(HasSidecar ? "Yes" : "No")}");
    }

    public void Wheelie()
    {
        Console.WriteLine("Performing a wheelie!");
    }
}

class Program
{
    static void Main()
    {
        Car car = new Car("Toyota", 2022, 2000, "Gasoline", 4, "Sedan");
        car.Start();
        car.OpenTrunk();

        Console.WriteLine();

        Motorcycle bike = new Motorcycle("Harley-Davidson", 2021, 1200, "Gasoline", false);
        bike.Start();
        bike.Wheelie();
    }
}
```

**Output:**
```
Toyota vehicle is starting...
Engine: 2000cc, Fuel: Gasoline
Type: Sedan, Doors: 4
Trunk is now open

Harley-Davidson vehicle is starting...
Engine: 1200cc, Fuel: Gasoline
Sidecar: No
Performing a wheelie!
```

### Example 4: Abstract Classes

```csharp
using System;

// Abstract base class
public abstract class Shape
{
    public string Name { get; set; }

    public Shape(string name)
    {
        Name = name;
    }

    // Abstract methods must be implemented by derived classes
    public abstract double CalculateArea();
    public abstract double CalculatePerimeter();

    // Concrete method available to all derived classes
    public void DisplayInfo()
    {
        Console.WriteLine($"Shape: {Name}");
        Console.WriteLine($"Area: {CalculateArea():F2}");
        Console.WriteLine($"Perimeter: {CalculatePerimeter():F2}");
    }
}

public class Circle : Shape
{
    public double Radius { get; set; }

    public Circle(double radius) : base("Circle")
    {
        Radius = radius;
    }

    public override double CalculateArea()
    {
        return Math.PI * Radius * Radius;
    }

    public override double CalculatePerimeter()
    {
        return 2 * Math.PI * Radius;
    }
}

public class Rectangle : Shape
{
    public double Width { get; set; }
    public double Height { get; set; }

    public Rectangle(double width, double height) : base("Rectangle")
    {
        Width = width;
        Height = height;
    }

    public override double CalculateArea()
    {
        return Width * Height;
    }

    public override double CalculatePerimeter()
    {
        return 2 * (Width + Height);
    }
}

public class Triangle : Shape
{
    public double SideA { get; set; }
    public double SideB { get; set; }
    public double SideC { get; set; }

    public Triangle(double a, double b, double c) : base("Triangle")
    {
        SideA = a;
        SideB = b;
        SideC = c;
    }

    public override double CalculateArea()
    {
        // Using Heron's formula
        double s = (SideA + SideB + SideC) / 2;
        return Math.Sqrt(s * (s - SideA) * (s - SideB) * (s - SideC));
    }

    public override double CalculatePerimeter()
    {
        return SideA + SideB + SideC;
    }
}

class Program
{
    static void Main()
    {
        Shape[] shapes = new Shape[]
        {
            new Circle(5),
            new Rectangle(4, 6),
            new Triangle(3, 4, 5)
        };

        foreach (Shape shape in shapes)
        {
            shape.DisplayInfo();
            Console.WriteLine();
        }
    }
}
```

**Output:**
```
Shape: Circle
Area: 78.54
Perimeter: 31.42

Shape: Rectangle
Area: 24.00
Perimeter: 20.00

Shape: Triangle
Area: 6.00
Perimeter: 12.00
```

## Key Points

1. **Code Reusability**: Inheritance promotes code reuse by allowing derived classes to use base class functionality
2. **Method Overriding**: Use `virtual` and `override` keywords to modify base class behavior
3. **Constructor Chaining**: Use `: base()` to call base class constructors
4. **Protected Access**: Use `protected` for members that should be accessible to derived classes
5. **Abstract Classes**: Cannot be instantiated directly; serve as base classes with abstract members that must be implemented
6. **Single Inheritance**: C# supports single class inheritance but multiple interface implementation
