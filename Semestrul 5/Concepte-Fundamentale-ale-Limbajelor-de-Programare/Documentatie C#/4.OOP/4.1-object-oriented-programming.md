# 4.1 Object-Oriented Programming (OOP)

## Description

Object-Oriented Programming (OOP) is a programming paradigm based on the concept of "objects" that contain data and code. The main principles of OOP are:

- **Encapsulation**: Bundling data and methods that operate on that data within a single unit (class)
- **Abstraction**: Hiding complex implementation details and showing only essential features
- **Inheritance**: Creating new classes from existing ones, inheriting their properties and methods
- **Polymorphism**: Ability of objects to take on multiple forms

C# is a fully object-oriented language where everything is treated as an object, including primitive types.

## Key Concepts

### Classes and Objects
A class is a blueprint for creating objects. It defines the structure and behavior of objects.

### Access Modifiers
- `public`: Accessible from anywhere
- `private`: Accessible only within the class
- `protected`: Accessible within the class and derived classes
- `internal`: Accessible within the same assembly

## Code Examples

### Example 1: Basic Class Definition

```csharp
using System;

// Define a class
public class Person
{
    // Fields (private by default)
    private string name;
    private int age;

    // Constructor
    public Person(string name, int age)
    {
        this.name = name;
        this.age = age;
    }

    // Properties (encapsulation)
    public string Name
    {
        get { return name; }
        set { name = value; }
    }

    public int Age
    {
        get { return age; }
        set
        {
            if (value >= 0)
                age = value;
        }
    }

    // Method
    public void Introduce()
    {
        Console.WriteLine($"Hello, I'm {name} and I'm {age} years old.");
    }
}

class Program
{
    static void Main()
    {
        // Create an object
        Person person1 = new Person("Alice", 25);
        person1.Introduce();

        // Modify properties
        person1.Age = 26;
        Console.WriteLine($"Updated age: {person1.Age}");
    }
}
```

**Output:**
```
Hello, I'm Alice and I'm 25 years old.
Updated age: 26
```

### Example 2: Encapsulation with Auto-Properties

```csharp
using System;

public class BankAccount
{
    // Auto-implemented properties
    public string AccountNumber { get; private set; }
    public decimal Balance { get; private set; }

    public BankAccount(string accountNumber, decimal initialBalance)
    {
        AccountNumber = accountNumber;
        Balance = initialBalance;
    }

    public void Deposit(decimal amount)
    {
        if (amount > 0)
        {
            Balance += amount;
            Console.WriteLine($"Deposited ${amount}. New balance: ${Balance}");
        }
    }

    public void Withdraw(decimal amount)
    {
        if (amount > 0 && amount <= Balance)
        {
            Balance -= amount;
            Console.WriteLine($"Withdrew ${amount}. New balance: ${Balance}");
        }
        else
        {
            Console.WriteLine("Insufficient funds or invalid amount.");
        }
    }
}

class Program
{
    static void Main()
    {
        BankAccount account = new BankAccount("123456", 1000m);
        account.Deposit(500m);
        account.Withdraw(200m);
        account.Withdraw(2000m);
    }
}
```

**Output:**
```
Deposited $500. New balance: $1500
Withdrew $200. New balance: $1300
Insufficient funds or invalid amount.
```

### Example 3: Multiple Objects

```csharp
using System;
using System.Collections.Generic;

public class Student
{
    public string Name { get; set; }
    public int StudentId { get; set; }
    public List<int> Grades { get; set; }

    public Student(string name, int studentId)
    {
        Name = name;
        StudentId = studentId;
        Grades = new List<int>();
    }

    public void AddGrade(int grade)
    {
        Grades.Add(grade);
    }

    public double GetAverage()
    {
        if (Grades.Count == 0) return 0;
        int sum = 0;
        foreach (int grade in Grades)
        {
            sum += grade;
        }
        return (double)sum / Grades.Count;
    }

    public void DisplayInfo()
    {
        Console.WriteLine($"Student: {Name} (ID: {StudentId})");
        Console.WriteLine($"Average Grade: {GetAverage():F2}");
    }
}

class Program
{
    static void Main()
    {
        Student student1 = new Student("John Doe", 101);
        student1.AddGrade(85);
        student1.AddGrade(90);
        student1.AddGrade(78);
        student1.DisplayInfo();

        Console.WriteLine();

        Student student2 = new Student("Jane Smith", 102);
        student2.AddGrade(95);
        student2.AddGrade(88);
        student2.AddGrade(92);
        student2.DisplayInfo();
    }
}
```

**Output:**
```
Student: John Doe (ID: 101)
Average Grade: 84.33

Student: Jane Smith (ID: 102)
Average Grade: 91.67
```

## Benefits of OOP

1. **Modularity**: Code is organized into discrete objects
2. **Reusability**: Classes can be reused across different programs
3. **Maintainability**: Easier to modify and update code
4. **Scalability**: Easier to extend functionality
5. **Data Security**: Encapsulation protects data from unauthorized access
