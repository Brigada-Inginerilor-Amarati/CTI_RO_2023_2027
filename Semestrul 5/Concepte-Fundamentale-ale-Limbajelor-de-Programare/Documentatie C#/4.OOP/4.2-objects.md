# 4.2 Objects

## Description

An **object** is an instance of a class. It is a concrete entity created from a class blueprint that contains actual values and occupies memory. Objects have:

- **State**: Represented by fields/properties (data)
- **Behavior**: Represented by methods (functions)
- **Identity**: Each object has a unique identity in memory

In C#, objects are created using the `new` keyword and stored as reference types on the heap.

## Key Concepts

### Object Creation
Objects are instantiated using constructors, which initialize the object's state.

### Object Lifetime
- **Creation**: Using the `new` keyword
- **Usage**: Accessing members through the dot (`.`) operator
- **Destruction**: Handled by the Garbage Collector when no longer referenced

### Reference vs Value Types
- Reference types (classes): Store reference to memory location
- Value types (structs): Store actual data

## Code Examples

### Example 1: Creating and Using Objects

```csharp
using System;

public class Car
{
    public string Brand { get; set; }
    public string Model { get; set; }
    public int Year { get; set; }
    public double Mileage { get; private set; }

    public Car(string brand, string model, int year)
    {
        Brand = brand;
        Model = model;
        Year = year;
        Mileage = 0;
    }

    public void Drive(double distance)
    {
        Mileage += distance;
        Console.WriteLine($"{Brand} {Model} drove {distance} km. Total mileage: {Mileage} km");
    }

    public void DisplayInfo()
    {
        Console.WriteLine($"{Year} {Brand} {Model} - Mileage: {Mileage} km");
    }
}

class Program
{
    static void Main()
    {
        // Create objects
        Car car1 = new Car("Toyota", "Camry", 2020);
        Car car2 = new Car("Honda", "Civic", 2021);

        // Use objects
        car1.Drive(150);
        car1.Drive(75);
        car1.DisplayInfo();

        Console.WriteLine();

        car2.Drive(200);
        car2.DisplayInfo();
    }
}
```

**Output:**
```
Toyota Camry drove 150 km. Total mileage: 150 km
Toyota Camry drove 75 km. Total mileage: 225 km
2020 Toyota Camry - Mileage: 225 km

Honda Civic drove 200 km. Total mileage: 200 km
2021 Honda Civic - Mileage: 200 km
```

### Example 2: Object References

```csharp
using System;

public class Point
{
    public int X { get; set; }
    public int Y { get; set; }

    public Point(int x, int y)
    {
        X = x;
        Y = y;
    }

    public void Display()
    {
        Console.WriteLine($"Point({X}, {Y})");
    }
}

class Program
{
    static void Main()
    {
        // Create first object
        Point p1 = new Point(10, 20);
        Console.Write("p1: ");
        p1.Display();

        // p2 references the same object as p1
        Point p2 = p1;
        Console.Write("p2: ");
        p2.Display();

        // Modify through p2
        p2.X = 30;
        p2.Y = 40;

        // Both references show the change
        Console.WriteLine("\nAfter modifying through p2:");
        Console.Write("p1: ");
        p1.Display();
        Console.Write("p2: ");
        p2.Display();

        // Create a new independent object
        Point p3 = new Point(50, 60);
        Console.Write("\np3: ");
        p3.Display();
    }
}
```

**Output:**
```
p1: Point(10, 20)
p2: Point(10, 20)

After modifying through p2:
p1: Point(30, 40)
p2: Point(30, 40)

p3: Point(50, 60)
```

### Example 3: Objects as Method Parameters

```csharp
using System;

public class Rectangle
{
    public double Width { get; set; }
    public double Height { get; set; }

    public Rectangle(double width, double height)
    {
        Width = width;
        Height = height;
    }

    public double CalculateArea()
    {
        return Width * Height;
    }

    public double CalculatePerimeter()
    {
        return 2 * (Width + Height);
    }

    public void Display()
    {
        Console.WriteLine($"Rectangle: {Width} x {Height}");
        Console.WriteLine($"Area: {CalculateArea()}");
        Console.WriteLine($"Perimeter: {CalculatePerimeter()}");
    }
}

class Program
{
    static void ScaleRectangle(Rectangle rect, double factor)
    {
        rect.Width *= factor;
        rect.Height *= factor;
        Console.WriteLine($"Scaled by factor {factor}");
    }

    static Rectangle CreateSquare(double side)
    {
        return new Rectangle(side, side);
    }

    static void Main()
    {
        Rectangle rect1 = new Rectangle(5, 10);
        Console.WriteLine("Original rectangle:");
        rect1.Display();

        Console.WriteLine("\nScaling rectangle by 2:");
        ScaleRectangle(rect1, 2);
        rect1.Display();

        Console.WriteLine("\nCreating a square:");
        Rectangle square = CreateSquare(7);
        square.Display();
    }
}
```

**Output:**
```
Original rectangle:
Rectangle: 5 x 10
Area: 50
Perimeter: 30

Scaling rectangle by 2:
Scaled by factor 2
Rectangle: 10 x 20
Area: 200
Perimeter: 60

Creating a square:
Rectangle: 7 x 7
Area: 49
Perimeter: 28
```

### Example 4: Object Initialization

```csharp
using System;

public class Book
{
    public string Title { get; set; }
    public string Author { get; set; }
    public int Pages { get; set; }
    public double Price { get; set; }

    public Book() { }

    public Book(string title, string author)
    {
        Title = title;
        Author = author;
    }

    public void DisplayInfo()
    {
        Console.WriteLine($"'{Title}' by {Author}");
        Console.WriteLine($"Pages: {Pages}, Price: ${Price}");
    }
}

class Program
{
    static void Main()
    {
        // Object initializer syntax
        Book book1 = new Book
        {
            Title = "The C# Programming Language",
            Author = "Anders Hejlsberg",
            Pages = 800,
            Price = 49.99
        };

        // Constructor with object initializer
        Book book2 = new Book("Clean Code", "Robert Martin")
        {
            Pages = 464,
            Price = 39.99
        };

        // Traditional constructor
        Book book3 = new Book("Design Patterns", "Gang of Four");
        book3.Pages = 395;
        book3.Price = 54.99;

        book1.DisplayInfo();
        Console.WriteLine();
        book2.DisplayInfo();
        Console.WriteLine();
        book3.DisplayInfo();
    }
}
```

**Output:**
```
'The C# Programming Language' by Anders Hejlsberg
Pages: 800, Price: $49.99

'Clean Code' by Robert Martin
Pages: 464, Price: $39.99

'Design Patterns' by Gang of Four
Pages: 395, Price: $54.99
```

## Key Points

1. **Objects are instances of classes**: Each object has its own copy of instance fields
2. **Reference semantics**: Multiple variables can reference the same object
3. **Object lifetime**: Managed by the garbage collector
4. **Null references**: Object references can be null if not initialized
5. **Initialization**: Objects can be initialized using constructors and object initializers
