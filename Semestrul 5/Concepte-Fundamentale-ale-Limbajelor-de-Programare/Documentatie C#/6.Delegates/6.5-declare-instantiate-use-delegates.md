# 6.5 Declare, Instantiate and Use Delegates

## Description

This section provides a comprehensive guide to the complete workflow of working with delegates in C#: from declaration to instantiation to usage. Understanding this complete process is essential for effective delegate programming.

### The Complete Delegate Workflow:
1. **Declaration**: Define the delegate type with signature
2. **Instantiation**: Create delegate instances pointing to methods
3. **Usage**: Invoke delegates and pass them as parameters
4. **Management**: Add/remove methods, handle null cases

### Key Steps:
- **Declare**: `public delegate ReturnType DelegateName(Parameters);`
- **Instantiate**: `DelegateName instance = new DelegateName(MethodName);`
- **Use**: `instance(parameters);` or pass as parameter
- **Manage**: Add/remove methods, check for null

## Code Examples

### Example 1: Complete Delegate Workflow

```csharp
using System;

// Step 1: Declaration
public delegate int CalculatorDelegate(int x, int y);
public delegate void DisplayDelegate(string message);

class Program
{
    // Step 2: Methods that match delegate signatures
    static int Add(int x, int y)
    {
        return x + y;
    }

    static int Multiply(int x, int y)
    {
        return x * y;
    }

    static void DisplayResult(string message)
    {
        Console.WriteLine($"Result: {message}");
    }

    static void DisplayError(string message)
    {
        Console.WriteLine($"Error: {message}");
    }

    static void Main()
    {
        // Step 3: Instantiation
        CalculatorDelegate addDelegate = new CalculatorDelegate(Add);
        CalculatorDelegate multiplyDelegate = new CalculatorDelegate(Multiply);
        
        DisplayDelegate displayDelegate = new DisplayDelegate(DisplayResult);
        DisplayDelegate errorDelegate = new DisplayDelegate(DisplayError);

        // Step 4: Usage
        int a = 10, b = 5;

        // Direct invocation
        int sum = addDelegate(a, b);
        displayDelegate($"Addition: {sum}");

        int product = multiplyDelegate(a, b);
        displayDelegate($"Multiplication: {product}");

        // Pass delegates as parameters
        ProcessCalculation(a, b, addDelegate, displayDelegate);
        ProcessCalculation(a, b, multiplyDelegate, errorDelegate);
    }

    // Method that accepts delegates as parameters
    static void ProcessCalculation(int x, int y, CalculatorDelegate calc, DisplayDelegate display)
    {
        try
        {
            int result = calc(x, y);
            display($"Calculation result: {result}");
        }
        catch (Exception ex)
        {
            display($"Calculation failed: {ex.Message}");
        }
    }
}
```

**Output:**
```
Result: Addition: 15
Result: Multiplication: 50
Calculation result: 15
Error: Calculation result: 50
```

### Example 2: Different Instantiation Methods

```csharp
using System;

public delegate string StringProcessor(string input);

class Program
{
    static string ToUpper(string input)
    {
        return input.ToUpper();
    }

    static string AddPrefix(string input)
    {
        return "PREFIX: " + input;
    }

    static string Reverse(string input)
    {
        char[] chars = input.ToCharArray();
        Array.Reverse(chars);
        return new string(chars);
    }

    static void Main()
    {
        string text = "hello world";

        // Method 1: Using new keyword
        StringProcessor processor1 = new StringProcessor(ToUpper);
        Console.WriteLine($"Method 1: {processor1(text)}");

        // Method 2: Direct assignment (C# 2.0+)
        StringProcessor processor2 = ToUpper;
        Console.WriteLine($"Method 2: {processor2(text)}");

        // Method 3: Using anonymous method
        StringProcessor processor3 = delegate(string input)
        {
            return input.ToLower();
        };
        Console.WriteLine($"Method 3: {processor3(text)}");

        // Method 4: Using lambda expression (C# 3.0+)
        StringProcessor processor4 = (input) => input.Replace(" ", "_");
        Console.WriteLine($"Method 4: {processor4(text)}");

        Console.WriteLine("\n" + new string('-', 40));

        // Multicast delegates
        StringProcessor multiProcessor = ToUpper;
        multiProcessor += AddPrefix;
        multiProcessor += Reverse;

        Console.WriteLine("Multicast processing:");
        string result = multiProcessor(text);
        Console.WriteLine($"Final result: {result}");
    }
}
```

**Output:**
```
Method 1: HELLO WORLD
Method 2: HELLO WORLD
Method 3: hello world
Method 4: hello_world

----------------------------------------
Multicast processing:
Final result: DLROW OLLEH :XIFERP
```

### Example 3: Delegate as Class Members

```csharp
using System;

public delegate void ProgressHandler(int percentage, string status);
public delegate void CompletionHandler(bool success, string message);

class FileProcessor
{
    // Delegate as field
    public ProgressHandler OnProgress;
    public CompletionHandler OnComplete;

    // Delegate as property
    public ProgressHandler ProgressHandler { get; set; }

    public void ProcessFile(string fileName)
    {
        Console.WriteLine($"Processing file: {fileName}");

        // Simulate file processing with progress updates
        for (int i = 0; i <= 100; i += 20)
        {
            // Check if delegate is not null before invoking
            OnProgress?.Invoke(i, $"Processing... {i}%");
            System.Threading.Thread.Sleep(100); // Simulate work
        }

        // Simulate completion
        bool success = true;
        string message = "File processed successfully";

        // Invoke completion handler
        OnComplete?.Invoke(success, message);
    }

    public void ProcessWithCustomProgress(string fileName, ProgressHandler customProgress)
    {
        Console.WriteLine($"Processing with custom progress: {fileName}");
        
        // Use the passed delegate
        customProgress?.Invoke(0, "Starting...");
        customProgress?.Invoke(50, "Halfway done...");
        customProgress?.Invoke(100, "Complete!");
    }
}

class Program
{
    static void OnProgressUpdate(int percentage, string status)
    {
        Console.WriteLine($"  Progress: {percentage}% - {status}");
    }

    static void OnCompletion(bool success, string message)
    {
        if (success)
        {
            Console.WriteLine($"  ✓ {message}");
        }
        else
        {
            Console.WriteLine($"  ✗ {message}");
        }
    }

    static void CustomProgressHandler(int percentage, string status)
    {
        Console.WriteLine($"  [CUSTOM] {percentage}% - {status}");
    }

    static void Main()
    {
        FileProcessor processor = new FileProcessor();

        // Set up delegates
        processor.OnProgress = OnProgressUpdate;
        processor.OnComplete = OnCompletion;

        // Process file
        processor.ProcessFile("document.txt");

        Console.WriteLine("\n" + new string('-', 40));

        // Use custom progress handler
        processor.ProcessWithCustomProgress("image.jpg", CustomProgressHandler);

        Console.WriteLine("\n" + new string('-', 40));

        // Multiple progress handlers
        processor.OnProgress += CustomProgressHandler;
        processor.ProcessFile("video.mp4");
    }
}
```

### Example 4: Delegate Factory Pattern

```csharp
using System;

public delegate int MathOperation(int x, int y);

class DelegateFactory
{
    public static MathOperation CreateAddOperation()
    {
        return new MathOperation((x, y) => x + y);
    }

    public static MathOperation CreateMultiplyOperation()
    {
        return new MathOperation((x, y) => x * y);
    }

    public static MathOperation CreateCustomOperation(string operation)
    {
        switch (operation.ToLower())
        {
            case "add":
                return (x, y) => x + y;
            case "subtract":
                return (x, y) => x - y;
            case "multiply":
                return (x, y) => x * y;
            case "divide":
                return (x, y) => y != 0 ? x / y : 0;
            case "power":
                return (x, y) => (int)Math.Pow(x, y);
            default:
                return (x, y) => 0;
        }
    }

    public static MathOperation CreateCompositeOperation(params string[] operations)
    {
        MathOperation composite = null;
        
        foreach (string op in operations)
        {
            MathOperation currentOp = CreateCustomOperation(op);
            if (composite == null)
            {
                composite = currentOp;
            }
            else
            {
                composite += currentOp;
            }
        }
        
        return composite;
    }
}

class Program
{
    static void TestOperation(MathOperation operation, int x, int y, string name)
    {
        try
        {
            int result = operation(x, y);
            Console.WriteLine($"{name}({x}, {y}) = {result}");
        }
        catch (Exception ex)
        {
            Console.WriteLine($"{name}({x}, {y}) = Error: {ex.Message}");
        }
    }

    static void Main()
    {
        int a = 10, b = 3;

        Console.WriteLine("Factory-created operations:");
        
        // Create individual operations
        MathOperation addOp = DelegateFactory.CreateAddOperation();
        MathOperation multiplyOp = DelegateFactory.CreateMultiplyOperation();
        
        TestOperation(addOp, a, b, "Add");
        TestOperation(multiplyOp, a, b, "Multiply");

        Console.WriteLine("\n" + new string('-', 30));

        // Create custom operations
        string[] operations = { "add", "subtract", "multiply", "divide", "power" };
        
        foreach (string op in operations)
        {
            MathOperation customOp = DelegateFactory.CreateCustomOperation(op);
            TestOperation(customOp, a, b, op.ToUpper());
        }

        Console.WriteLine("\n" + new string('-', 30));

        // Create composite operation
        MathOperation compositeOp = DelegateFactory.CreateCompositeOperation("add", "multiply");
        Console.WriteLine("Composite operation (add + multiply):");
        TestOperation(compositeOp, a, b, "Composite");
    }
}
```

**Output:**
```
Factory-created operations:
Add(10, 3) = 13
Multiply(10, 3) = 30

------------------------------
ADD(10, 3) = 13
SUBTRACT(10, 3) = 7
MULTIPLY(10, 3) = 30
DIVIDE(10, 3) = 3
POWER(10, 3) = 1000

------------------------------
Composite operation (add + multiply):
Composite(10, 3) = 30
```

### Example 5: Delegate Lifecycle Management

```csharp
using System;
using System.Collections.Generic;

public delegate void EventHandler(string message);

class EventManager
{
    private List<EventHandler> handlers = new List<EventHandler>();

    public void Subscribe(EventHandler handler)
    {
        if (handler != null && !handlers.Contains(handler))
        {
            handlers.Add(handler);
            Console.WriteLine($"Subscribed: {handler.Method.Name}");
        }
    }

    public void Unsubscribe(EventHandler handler)
    {
        if (handlers.Contains(handler))
        {
            handlers.Remove(handler);
            Console.WriteLine($"Unsubscribed: {handler.Method.Name}");
        }
    }

    public void TriggerEvent(string message)
    {
        Console.WriteLine($"\nTriggering event: {message}");
        
        // Create a copy to avoid modification during iteration
        var handlersCopy = new List<EventHandler>(handlers);
        
        foreach (var handler in handlersCopy)
        {
            try
            {
                handler?.Invoke(message);
            }
            catch (Exception ex)
            {
                Console.WriteLine($"Error in {handler?.Method.Name}: {ex.Message}");
            }
        }
    }

    public int HandlerCount => handlers.Count;
}

class Program
{
    static void LogHandler(string message)
    {
        Console.WriteLine($"  [LOG] {message}");
    }

    static void EmailHandler(string message)
    {
        Console.WriteLine($"  [EMAIL] {message}");
    }

    static void ErrorHandler(string message)
    {
        Console.WriteLine($"  [ERROR] {message}");
        if (message.Contains("CRITICAL"))
        {
            throw new Exception("Critical error occurred");
        }
    }

    static void Main()
    {
        EventManager manager = new EventManager();

        Console.WriteLine("Delegate Lifecycle Management Demo");
        Console.WriteLine("=" * 40);

        // Subscribe handlers
        manager.Subscribe(LogHandler);
        manager.Subscribe(EmailHandler);
        manager.Subscribe(ErrorHandler);

        Console.WriteLine($"\nTotal handlers: {manager.HandlerCount}");

        // Trigger events
        manager.TriggerEvent("User logged in");
        manager.TriggerEvent("Data updated");
        manager.TriggerEvent("CRITICAL: System failure");

        Console.WriteLine($"\nTotal handlers after errors: {manager.HandlerCount}");

        // Unsubscribe one handler
        manager.Unsubscribe(EmailHandler);
        Console.WriteLine($"\nTotal handlers after unsubscribe: {manager.HandlerCount}");

        // Trigger another event
        manager.TriggerEvent("User logged out");

        // Subscribe the same handler again
        manager.Subscribe(EmailHandler);
        Console.WriteLine($"\nTotal handlers after resubscribe: {manager.HandlerCount}");

        // Test null handling
        EventHandler nullHandler = null;
        manager.Subscribe(nullHandler); // Should not add null
        Console.WriteLine($"Total handlers after null subscribe: {manager.HandlerCount}");

        // Test duplicate subscription
        manager.Subscribe(LogHandler); // Should not add duplicate
        Console.WriteLine($"Total handlers after duplicate subscribe: {manager.HandlerCount}");
    }
}
```

## Key Points

### Declaration
- **Syntax**: `public delegate ReturnType DelegateName(Parameters);`
- **Accessibility**: Can be public, private, protected, or internal
- **Signature**: Must match the methods you want to reference

### Instantiation
- **Constructor**: `new DelegateName(MethodName)`
- **Direct assignment**: `DelegateName instance = MethodName;`
- **Anonymous methods**: `DelegateName instance = delegate(parameters) { /* code */ };`
- **Lambda expressions**: `DelegateName instance = (parameters) => { /* code */ };`

### Usage
- **Invocation**: `delegateInstance(parameters);`
- **Null safety**: Always check for null before invoking
- **Parameters**: Pass as method parameters
- **Return values**: Handle return values appropriately

### Management
- **Multicast**: Use `+=` to add, `-=` to remove
- **Lifecycle**: Properly manage delegate references
- **Error handling**: Implement safe invocation patterns
- **Memory**: Remove delegates when no longer needed

### Best Practices
- **Null checks**: Always check for null before invocation
- **Error handling**: Implement try-catch for delegate invocation
- **Memory management**: Remove delegates to prevent memory leaks
- **Type safety**: Use strongly-typed delegates
- **Documentation**: Document delegate signatures and usage
