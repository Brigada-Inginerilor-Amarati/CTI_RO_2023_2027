# 6.1 Delegates

## Description

A **delegate** in C# is a type that represents references to methods with a particular parameter list and return type. Delegates are similar to function pointers in C and C++, but they are type-safe and secure.

### Key Characteristics:
- **Type-safe**: Compile-time checking ensures method signatures match
- **Object-oriented**: Delegates are objects that can be passed as parameters
- **Multicast capable**: Can hold references to multiple methods
- **Asynchronous**: Can be used for callbacks and event handling

### When to Use Delegates:
- **Callbacks**: Notify when an operation completes
- **Event handling**: Respond to user interactions or system events
- **Method parameters**: Pass methods as arguments to other methods
- **Plugin architecture**: Allow external code to extend functionality

## Code Examples

### Example 1: Basic Delegate Declaration and Usage

```csharp
using System;

// Declare a delegate type
public delegate int MathOperation(int x, int y);

class Program
{
    // Methods that match the delegate signature
    static int Add(int x, int y)
    {
        return x + y;
    }

    static int Multiply(int x, int y)
    {
        return x * y;
    }

    static int Subtract(int x, int y)
    {
        return x - y;
    }

    static void Main()
    {
        // Create delegate instances
        MathOperation addOp = new MathOperation(Add);
        MathOperation multiplyOp = new MathOperation(Multiply);
        MathOperation subtractOp = new MathOperation(Subtract);

        // Use delegates
        int result1 = addOp(10, 5);
        int result2 = multiplyOp(10, 5);
        int result3 = subtractOp(10, 5);

        Console.WriteLine($"10 + 5 = {result1}");
        Console.WriteLine($"10 * 5 = {result2}");
        Console.WriteLine($"10 - 5 = {result3}");
    }
}
```

**Output:**
```
10 + 5 = 15
10 * 5 = 50
10 - 5 = 5
```

### Example 2: Delegate as Method Parameter

```csharp
using System;

public delegate void DisplayMessage(string message);

class MessageProcessor
{
    public static void ProcessMessage(string message, DisplayMessage displayMethod)
    {
        // Process the message
        string processedMessage = message.ToUpper();
        
        // Use the delegate to display
        displayMethod(processedMessage);
    }
}

class Program
{
    static void DisplayToConsole(string message)
    {
        Console.WriteLine($"Console: {message}");
    }

    static void DisplayToFile(string message)
    {
        Console.WriteLine($"File: {message}");
    }

    static void Main()
    {
        string text = "Hello, World!";
        
        // Pass different display methods
        MessageProcessor.ProcessMessage(text, DisplayToConsole);
        MessageProcessor.ProcessMessage(text, DisplayToFile);
    }
}
```

**Output:**
```
Console: HELLO, WORLD!
File: HELLO, WORLD!
```

### Example 3: Built-in Delegate Types

```csharp
using System;

class Program
{
    static void Main()
    {
        // Using Action delegate (no return value)
        Action<string> printAction = (message) => Console.WriteLine($"Action: {message}");
        printAction("Hello from Action!");

        // Using Func delegate (with return value)
        Func<int, int, int> addFunc = (x, y) => x + y;
        int sum = addFunc(10, 20);
        Console.WriteLine($"Func result: {sum}");

        // Using Predicate delegate (returns bool)
        Predicate<int> isEven = (number) => number % 2 == 0;
        Console.WriteLine($"Is 15 even? {isEven(15)}");
        Console.WriteLine($"Is 16 even? {isEven(16)}");
    }
}
```

**Output:**
```
Action: Hello from Action!
Func result: 30
Is 15 even? False
Is 16 even? True
```

### Example 4: Delegate with Different Method Signatures

```csharp
using System;

// Delegate for methods with no parameters and no return value
public delegate void SimpleDelegate();

// Delegate for methods with one parameter
public delegate void ParameterDelegate(string message);

// Delegate for methods with return value
public delegate string ReturnDelegate();

class Program
{
    static void SayHello()
    {
        Console.WriteLine("Hello!");
    }

    static void SayGoodbye()
    {
        Console.WriteLine("Goodbye!");
    }

    static void Greet(string name)
    {
        Console.WriteLine($"Hello, {name}!");
    }

    static void Farewell(string name)
    {
        Console.WriteLine($"Goodbye, {name}!");
    }

    static string GetCurrentTime()
    {
        return DateTime.Now.ToString("HH:mm:ss");
    }

    static string GetCurrentDate()
    {
        return DateTime.Now.ToString("yyyy-MM-dd");
    }

    static void Main()
    {
        // Simple delegates
        SimpleDelegate simple1 = SayHello;
        SimpleDelegate simple2 = SayGoodbye;
        
        simple1();
        simple2();

        // Parameter delegates
        ParameterDelegate param1 = Greet;
        ParameterDelegate param2 = Farewell;
        
        param1("Alice");
        param2("Bob");

        // Return delegates
        ReturnDelegate return1 = GetCurrentTime;
        ReturnDelegate return2 = GetCurrentDate;
        
        Console.WriteLine($"Current time: {return1()}");
        Console.WriteLine($"Current date: {return2()}");
    }
}
```

**Output:**
```
Hello!
Goodbye!
Hello, Alice!
Goodbye, Bob!
Current time: 14:30:25
Current date: 2024-01-15
```

### Example 5: Delegate with Instance Methods

```csharp
using System;

public delegate void NotifyDelegate(string message);

class Calculator
{
    private string name;

    public Calculator(string name)
    {
        this.name = name;
    }

    public void OnCalculationComplete(string result)
    {
        Console.WriteLine($"[{name}] Calculation complete: {result}");
    }

    public void OnError(string error)
    {
        Console.WriteLine($"[{name}] Error: {error}");
    }

    public int Add(int a, int b, NotifyDelegate notify)
    {
        try
        {
            int result = a + b;
            notify($"Addition result: {result}");
            return result;
        }
        catch (Exception ex)
        {
            notify($"Error: {ex.Message}");
            return 0;
        }
    }
}

class Program
{
    static void Main()
    {
        Calculator calc1 = new Calculator("Calculator A");
        Calculator calc2 = new Calculator("Calculator B");

        // Use instance methods as delegates
        NotifyDelegate notify1 = calc1.OnCalculationComplete;
        NotifyDelegate notify2 = calc2.OnError;

        int result1 = calc1.Add(10, 5, notify1);
        int result2 = calc2.Add(10, 5, notify2);
    }
}
```

**Output:**
```
[Calculator A] Calculation complete: Addition result: 15
[Calculator B] Error: Addition result: 15
```

## Key Points

- **Type Safety**: Delegates provide compile-time type checking
- **Method Signatures**: Must match exactly (parameters and return type)
- **Instance vs Static**: Can reference both instance and static methods
- **Built-in Types**: `Action`, `Func`, and `Predicate` are commonly used
- **Flexibility**: Enable callback patterns and method passing
- **Foundation**: Basis for events and advanced programming patterns
