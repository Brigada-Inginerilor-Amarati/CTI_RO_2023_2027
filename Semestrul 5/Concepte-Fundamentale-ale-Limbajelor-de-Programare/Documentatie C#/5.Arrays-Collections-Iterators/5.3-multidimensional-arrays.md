# 5.3 Multidimensional Arrays

## Description

A **multidimensional array** is an array with more than one dimension. C# supports two types:

1. **Rectangular arrays**: All rows have the same number of columns (true multidimensional)
2. **Jagged arrays**: Arrays of arrays where each row can have different lengths (covered in 5.4)

Rectangular arrays store elements in a grid-like structure with fixed dimensions.

## Key Concepts

### Declaration Syntax
```csharp
// 2D array
type[,] arrayName = new type[rows, columns];

// 3D array
type[,,] arrayName = new type[dim1, dim2, dim3];

// N-dimensional
type[,,,] arrayName = new type[d1, d2, d3, d4];
```

### Memory Layout
Elements are stored in row-major order in contiguous memory.

### Properties
- `Length`: Total number of elements
- `Rank`: Number of dimensions
- `GetLength(dimension)`: Size of specific dimension

## Code Examples

### Example 1: 2D Array Basics

```csharp
using System;

class Program
{
    static void Main()
    {
        // Declaration and initialization - Method 1
        int[,] matrix1 = new int[3, 4]; // 3 rows, 4 columns

        // Populate with values
        int counter = 1;
        for (int i = 0; i < 3; i++)
        {
            for (int j = 0; j < 4; j++)
            {
                matrix1[i, j] = counter++;
            }
        }

        Console.WriteLine("Matrix 1 (3x4):");
        PrintMatrix(matrix1);

        // Declaration and initialization - Method 2
        int[,] matrix2 = {
            { 1, 2, 3 },
            { 4, 5, 6 },
            { 7, 8, 9 }
        };

        Console.WriteLine("\nMatrix 2 (3x3):");
        PrintMatrix(matrix2);

        // Array properties
        Console.WriteLine($"\nMatrix2 Properties:");
        Console.WriteLine($"Rank (dimensions): {matrix2.Rank}");
        Console.WriteLine($"Total Length: {matrix2.Length}");
        Console.WriteLine($"Rows: {matrix2.GetLength(0)}");
        Console.WriteLine($"Columns: {matrix2.GetLength(1)}");

        // Access individual elements
        Console.WriteLine($"\nElement at [1,2]: {matrix2[1, 2]}");
        Console.WriteLine($"Element at [0,0]: {matrix2[0, 0]}");
    }

    static void PrintMatrix(int[,] matrix)
    {
        int rows = matrix.GetLength(0);
        int cols = matrix.GetLength(1);

        for (int i = 0; i < rows; i++)
        {
            for (int j = 0; j < cols; j++)
            {
                Console.Write($"{matrix[i, j],4}");
            }
            Console.WriteLine();
        }
    }
}
```

### Example 2: Matrix Operations

```csharp
using System;

class Program
{
    static void Main()
    {
        int[,] matrixA = {
            { 1, 2, 3 },
            { 4, 5, 6 },
            { 7, 8, 9 }
        };

        int[,] matrixB = {
            { 9, 8, 7 },
            { 6, 5, 4 },
            { 3, 2, 1 }
        };

        Console.WriteLine("Matrix A:");
        PrintMatrix(matrixA);

        Console.WriteLine("\nMatrix B:");
        PrintMatrix(matrixB);

        // Matrix addition
        int[,] sum = AddMatrices(matrixA, matrixB);
        Console.WriteLine("\nMatrix A + B:");
        PrintMatrix(sum);

        // Matrix subtraction
        int[,] diff = SubtractMatrices(matrixA, matrixB);
        Console.WriteLine("\nMatrix A - B:");
        PrintMatrix(diff);

        // Scalar multiplication
        int[,] scaled = ScalarMultiply(matrixA, 2);
        Console.WriteLine("\nMatrix A * 2:");
        PrintMatrix(scaled);

        // Transpose
        int[,] transposed = Transpose(matrixA);
        Console.WriteLine("\nTranspose of Matrix A:");
        PrintMatrix(transposed);

        // Sum of all elements
        int totalSum = SumAllElements(matrixA);
        Console.WriteLine($"\nSum of all elements in Matrix A: {totalSum}");
    }

    static int[,] AddMatrices(int[,] a, int[,] b)
    {
        int rows = a.GetLength(0);
        int cols = a.GetLength(1);
        int[,] result = new int[rows, cols];

        for (int i = 0; i < rows; i++)
        {
            for (int j = 0; j < cols; j++)
            {
                result[i, j] = a[i, j] + b[i, j];
            }
        }
        return result;
    }

    static int[,] SubtractMatrices(int[,] a, int[,] b)
    {
        int rows = a.GetLength(0);
        int cols = a.GetLength(1);
        int[,] result = new int[rows, cols];

        for (int i = 0; i < rows; i++)
        {
            for (int j = 0; j < cols; j++)
            {
                result[i, j] = a[i, j] - b[i, j];
            }
        }
        return result;
    }

    static int[,] ScalarMultiply(int[,] matrix, int scalar)
    {
        int rows = matrix.GetLength(0);
        int cols = matrix.GetLength(1);
        int[,] result = new int[rows, cols];

        for (int i = 0; i < rows; i++)
        {
            for (int j = 0; j < cols; j++)
            {
                result[i, j] = matrix[i, j] * scalar;
            }
        }
        return result;
    }

    static int[,] Transpose(int[,] matrix)
    {
        int rows = matrix.GetLength(0);
        int cols = matrix.GetLength(1);
        int[,] result = new int[cols, rows];

        for (int i = 0; i < rows; i++)
        {
            for (int j = 0; j < cols; j++)
            {
                result[j, i] = matrix[i, j];
            }
        }
        return result;
    }

    static int SumAllElements(int[,] matrix)
    {
        int sum = 0;
        foreach (int element in matrix)
        {
            sum += element;
        }
        return sum;
    }

    static void PrintMatrix(int[,] matrix)
    {
        int rows = matrix.GetLength(0);
        int cols = matrix.GetLength(1);

        for (int i = 0; i < rows; i++)
        {
            for (int j = 0; j < cols; j++)
            {
                Console.Write($"{matrix[i, j],4}");
            }
            Console.WriteLine();
        }
    }
}
```

### Example 3: Practical Application - Seating Chart

```csharp
using System;

class Program
{
    static void Main()
    {
        // Theater seating: 5 rows, 8 seats per row
        // 0 = available, 1 = booked
        int[,] seats = new int[5, 8];

        // Book some seats
        seats[0, 3] = 1;
        seats[0, 4] = 1;
        seats[1, 2] = 1;
        seats[2, 5] = 1;
        seats[2, 6] = 1;
        seats[3, 0] = 1;
        seats[4, 7] = 1;

        Console.WriteLine("=== Theater Seating Chart ===\n");
        Console.WriteLine("    1  2  3  4  5  6  7  8");
        Console.WriteLine("   ------------------------");

        for (int row = 0; row < seats.GetLength(0); row++)
        {
            Console.Write($"{(char)('A' + row)} |");
            for (int seat = 0; seat < seats.GetLength(1); seat++)
            {
                string symbol = seats[row, seat] == 0 ? " ○" : " ×";
                Console.Write(symbol);
            }
            Console.WriteLine(" |");
        }

        Console.WriteLine("   ------------------------");
        Console.WriteLine("\n○ = Available  × = Booked");

        // Count available seats
        int totalSeats = seats.Length;
        int bookedSeats = 0;
        foreach (int seat in seats)
        {
            if (seat == 1) bookedSeats++;
        }
        int availableSeats = totalSeats - bookedSeats;

        Console.WriteLine($"\nTotal Seats: {totalSeats}");
        Console.WriteLine($"Booked: {bookedSeats}");
        Console.WriteLine($"Available: {availableSeats}");

        // Find best available seats (consecutive)
        Console.WriteLine("\nBest available pairs:");
        for (int row = 0; row < seats.GetLength(0); row++)
        {
            for (int seat = 0; seat < seats.GetLength(1) - 1; seat++)
            {
                if (seats[row, seat] == 0 && seats[row, seat + 1] == 0)
                {
                    Console.WriteLine($"Row {(char)('A' + row)}, Seats {seat + 1}-{seat + 2}");
                }
            }
        }
    }
}
```

### Example 4: 3D Array Example

```csharp
using System;

class Program
{
    static void Main()
    {
        // 3D array: Building with floors, rooms, sensors
        // Dimensions: [floors, rooms, sensor_readings]
        int[,,] buildingData = new int[3, 4, 2]; // 3 floors, 4 rooms each, 2 sensors (temp, humidity)

        // Populate with sample data
        Random rand = new Random(42);
        for (int floor = 0; floor < 3; floor++)
        {
            for (int room = 0; room < 4; room++)
            {
                buildingData[floor, room, 0] = rand.Next(18, 26); // Temperature 18-25°C
                buildingData[floor, room, 1] = rand.Next(40, 70); // Humidity 40-70%
            }
        }

        Console.WriteLine("=== Building Sensor Data ===\n");

        // Display data for each floor
        for (int floor = 0; floor < 3; floor++)
        {
            Console.WriteLine($"Floor {floor + 1}:");
            Console.WriteLine("Room | Temp(°C) | Humidity(%)");
            Console.WriteLine("-----|----------|------------");

            for (int room = 0; room < 4; room++)
            {
                int temp = buildingData[floor, room, 0];
                int humidity = buildingData[floor, room, 1];
                Console.WriteLine($"  {room + 1}  |    {temp,2}    |     {humidity,2}");
            }
            Console.WriteLine();
        }

        // Calculate averages per floor
        Console.WriteLine("Floor Averages:");
        for (int floor = 0; floor < 3; floor++)
        {
            int totalTemp = 0;
            int totalHumidity = 0;

            for (int room = 0; room < 4; room++)
            {
                totalTemp += buildingData[floor, room, 0];
                totalHumidity += buildingData[floor, room, 1];
            }

            double avgTemp = totalTemp / 4.0;
            double avgHumidity = totalHumidity / 4.0;

            Console.WriteLine($"Floor {floor + 1}: Avg Temp = {avgTemp:F1}°C, Avg Humidity = {avgHumidity:F1}%");
        }

        // Find hottest room
        int maxTemp = 0;
        int maxFloor = 0, maxRoom = 0;

        for (int floor = 0; floor < 3; floor++)
        {
            for (int room = 0; room < 4; room++)
            {
                if (buildingData[floor, room, 0] > maxTemp)
                {
                    maxTemp = buildingData[floor, room, 0];
                    maxFloor = floor;
                    maxRoom = room;
                }
            }
        }

        Console.WriteLine($"\nHottest Room: Floor {maxFloor + 1}, Room {maxRoom + 1} ({maxTemp}°C)");

        // Array properties
        Console.WriteLine($"\n3D Array Properties:");
        Console.WriteLine($"Rank: {buildingData.Rank}");
        Console.WriteLine($"Total elements: {buildingData.Length}");
        Console.WriteLine($"Dimension 0 (floors): {buildingData.GetLength(0)}");
        Console.WriteLine($"Dimension 1 (rooms): {buildingData.GetLength(1)}");
        Console.WriteLine($"Dimension 2 (sensors): {buildingData.GetLength(2)}");
    }
}
```

### Example 5: Image Representation (2D Array)

```csharp
using System;

class Program
{
    static void Main()
    {
        // Simple 8x8 grayscale image (0=black, 9=white)
        int[,] image = {
            { 0, 0, 0, 0, 0, 0, 0, 0 },
            { 0, 9, 9, 9, 9, 9, 9, 0 },
            { 0, 9, 0, 0, 0, 0, 9, 0 },
            { 0, 9, 0, 9, 9, 0, 9, 0 },
            { 0, 9, 0, 9, 9, 0, 9, 0 },
            { 0, 9, 0, 0, 0, 0, 9, 0 },
            { 0, 9, 9, 9, 9, 9, 9, 0 },
            { 0, 0, 0, 0, 0, 0, 0, 0 }
        };

        Console.WriteLine("Original Image:");
        DisplayImage(image);

        // Invert colors
        int[,] inverted = InvertImage(image);
        Console.WriteLine("\nInverted Image:");
        DisplayImage(inverted);

        // Brighten image
        int[,] brightened = BrightenImage(image, 2);
        Console.WriteLine("\nBrightened Image (+2):");
        DisplayImage(brightened);

        // Calculate average brightness
        double avgBrightness = CalculateAverageBrightness(image);
        Console.WriteLine($"\nAverage Brightness: {avgBrightness:F2}");
    }

    static void DisplayImage(int[,] img)
    {
        char[] shades = { '█', '▓', '▒', '░', '·', '·', '░', '▒', '▓', '█' };

        for (int i = 0; i < img.GetLength(0); i++)
        {
            for (int j = 0; j < img.GetLength(1); j++)
            {
                Console.Write($"{shades[img[i, j]]} ");
            }
            Console.WriteLine();
        }
    }

    static int[,] InvertImage(int[,] img)
    {
        int rows = img.GetLength(0);
        int cols = img.GetLength(1);
        int[,] result = new int[rows, cols];

        for (int i = 0; i < rows; i++)
        {
            for (int j = 0; j < cols; j++)
            {
                result[i, j] = 9 - img[i, j];
            }
        }
        return result;
    }

    static int[,] BrightenImage(int[,] img, int amount)
    {
        int rows = img.GetLength(0);
        int cols = img.GetLength(1);
        int[,] result = new int[rows, cols];

        for (int i = 0; i < rows; i++)
        {
            for (int j = 0; j < cols; j++)
            {
                result[i, j] = Math.Min(9, img[i, j] + amount);
            }
        }
        return result;
    }

    static double CalculateAverageBrightness(int[,] img)
    {
        int sum = 0;
        int count = 0;

        foreach (int pixel in img)
        {
            sum += pixel;
            count++;
        }

        return (double)sum / count;
    }
}
```

## Advantages of Multidimensional Arrays

1. Natural representation of matrices and grids
2. Contiguous memory allocation
3. Better cache performance than jagged arrays
4. Simple syntax for accessing elements

## Disadvantages

1. All dimensions must have fixed sizes
2. Cannot have rows of different lengths
3. Less flexible than jagged arrays
4. Memory allocated for entire structure even if partially used

## Common Use Cases

- Mathematical matrices
- Game boards (chess, tic-tac-toe)
- Image processing
- Spreadsheet data
- Scientific simulations
- Geographic grids
