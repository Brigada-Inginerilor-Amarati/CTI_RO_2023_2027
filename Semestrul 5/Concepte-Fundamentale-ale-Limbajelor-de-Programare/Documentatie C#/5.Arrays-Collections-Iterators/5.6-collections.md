# 5.6 Collections

## Description

**Collections** are classes that provide more flexible ways to work with groups of objects than arrays. Unlike arrays, collections can dynamically grow and shrink, and provide specialized data structures for different scenarios.

C# provides collections in the `System.Collections.Generic` namespace (type-safe, generic) and `System.Collections` namespace (non-generic, legacy).

## Common Collection Types

### Generic Collections (Recommended)
- **`List<T>`**: Dynamic array, most commonly used
- **`Dictionary<TKey, TValue>`**: Key-value pairs (hash table)
- **`Queue<T>`**: First-In-First-Out (FIFO) collection
- **`Stack<T>`**: Last-In-First-Out (LIFO) collection
- **`HashSet<T>`**: Unique elements, fast lookups
- **`LinkedList<T>`**: Doubly-linked list

### Key Advantages Over Arrays
- Dynamic sizing (grow/shrink automatically)
- Built-in methods for common operations
- Type safety (with generics)
- Specialized data structures

## Code Examples

### Example 1: List<T> - Dynamic Array

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Create a list
        List<string> fruits = new List<string>();

        // Add elements
        fruits.Add("Apple");
        fruits.Add("Banana");
        fruits.Add("Orange");

        Console.WriteLine("Initial list:");
        PrintList(fruits);

        // Add multiple elements
        fruits.AddRange(new[] { "Grape", "Mango" });
        Console.WriteLine("\nAfter AddRange:");
        PrintList(fruits);

        // Insert at specific position
        fruits.Insert(1, "Strawberry");
        Console.WriteLine("\nAfter Insert at index 1:");
        PrintList(fruits);

        // Remove element
        fruits.Remove("Banana");
        Console.WriteLine("\nAfter Remove 'Banana':");
        PrintList(fruits);

        // Remove at index
        fruits.RemoveAt(0);
        Console.WriteLine("\nAfter RemoveAt(0):");
        PrintList(fruits);

        // Check if contains
        bool hasOrange = fruits.Contains("Orange");
        Console.WriteLine($"\nContains 'Orange': {hasOrange}");

        // Find index
        int index = fruits.IndexOf("Mango");
        Console.WriteLine($"Index of 'Mango': {index}");

        // Access by index
        Console.WriteLine($"\nElement at index 2: {fruits[2]}");

        // Count
        Console.WriteLine($"Total fruits: {fruits.Count}");

        // Sort
        fruits.Sort();
        Console.WriteLine("\nAfter sorting:");
        PrintList(fruits);

        // Clear all
        fruits.Clear();
        Console.WriteLine($"\nAfter Clear: Count = {fruits.Count}");
    }

    static void PrintList(List<string> list)
    {
        Console.Write("[ ");
        for (int i = 0; i < list.Count; i++)
        {
            Console.Write(list[i]);
            if (i < list.Count - 1)
                Console.Write(", ");
        }
        Console.WriteLine(" ]");
    }
}
```

**Output:**
```
Initial list:
[ Apple, Banana, Orange ]

After AddRange:
[ Apple, Banana, Orange, Grape, Mango ]

After Insert at index 1:
[ Apple, Strawberry, Banana, Orange, Grape, Mango ]

After Remove 'Banana':
[ Apple, Strawberry, Orange, Grape, Mango ]

After RemoveAt(0):
[ Strawberry, Orange, Grape, Mango ]

Contains 'Orange': True
Index of 'Mango': 3

Element at index 2: Grape
Total fruits: 4

After sorting:
[ Grape, Mango, Orange, Strawberry ]

After Clear: Count = 0
```

### Example 2: Dictionary<TKey, TValue> - Key-Value Pairs

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Create a dictionary
        Dictionary<string, int> ages = new Dictionary<string, int>();

        // Add key-value pairs
        ages.Add("Alice", 25);
        ages.Add("Bob", 30);
        ages.Add("Charlie", 28);

        Console.WriteLine("=== Dictionary Example ===\n");

        // Access value by key
        Console.WriteLine($"Alice's age: {ages["Alice"]}");

        // Add using indexer
        ages["Diana"] = 32;
        Console.WriteLine($"Added Diana's age: {ages["Diana"]}");

        // Check if key exists
        if (ages.ContainsKey("Bob"))
        {
            Console.WriteLine($"\nBob is in the dictionary, age: {ages["Bob"]}");
        }

        // TryGetValue (safer than direct access)
        if (ages.TryGetValue("Eve", out int eveAge))
        {
            Console.WriteLine($"Eve's age: {eveAge}");
        }
        else
        {
            Console.WriteLine("\nEve not found in dictionary");
        }

        // Update value
        ages["Alice"] = 26;
        Console.WriteLine($"Updated Alice's age: {ages["Alice"]}");

        // Iterate through dictionary
        Console.WriteLine("\nAll entries:");
        foreach (KeyValuePair<string, int> kvp in ages)
        {
            Console.WriteLine($"{kvp.Key}: {kvp.Value} years old");
        }

        // Alternative iteration
        Console.WriteLine("\nUsing var:");
        foreach (var person in ages)
        {
            Console.WriteLine($"{person.Key} -> {person.Value}");
        }

        // Get all keys
        Console.WriteLine("\nAll keys:");
        foreach (string name in ages.Keys)
        {
            Console.Write($"{name} ");
        }
        Console.WriteLine();

        // Get all values
        Console.WriteLine("\nAll values:");
        foreach (int age in ages.Values)
        {
            Console.Write($"{age} ");
        }
        Console.WriteLine();

        // Remove an entry
        ages.Remove("Bob");
        Console.WriteLine($"\nAfter removing Bob: Count = {ages.Count}");

        // Product prices example
        Console.WriteLine("\n=== Product Catalog ===\n");
        Dictionary<string, double> prices = new Dictionary<string, double>
        {
            { "Laptop", 999.99 },
            { "Mouse", 25.99 },
            { "Keyboard", 79.99 },
            { "Monitor", 299.99 }
        };

        foreach (var product in prices)
        {
            Console.WriteLine($"{product.Key}: ${product.Value}");
        }

        // Calculate total
        double total = 0;
        foreach (var price in prices.Values)
        {
            total += price;
        }
        Console.WriteLine($"\nTotal inventory value: ${total:F2}");
    }
}
```

### Example 3: Queue<T> and Stack<T>

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== QUEUE (FIFO) ===\n");

        // Create a queue
        Queue<string> customers = new Queue<string>();

        // Enqueue (add to end)
        customers.Enqueue("Alice");
        customers.Enqueue("Bob");
        customers.Enqueue("Charlie");
        customers.Enqueue("Diana");

        Console.WriteLine($"Customers in queue: {customers.Count}");
        Console.WriteLine("Queue: " + string.Join(" <- ", customers));

        // Peek (look at first without removing)
        string next = customers.Peek();
        Console.WriteLine($"\nNext customer (Peek): {next}");
        Console.WriteLine($"Count after Peek: {customers.Count}");

        // Dequeue (remove from front)
        Console.WriteLine("\nServing customers:");
        while (customers.Count > 0)
        {
            string customer = customers.Dequeue();
            Console.WriteLine($"Serving: {customer} (Remaining: {customers.Count})");
        }

        Console.WriteLine("\n=== STACK (LIFO) ===\n");

        // Create a stack
        Stack<string> plates = new Stack<string>();

        // Push (add to top)
        plates.Push("Plate 1");
        plates.Push("Plate 2");
        plates.Push("Plate 3");
        plates.Push("Plate 4");

        Console.WriteLine($"Plates in stack: {plates.Count}");
        Console.WriteLine("Stack: " + string.Join(" | ", plates));

        // Peek (look at top without removing)
        string topPlate = plates.Peek();
        Console.WriteLine($"\nTop plate (Peek): {topPlate}");
        Console.WriteLine($"Count after Peek: {plates.Count}");

        // Pop (remove from top)
        Console.WriteLine("\nRemoving plates:");
        while (plates.Count > 0)
        {
            string plate = plates.Pop();
            Console.WriteLine($"Removed: {plate} (Remaining: {plates.Count})");
        }

        // Practical example: Undo functionality
        Console.WriteLine("\n=== Undo System Example ===\n");
        Stack<string> actions = new Stack<string>();

        actions.Push("Typed 'Hello'");
        Console.WriteLine("Action: Typed 'Hello'");

        actions.Push("Typed ' World'");
        Console.WriteLine("Action: Typed ' World'");

        actions.Push("Changed font to Bold");
        Console.WriteLine("Action: Changed font to Bold");

        Console.WriteLine("\nPerforming undo operations:");
        for (int i = 0; i < 2; i++)
        {
            if (actions.Count > 0)
            {
                string undone = actions.Pop();
                Console.WriteLine($"Undo: {undone}");
            }
        }

        Console.WriteLine($"\nActions remaining: {actions.Count}");
        if (actions.Count > 0)
        {
            Console.WriteLine($"Current state: {actions.Peek()}");
        }
    }
}
```

**Output:**
```

Queue: Alice <- Bob <- Charlie <- Diana

Next customer (Peek): Alice
Count after Peek: 4

Serving customers:
Serving: Alice (Remaining: 3)
Serving: Bob (Remaining: 2)
Serving: Charlie (Remaining: 1)
Serving: Diana (Remaining: 0)

=== STACK (LIFO) ===

```

### Example 4: HashSet<T> - Unique Elements

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== HashSet Example ===\n");

        // Create a hashset
        HashSet<int> numbers = new HashSet<int>();

        // Add elements
        numbers.Add(5);
        numbers.Add(10);
        numbers.Add(15);
        numbers.Add(10); // Duplicate - won't be added

        Console.WriteLine($"HashSet Count: {numbers.Count}");
        Console.WriteLine("Elements: " + string.Join(", ", numbers));

        // Check if contains
        Console.WriteLine($"\nContains 10: {numbers.Contains(10)}");
        Console.WriteLine($"Contains 20: {numbers.Contains(20)}");

        // Set operations
        HashSet<int> setA = new HashSet<int> { 1, 2, 3, 4, 5 };
        HashSet<int> setB = new HashSet<int> { 4, 5, 6, 7, 8 };

        Console.WriteLine("\n=== Set Operations ===");
        Console.WriteLine($"Set A: {string.Join(", ", setA)}");
        Console.WriteLine($"Set B: {string.Join(", ", setB)}");

        // Union
        HashSet<int> union = new HashSet<int>(setA);
        union.UnionWith(setB);
        Console.WriteLine($"\nUnion (A ∪ B): {string.Join(", ", union)}");

        // Intersection
        HashSet<int> intersection = new HashSet<int>(setA);
        intersection.IntersectWith(setB);
        Console.WriteLine($"Intersection (A ∩ B): {string.Join(", ", intersection)}");

        // Difference
        HashSet<int> difference = new HashSet<int>(setA);
        difference.ExceptWith(setB);
        Console.WriteLine($"Difference (A - B): {string.Join(", ", difference)}");

        // Symmetric Difference
        HashSet<int> symDiff = new HashSet<int>(setA);
        symDiff.SymmetricExceptWith(setB);
        Console.WriteLine($"Symmetric Diff (A △ B): {string.Join(", ", symDiff)}");

        // Practical example: Removing duplicates
        Console.WriteLine("\n=== Removing Duplicates ===");
        int[] numbersWithDuplicates = { 1, 2, 3, 2, 4, 1, 5, 3, 6 };
        Console.WriteLine($"Original: {string.Join(", ", numbersWithDuplicates)}");

        HashSet<int> unique = new HashSet<int>(numbersWithDuplicates);
        Console.WriteLine($"Unique: {string.Join(", ", unique)}");

        // Practical example: Unique visitors
        Console.WriteLine("\n=== Website Visitors ===");
        HashSet<string> visitors = new HashSet<string>();

        string[] visits = { "Alice", "Bob", "Alice", "Charlie", "Bob", "Diana", "Alice" };

        Console.WriteLine("Visit log:");
        foreach (string visitor in visits)
        {
            bool isNew = visitors.Add(visitor);
            string status = isNew ? "New visitor" : "Returning visitor";
            Console.WriteLine($"{visitor} - {status}");
        }

        Console.WriteLine($"\nTotal unique visitors: {visitors.Count}");
        Console.WriteLine($"Unique visitors: {string.Join(", ", visitors)}");
    }
}
```

### Example 5: Practical Application - Student Management System

```csharp
using System;
using System.Collections.Generic;

class Student
{
    public int Id { get; set; }
    public string Name { get; set; }
    public List<int> Grades { get; set; }

    public Student(int id, string name)
    {
        Id = id;
        Name = name;
        Grades = new List<int>();
    }

    public double GetAverage()
    {
        if (Grades.Count == 0) return 0;
        int sum = 0;
        foreach (int grade in Grades)
        {
            sum += grade;
        }
        return (double)sum / Grades.Count;
    }
}

class Program
{
    static void Main()
    {
        // Dictionary to store students by ID
        Dictionary<int, Student> students = new Dictionary<int, Student>();

        // Add students
        students.Add(101, new Student(101, "Alice"));
        students.Add(102, new Student(102, "Bob"));
        students.Add(103, new Student(103, "Charlie"));

        // Add grades
        students[101].Grades.AddRange(new[] { 85, 90, 88 });
        students[102].Grades.AddRange(new[] { 78, 82, 80, 85 });
        students[103].Grades.AddRange(new[] { 92, 95, 93 });

        Console.WriteLine("=== Student Management System ===\n");

        // Display all students
        foreach (var kvp in students)
        {
            Student s = kvp.Value;
            Console.WriteLine($"ID: {s.Id} | Name: {s.Name}");
            Console.WriteLine($"Grades: {string.Join(", ", s.Grades)}");
            Console.WriteLine($"Average: {s.GetAverage():F2}\n");
        }

        // Find student with highest average
        double highestAvg = 0;
        Student topStudent = null;

        foreach (var student in students.Values)
        {
            double avg = student.GetAverage();
            if (avg > highestAvg)
            {
                highestAvg = avg;
                topStudent = student;
            }
        }

        Console.WriteLine($"Top Student: {topStudent.Name} with average {highestAvg:F2}");

        // List of students who need improvement (average < 80)
        Console.WriteLine("\n=== Students Needing Improvement ===");
        List<Student> needsImprovement = new List<Student>();

        foreach (var student in students.Values)
        {
            if (student.GetAverage() < 85)
            {
                needsImprovement.Add(student);
            }
        }

        if (needsImprovement.Count > 0)
        {
            foreach (var student in needsImprovement)
            {
                Console.WriteLine($"{student.Name}: {student.GetAverage():F2}");
            }
        }
        else
        {
            Console.WriteLine("All students performing well!");
        }

        // Class statistics
        Console.WriteLine("\n=== Class Statistics ===");
        Console.WriteLine($"Total Students: {students.Count}");

        int totalGrades = 0;
        int gradeCount = 0;
        foreach (var student in students.Values)
        {
            foreach (int grade in student.Grades)
            {
                totalGrades += grade;
                gradeCount++;
            }
        }

        double classAverage = (double)totalGrades / gradeCount;
        Console.WriteLine($"Class Average: {classAverage:F2}");
        Console.WriteLine($"Total Grades Recorded: {gradeCount}");
    }
}
```

```

Grades: 85, 90, 88
Average: 87.67

ID: 102 | Name: Bob
Grades: 78, 82, 80, 85
Average: 81.25

Grades: 92, 95, 93
Average: 93.33

Top Student: Charlie with average 93.33

```

## Collection Comparison

| Collection | Ordered | Unique | Access | Use Case |
|------------|---------|--------|--------|----------|
| `List<T>` | Yes | No | By index | General purpose, dynamic array |
| `Dictionary<K,V>` | No | Keys only | By key | Key-value lookups |
| `Queue<T>` | FIFO | No | First only | Task scheduling, BFS |
| `Stack<T>` | LIFO | No | Last only | Undo, DFS, parsing |
| `HashSet<T>` | No | Yes | By value | Unique items, set operations |
| `LinkedList<T>` | Yes | No | Sequential | Frequent insert/delete |

## Best Practices

1. Use `List<T>` as default collection for most scenarios
2. Use `Dictionary<K,V>` for fast key-based lookups
3. Use `HashSet<T>` when uniqueness matters
4. Use `Queue<T>` and `Stack<T>` for specific ordering requirements
5. Prefer generic collections over non-generic ones
6. Initialize collections with capacity if size is known
