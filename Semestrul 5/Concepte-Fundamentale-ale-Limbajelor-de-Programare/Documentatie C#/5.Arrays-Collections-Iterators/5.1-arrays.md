# 5.1 Arrays

## Description

An **array** is a data structure that stores a fixed-size sequential collection of elements of the same type. Arrays are reference types derived from the `System.Array` class.

Key characteristics:
- **Fixed size**: Size is determined at creation and cannot be changed
- **Zero-indexed**: First element is at index 0
- **Type-safe**: All elements must be of the same type
- **Contiguous memory**: Elements are stored in consecutive memory locations

## Key Concepts

### Array Declaration and Initialization
```csharp
// Declaration
type[] arrayName;

// Initialization
arrayName = new type[size];

// Combined
type[] arrayName = new type[size];
```

### Array Properties and Methods
- `Length`: Returns the total number of elements
- `Rank`: Returns the number of dimensions
- `GetLength(dimension)`: Returns the size of a specific dimension
- `Clone()`: Creates a shallow copy

## Code Examples

### Example 1: Basic Array Operations

```csharp
using System;

class Program
{
    static void Main()
    {
        // Declaration and initialization
        int[] numbers = new int[5];

        // Assigning values
        numbers[0] = 10;
        numbers[1] = 20;
        numbers[2] = 30;
        numbers[3] = 40;
        numbers[4] = 50;

        // Accessing elements
        Console.WriteLine("Array elements:");
        for (int i = 0; i < numbers.Length; i++)
        {
            Console.WriteLine($"numbers[{i}] = {numbers[i]}");
        }

        // Modifying elements
        numbers[2] = 35;
        Console.WriteLine($"\nAfter modification, numbers[2] = {numbers[2]}");

        // Array properties
        Console.WriteLine($"\nArray Length: {numbers.Length}");
        Console.WriteLine($"Array Rank: {numbers.Rank}");
    }
}
```

**Output:**
```
Array elements:
numbers[0] = 10
numbers[1] = 20
numbers[2] = 30
numbers[3] = 40
numbers[4] = 50

After modification, numbers[2] = 35

Array Length: 5
Array Rank: 1
```

### Example 2: Array Initialization Syntax

```csharp
using System;

class Program
{
    static void Main()
    {
        // Method 1: Size specified, default values
        int[] arr1 = new int[3];
        Console.WriteLine("arr1 (default values):");
        foreach (int num in arr1)
            Console.Write($"{num} ");

        // Method 2: Initialize with values
        int[] arr2 = new int[] { 1, 2, 3, 4, 5 };
        Console.WriteLine("\n\narr2 (with values):");
        foreach (int num in arr2)
            Console.Write($"{num} ");

        // Method 3: Shorthand initialization
        int[] arr3 = { 10, 20, 30, 40 };
        Console.WriteLine("\n\narr3 (shorthand):");
        foreach (int num in arr3)
            Console.Write($"{num} ");

        // Method 4: String array
        string[] names = { "Alice", "Bob", "Charlie", "Diana" };
        Console.WriteLine("\n\nString array:");
        foreach (string name in names)
            Console.WriteLine($"- {name}");

        // Method 5: Mixed initialization
        double[] prices = new double[4] { 9.99, 19.99, 29.99, 39.99 };
        Console.WriteLine("\nPrices:");
        for (int i = 0; i < prices.Length; i++)
            Console.WriteLine($"Item {i + 1}: ${prices[i]}");
    }
}
```

**Output:**
```
arr1 (default values):
0 0 0

arr2 (with values):
1 2 3 4 5

arr3 (shorthand):
10 20 30 40

String array:
- Alice
- Bob
- Charlie
- Diana

Prices:
Item 1: $9.99
Item 2: $19.99
Item 3: $29.99
Item 4: $39.99
```

### Example 3: Array Methods and Operations

```csharp
using System;

class Program
{
    static void Main()
    {
        int[] numbers = { 5, 2, 8, 1, 9, 3, 7, 4, 6 };

        Console.WriteLine("Original array:");
        PrintArray(numbers);

        // Array.Sort - sorts in ascending order
        Array.Sort(numbers);
        Console.WriteLine("\nAfter sorting:");
        PrintArray(numbers);

        // Array.Reverse - reverses the array
        Array.Reverse(numbers);
        Console.WriteLine("\nAfter reversing:");
        PrintArray(numbers);

        // Array.IndexOf - finds index of element
        int searchValue = 7;
        int index = Array.IndexOf(numbers, searchValue);
        Console.WriteLine($"\nIndex of {searchValue}: {index}");

        // Array.Find - finds first element matching condition
        int firstEven = Array.Find(numbers, x => x % 2 == 0);
        Console.WriteLine($"First even number: {firstEven}");

        // Array.FindAll - finds all elements matching condition
        int[] evenNumbers = Array.FindAll(numbers, x => x % 2 == 0);
        Console.WriteLine("\nAll even numbers:");
        PrintArray(evenNumbers);

        // Array.Exists - checks if element exists
        bool hasNine = Array.Exists(numbers, x => x == 9);
        Console.WriteLine($"\nArray contains 9: {hasNine}");

        // Array.Clear - sets elements to default value
        int[] temp = { 1, 2, 3, 4, 5 };
        Console.WriteLine("\nBefore Clear:");
        PrintArray(temp);
        Array.Clear(temp, 1, 3); // Clear 3 elements starting at index 1
        Console.WriteLine("After Clear (index 1-3):");
        PrintArray(temp);
    }

    static void PrintArray(int[] arr)
    {
        foreach (int num in arr)
            Console.Write($"{num} ");
        Console.WriteLine();
    }
}
```

**Output:**
```
Original array:
5 2 8 1 9 3 7 4 6

After sorting:
1 2 3 4 5 6 7 8 9

After reversing:
9 8 7 6 5 4 3 2 1

Index of 7: 2
First even number: 8

All even numbers:
8 6 4 2

Array contains 9: True

Before Clear:
1 2 3 4 5
After Clear (index 1-3):
1 0 0 0 5
```

### Example 4: Working with Arrays - Practical Example

```csharp
using System;

class Program
{
    static void Main()
    {
        // Student grades example
        Console.WriteLine("=== Student Grades Analysis ===\n");

        string[] students = { "Alice", "Bob", "Charlie", "Diana", "Eve" };
        double[] grades = { 85.5, 92.0, 78.5, 88.0, 95.5 };

        // Display all grades
        Console.WriteLine("Student Grades:");
        for (int i = 0; i < students.Length; i++)
        {
            Console.WriteLine($"{students[i]}: {grades[i]:F1}");
        }

        // Calculate average
        double sum = 0;
        foreach (double grade in grades)
        {
            sum += grade;
        }
        double average = sum / grades.Length;
        Console.WriteLine($"\nClass Average: {average:F2}");

        // Find highest grade
        double highest = grades[0];
        int highestIndex = 0;
        for (int i = 1; i < grades.Length; i++)
        {
            if (grades[i] > highest)
            {
                highest = grades[i];
                highestIndex = i;
            }
        }
        Console.WriteLine($"Highest Grade: {students[highestIndex]} with {highest:F1}");

        // Find lowest grade
        double lowest = grades[0];
        int lowestIndex = 0;
        for (int i = 1; i < grades.Length; i++)
        {
            if (grades[i] < lowest)
            {
                lowest = grades[i];
                lowestIndex = i;
            }
        }
        Console.WriteLine($"Lowest Grade: {students[lowestIndex]} with {lowest:F1}");

        // Count students above average
        int aboveAverage = 0;
        foreach (double grade in grades)
        {
            if (grade > average)
                aboveAverage++;
        }
        Console.WriteLine($"\nStudents above average: {aboveAverage} out of {students.Length}");

        // Grade distribution
        Console.WriteLine("\nGrade Distribution:");
        for (int i = 0; i < students.Length; i++)
        {
            string status = grades[i] >= average ? "Above" : "Below";
            Console.WriteLine($"{students[i]}: {status} average");
        }
    }
}
```

**Output:**
```
=== Student Grades Analysis ===

Student Grades:
Alice: 85.5
Bob: 92.0
Charlie: 78.5
Diana: 88.0
Eve: 95.5

Class Average: 87.90
Highest Grade: Eve with 95.5
Lowest Grade: Charlie with 78.5

Students above average: 3 out of 5

Grade Distribution:
Alice: Below average
Bob: Above average
Charlie: Below average
Diana: Above average
Eve: Above average
```

### Example 5: Copying and Cloning Arrays

```csharp
using System;

class Program
{
    static void Main()
    {
        int[] original = { 1, 2, 3, 4, 5 };

        Console.WriteLine("Original array:");
        PrintArray(original);

        // Method 1: Reference copy (points to same array)
        int[] reference = original;
        reference[0] = 999;
        Console.WriteLine("\nAfter modifying reference:");
        Console.Write("Original: ");
        PrintArray(original);
        Console.Write("Reference: ");
        PrintArray(reference);

        // Reset original
        original[0] = 1;

        // Method 2: Clone (shallow copy)
        int[] cloned = (int[])original.Clone();
        cloned[0] = 100;
        Console.WriteLine("\nAfter modifying clone:");
        Console.Write("Original: ");
        PrintArray(original);
        Console.Write("Cloned: ");
        PrintArray(cloned);

        // Method 3: Array.Copy
        int[] copied = new int[original.Length];
        Array.Copy(original, copied, original.Length);
        copied[0] = 200;
        Console.WriteLine("\nAfter modifying copy:");
        Console.Write("Original: ");
        PrintArray(original);
        Console.Write("Copied: ");
        PrintArray(copied);

        // Method 4: Partial copy
        int[] partial = new int[3];
        Array.Copy(original, 1, partial, 0, 3); // Copy 3 elements starting from index 1
        Console.WriteLine("\nPartial copy (elements 1-3 of original):");
        PrintArray(partial);
    }

    static void PrintArray(int[] arr)
    {
        foreach (int num in arr)
            Console.Write($"{num} ");
        Console.WriteLine();
    }
}
```

**Output:**
```
Original array:
1 2 3 4 5

After modifying reference:
Original: 999 2 3 4 5
Reference: 999 2 3 4 5

After modifying clone:
Original: 1 2 3 4 5
Cloned: 100 2 3 4 5

After modifying copy:
Original: 1 2 3 4 5
Copied: 200 2 3 4 5

Partial copy (elements 1-3 of original):
2 3 4
```

## Common Array Pitfalls

1. **Index Out of Bounds**: Accessing index >= Length throws `IndexOutOfRangeException`
2. **Null Reference**: Array variable can be null if not initialized
3. **Fixed Size**: Cannot resize array after creation
4. **Reference Copy**: Assigning array to another variable creates a reference, not a copy

## Best Practices

- Initialize arrays before use to avoid null reference exceptions
- Check array bounds before accessing elements
- Use `foreach` loop when you don't need the index
- Use `Length` property instead of hard-coded values
- Consider using `List<T>` if you need dynamic sizing
