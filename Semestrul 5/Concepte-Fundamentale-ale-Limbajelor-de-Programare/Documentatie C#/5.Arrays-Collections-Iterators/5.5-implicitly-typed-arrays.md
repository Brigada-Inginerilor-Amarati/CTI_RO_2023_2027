# 5.5 Implicitly Typed Arrays

## Description

**Implicitly typed arrays** use the `var` keyword to let the compiler infer the array type from the initialization expression. The compiler determines the type based on the elements provided in the array initializer.

Key characteristics:
- Uses `var` keyword instead of explicit type
- Type is determined at compile-time (not runtime)
- All elements must be implicitly convertible to a single type
- Cannot be used without initialization
- Type inference is based on the best common type of all elements

## Key Concepts

### Syntax
```csharp
var arrayName = new[] { element1, element2, element3 };
```

### Type Inference Rules
1. Compiler finds the best common type among all elements
2. All elements must be implicitly convertible to that type
3. If no common type can be determined, compilation fails
4. Once inferred, the type is fixed and type-safe

### When to Use
- When the type is obvious from the initializer
- To reduce code verbosity
- When working with anonymous types
- For improved code readability

## Code Examples

### Example 1: Basic Implicit Type Inference

```csharp
using System;

class Program
{
    static void Main()
    {
        // Explicitly typed array
        int[] explicitArray = new int[] { 1, 2, 3, 4, 5 };
        Console.WriteLine($"Explicit type: {explicitArray.GetType()}");
        Console.WriteLine($"Elements: {string.Join(", ", explicitArray)}\n");

        // Implicitly typed array - compiler infers int[]
        var implicitArray = new[] { 1, 2, 3, 4, 5 };
        Console.WriteLine($"Implicit type: {implicitArray.GetType()}");
        Console.WriteLine($"Elements: {string.Join(", ", implicitArray)}\n");

        // String array
        var names = new[] { "Alice", "Bob", "Charlie" };
        Console.WriteLine($"String array type: {names.GetType()}");
        Console.WriteLine($"Elements: {string.Join(", ", names)}\n");

        // Double array
        var prices = new[] { 9.99, 19.99, 29.99, 39.99 };
        Console.WriteLine($"Double array type: {prices.GetType()}");
        Console.WriteLine($"Elements: {string.Join(", ", prices)}\n");

        // Boolean array
        var flags = new[] { true, false, true, true };
        Console.WriteLine($"Boolean array type: {flags.GetType()}");
        Console.WriteLine($"Elements: {string.Join(", ", flags)}");
    }
}
```

**Output:**
```
Explicit type: System.Int32[]
Elements: 1, 2, 3, 4, 5

Implicit type: System.Int32[]
Elements: 1, 2, 3, 4, 5

String array type: System.String[]
Elements: Alice, Bob, Charlie

Double array type: System.Double[]
Elements: 9.99, 19.99, 29.99, 39.99

Boolean array type: System.Boolean[]
Elements: True, False, True, True
```

### Example 2: Type Inference with Mixed Types

```csharp
using System;

class Program
{
    static void Main()
    {
        // Mixed int and double - inferred as double[]
        var mixedNumbers = new[] { 1, 2.5, 3, 4.7, 5 };
        Console.WriteLine($"Mixed numbers type: {mixedNumbers.GetType()}");
        Console.WriteLine($"Elements: {string.Join(", ", mixedNumbers)}");
        Console.WriteLine($"All values stored as: {mixedNumbers[0].GetType()}\n");

        // All integers - inferred as int[]
        var integers = new[] { 1, 2, 3, 4, 5 };
        Console.WriteLine($"Integers type: {integers.GetType()}");
        Console.WriteLine($"Elements: {string.Join(", ", integers)}\n");

        // Mixed short and int - inferred as int[]
        short s1 = 10, s2 = 20;
        var mixedInts = new[] { s1, s2, 30, 40 };
        Console.WriteLine($"Mixed short/int type: {mixedInts.GetType()}");
        Console.WriteLine($"Elements: {string.Join(", ", mixedInts)}\n");

        // Nullable types
        var nullableInts = new int?[] { 1, 2, null, 4, null };
        Console.WriteLine($"Nullable int type: {nullableInts.GetType()}");
        Console.Write("Elements: ");
        foreach (var n in nullableInts)
        {
            Console.Write($"{(n.HasValue ? n.Value.ToString() : "null")} ");
        }
        Console.WriteLine("\n");

        // This would cause compilation error - no common type:
        // var invalid = new[] { 1, "two", 3.0 }; // ERROR!
    }
}
```

**Output:**
```
Mixed numbers type: System.Double[]
Elements: 1, 2.5, 3, 4.7, 5
All values stored as: System.Double

Integers type: System.Int32[]
Elements: 1, 2, 3, 4, 5

Mixed short/int type: System.Int32[]
Elements: 10, 20, 30, 40

Nullable int type: System.Nullable`1[System.Int32][]
Elements: 1 2 null 4 null
```

### Example 3: Implicitly Typed Multidimensional and Jagged Arrays

```csharp
using System;

class Program
{
    static void Main()
    {
        // Implicitly typed 2D array
        var matrix = new[,] {
            { 1, 2, 3 },
            { 4, 5, 6 },
            { 7, 8, 9 }
        };

        Console.WriteLine("2D Array:");
        Console.WriteLine($"Type: {matrix.GetType()}");
        Console.WriteLine($"Rank: {matrix.Rank}");
        Console.WriteLine("Elements:");
        for (int i = 0; i < matrix.GetLength(0); i++)
        {
            for (int j = 0; j < matrix.GetLength(1); j++)
            {
                Console.Write($"{matrix[i, j],3}");
            }
            Console.WriteLine();
        }

        // Implicitly typed jagged array
        var jagged = new[] {
            new[] { 1, 2, 3 },
            new[] { 4, 5 },
            new[] { 6, 7, 8, 9 }
        };

        Console.WriteLine("\nJagged Array:");
        Console.WriteLine($"Type: {jagged.GetType()}");
        Console.WriteLine("Elements:");
        for (int i = 0; i < jagged.Length; i++)
        {
            Console.Write($"Row {i}: ");
            foreach (int num in jagged[i])
            {
                Console.Write($"{num} ");
            }
            Console.WriteLine();
        }

        // Implicitly typed array of strings (jagged structure)
        var names = new[] {
            new[] { "Alice", "Bob" },
            new[] { "Charlie" },
            new[] { "David", "Eve", "Frank" }
        };

        Console.WriteLine("\nJagged String Array:");
        Console.WriteLine($"Type: {names.GetType()}");
        for (int i = 0; i < names.Length; i++)
        {
            Console.WriteLine($"Group {i + 1}: {string.Join(", ", names[i])}");
        }
    }
}
```

**Output:**
```
2D Array:
Type: System.Int32[,]
Rank: 2
Elements:
  1  2  3
  4  5  6
  7  8  9

Jagged Array:
Type: System.Int32[][]
Elements:
Row 0: 1 2 3
Row 1: 4 5
Row 2: 6 7 8 9

Jagged String Array:
Type: System.String[][]
Group 1: Alice, Bob
Group 2: Charlie
Group 3: David, Eve, Frank
```

### Example 4: Anonymous Types with Implicit Arrays

```csharp
using System;

class Program
{
    static void Main()
    {
        // Array of anonymous types - MUST use var
        var employees = new[] {
            new { Id = 1, Name = "Alice", Salary = 50000 },
            new { Id = 2, Name = "Bob", Salary = 60000 },
            new { Id = 3, Name = "Charlie", Salary = 55000 }
        };

        Console.WriteLine("Employee Array:");
        Console.WriteLine($"Type: {employees.GetType()}\n");

        foreach (var emp in employees)
        {
            Console.WriteLine($"ID: {emp.Id}, Name: {emp.Name}, Salary: ${emp.Salary:N0}");
        }

        // Calculate average salary
        int totalSalary = 0;
        foreach (var emp in employees)
        {
            totalSalary += emp.Salary;
        }
        double avgSalary = totalSalary / (double)employees.Length;
        Console.WriteLine($"\nAverage Salary: ${avgSalary:N2}");

        // Array of products
        var products = new[] {
            new { Code = "A01", Name = "Laptop", Price = 999.99, InStock = true },
            new { Code = "A02", Name = "Mouse", Price = 25.99, InStock = true },
            new { Code = "A03", Name = "Keyboard", Price = 79.99, InStock = false }
        };

        Console.WriteLine("\n=== Product Catalog ===");
        foreach (var product in products)
        {
            string status = product.InStock ? "✓ In Stock" : "✗ Out of Stock";
            Console.WriteLine($"[{product.Code}] {product.Name}: ${product.Price} - {status}");
        }

        // Count products in stock
        int inStockCount = 0;
        foreach (var p in products)
        {
            if (p.InStock) inStockCount++;
        }
        Console.WriteLine($"\nProducts in stock: {inStockCount}/{products.Length}");
    }
}
```

**Output:**
```
Employee Array:
Type: <>f__AnonymousType0`3[System.Int32,System.String,System.Int32][]

ID: 1, Name: Alice, Salary: $50,000
ID: 2, Name: Bob, Salary: $60,000
ID: 3, Name: Charlie, Salary: $55,000

Average Salary: $55,000.00

=== Product Catalog ===
[A01] Laptop: $999.99 - ✓ In Stock
[A02] Mouse: $25.99 - ✓ In Stock
[A03] Keyboard: $79.99 - ✗ Out of Stock

Products in stock: 2/3
```

### Example 5: Practical Example - Data Processing

```csharp
using System;

class Program
{
    static void Main()
    {
        // Temperature readings from different sensors
        var sensorData = new[] {
            new { SensorId = "S01", Location = "Room A", Temperature = 22.5 },
            new { SensorId = "S02", Location = "Room B", Temperature = 24.0 },
            new { SensorId = "S03", Location = "Room C", Temperature = 21.5 },
            new { SensorId = "S04", Location = "Room D", Temperature = 25.5 },
            new { SensorId = "S05", Location = "Room E", Temperature = 23.0 }
        };

        Console.WriteLine("=== Temperature Monitoring System ===\n");
        Console.WriteLine("Sensor ID | Location | Temperature");
        Console.WriteLine("----------|----------|------------");

        double sum = 0;
        double maxTemp = sensorData[0].Temperature;
        double minTemp = sensorData[0].Temperature;
        string hottest = sensorData[0].Location;
        string coldest = sensorData[0].Location;

        foreach (var sensor in sensorData)
        {
            Console.WriteLine($"   {sensor.SensorId}    | {sensor.Location,-8} |   {sensor.Temperature}°C");

            sum += sensor.Temperature;

            if (sensor.Temperature > maxTemp)
            {
                maxTemp = sensor.Temperature;
                hottest = sensor.Location;
            }

            if (sensor.Temperature < minTemp)
            {
                minTemp = sensor.Temperature;
                coldest = sensor.Location;
            }
        }

        double average = sum / sensorData.Length;

        Console.WriteLine("\n=== Statistics ===");
        Console.WriteLine($"Average Temperature: {average:F2}°C");
        Console.WriteLine($"Hottest Location: {hottest} ({maxTemp}°C)");
        Console.WriteLine($"Coldest Location: {coldest} ({minTemp}°C)");

        // Alert for temperatures outside comfort zone (20-24°C)
        Console.WriteLine("\n=== Temperature Alerts ===");
        bool hasAlerts = false;
        foreach (var sensor in sensorData)
        {
            if (sensor.Temperature < 20 || sensor.Temperature > 24)
            {
                string alert = sensor.Temperature > 24 ? "⚠ TOO HOT" : "⚠ TOO COLD";
                Console.WriteLine($"{sensor.Location}: {sensor.Temperature}°C {alert}");
                hasAlerts = true;
            }
        }

        if (!hasAlerts)
        {
            Console.WriteLine("✓ All temperatures within normal range");
        }
    }
}
```

**Output:**
```
=== Temperature Monitoring System ===

Sensor ID | Location | Temperature
----------|----------|------------
   S01    | Room A   |   22.5°C
   S02    | Room B   |   24°C
   S03    | Room C   |   21.5°C
   S04    | Room D   |   25.5°C
   S05    | Room E   |   23°C

=== Statistics ===
Average Temperature: 23.30°C
Hottest Location: Room D (25.5°C)
Coldest Location: Room C (21.5°C)

=== Temperature Alerts ===
Room D: 25.5°C ⚠ TOO HOT
```

## Rules and Limitations

### ✓ Valid
```csharp
var arr1 = new[] { 1, 2, 3 };                    // int[]
var arr2 = new[] { "a", "b", "c" };              // string[]
var arr3 = new[] { 1, 2.5, 3 };                  // double[]
var arr4 = new[] { new { X = 1, Y = 2 } };       // anonymous type[]
```

### ✗ Invalid
```csharp
var arr1 = new[] { 1, "two", 3.0 };              // ERROR: No common type
var arr2;                                         // ERROR: Must initialize
arr2 = new[] { 1, 2, 3 };
var arr3 = { 1, 2, 3 };                          // ERROR: Must use 'new[]'
```

## Advantages

1. **Reduced verbosity**: Less typing, especially with complex types
2. **Required for anonymous types**: Only way to create arrays of anonymous types
3. **Improved readability**: Type is obvious from initialization
4. **Refactoring friendly**: Change element type without changing declaration

## Disadvantages

1. **Less explicit**: Type not immediately visible in code
2. **Must initialize immediately**: Cannot declare without initialization
3. **Requires compatible types**: All elements must share common type
4. **Can reduce clarity**: For complex types, explicit declaration may be clearer

## Best Practices

1. Use when type is obvious from the initializer
2. Use for anonymous type arrays (required)
3. Avoid when it makes code less readable
4. Use explicit typing for public APIs and method signatures
5. Consistent within your codebase
