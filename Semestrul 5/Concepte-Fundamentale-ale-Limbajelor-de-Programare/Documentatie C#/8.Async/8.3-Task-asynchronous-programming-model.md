# 8.3 Task Asynchronous Programming Model

## Description

The Task-based Asynchronous Pattern (TAP) is the recommended approach for asynchronous programming in .NET. It uses the `Task` and `Task<T>` types to represent asynchronous operations. This model provides a consistent way to handle asynchronous operations, exceptions, and cancellation across different types of operations.

## Key Components

- **Task**: Represents an asynchronous operation that doesn't return a value
- **Task<T>**: Represents an asynchronous operation that returns a value of type T
- **TaskCompletionSource<T>**: Allows manual control over task completion
- **Task.Run**: Queues work to run on the ThreadPool
- **Task.FromResult**: Creates a completed task with a result

## Code Examples

### Basic Task Operations

```csharp
using System;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Task Asynchronous Programming Model Demo");
        
        // Basic task operations
        await BasicTaskOperations();
        
        // Task creation methods
        await TaskCreationMethods();
        
        // Task continuation
        await TaskContinuationDemo();
    }
    
    static async Task BasicTaskOperations()
    {
        Console.WriteLine("\n=== Basic Task Operations ===");
        
        // Create and run a task
        Task task1 = Task.Run(() => {
            Console.WriteLine("Task 1: Running on background thread");
            Task.Delay(1000).Wait();
            Console.WriteLine("Task 1: Completed");
        });
        
        // Create a task that returns a value
        Task<string> task2 = Task.Run(() => {
            Console.WriteLine("Task 2: Processing data...");
            Task.Delay(1500).Wait();
            Console.WriteLine("Task 2: Data processed");
            return "Task 2 Result";
        });
        
        // Wait for tasks to complete
        await task1;
        string result = await task2;
        
        Console.WriteLine($"Task 2 returned: {result}");
    }
    
    static async Task TaskCreationMethods()
    {
        Console.WriteLine("\n=== Task Creation Methods ===");
        
        // Task.FromResult - completed task with result
        Task<string> completedTask = Task.FromResult("Immediate result");
        string immediateResult = await completedTask;
        Console.WriteLine($"FromResult: {immediateResult}");
        
        // Task.Delay - async delay
        Console.WriteLine("Starting delay...");
        await Task.Delay(1000);
        Console.WriteLine("Delay completed!");
        
        // Task.Run with different return types
        Task<int> intTask = Task.Run(() => {
            Console.WriteLine("Calculating...");
            return 42;
        });
        
        Task<double> doubleTask = Task.Run(() => {
            Console.WriteLine("Computing...");
            return 3.14159;
        });
        
        var results = await Task.WhenAll(intTask, doubleTask);
        Console.WriteLine($"Results: {results[0]}, {results[1]}");
    }
    
    static async Task TaskContinuationDemo()
    {
        Console.WriteLine("\n=== Task Continuation Demo ===");
        
        // Chain tasks using ContinueWith
        Task<string> task = Task.Run(() => {
            Console.WriteLine("Initial task: Processing...");
            return "Initial data";
        });
        
        Task<string> continuation = task.ContinueWith(antecedent => {
            Console.WriteLine($"Continuation: Processing {antecedent.Result}");
            return $"Processed: {antecedent.Result}";
        });
        
        string finalResult = await continuation;
        Console.WriteLine($"Final result: {finalResult}");
    }
}
```

**Expected Output:**
```

Task 1: Running on background thread
Task 2: Processing data...
Task 1: Completed
Task 2: Data processed
Task 2 returned: Task 2 Result

=== Task Creation Methods ===
FromResult: Immediate result
Starting delay...
Delay completed!
Calculating...
Computing...
Results: 42, 3.14159

=== Task Continuation Demo ===

Continuation: Processing Initial data
Final result: Processed: Initial data
```

### TaskCompletionSource Example

```csharp
using System;
using System.Threading;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("TaskCompletionSource Demo");
        
        // Manual task control
        await ManualTaskControl();
        
        // Event-based task completion
        await EventBasedTask();
    }
    
    static async Task ManualTaskControl()
    {
        Console.WriteLine("\n=== Manual Task Control ===");
        
        var tcs = new TaskCompletionSource<string>();
        
        // Simulate some work that will complete the task
        Task.Run(async () => {
            await Task.Delay(2000);
            tcs.SetResult("Manual task completed successfully!");
        });
        
        Console.WriteLine("Waiting for manual task completion...");
        string result = await tcs.Task;
        Console.WriteLine($"Result: {result}");
    }
    
    static async Task EventBasedTask()
    {
        Console.WriteLine("\n=== Event-Based Task ===");
        
        var tcs = new TaskCompletionSource<int>();
        var random = new Random();
        
        // Simulate an event that will complete the task
        Task.Run(async () => {
            await Task.Delay(1500);
            int value = random.Next(1, 100);
            tcs.SetResult(value);
        });
        
        Console.WriteLine("Waiting for event-based task...");
        int result = await tcs.Task;
        Console.WriteLine($"Event result: {result}");
    }
}
```

**Expected Output:**
```
TaskCompletionSource Demo

=== Manual Task Control ===
Waiting for manual task completion...
Result: Manual task completed successfully!

=== Event-Based Task ===
Waiting for event-based task...
Event result: 73
```

### Task Exception Handling

```csharp
using System;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Task Exception Handling Demo");
        
        await TaskExceptionHandling();
    }
    
    static async Task TaskExceptionHandling()
    {
        Console.WriteLine("\n=== Task Exception Handling ===");
        
        // Task that throws an exception
        Task<string> faultedTask = Task.Run(() => {
            Console.WriteLine("Task starting...");
            throw new InvalidOperationException("Something went wrong!");
        });
        
        try
        {
            string result = await faultedTask;
            Console.WriteLine($"Result: {result}");
        }
        catch (InvalidOperationException ex)
        {
            Console.WriteLine($"Caught exception: {ex.Message}");
        }
        
        // Check task status
        Console.WriteLine($"Task status: {faultedTask.Status}");
        Console.WriteLine($"Task faulted: {faultedTask.IsFaulted}");
        
        if (faultedTask.Exception != null)
        {
            Console.WriteLine($"Inner exception: {faultedTask.Exception.InnerException?.Message}");
        }
    }
}
```

**Expected Output:**
```
Task Exception Handling Demo

=== Task Exception Handling ===
Task starting...
Caught exception: Something went wrong!
Task status: Faulted
Task faulted: True
Inner exception: Something went wrong!
```

### Task Combinators

```csharp
using System;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Task Combinators Demo");
        
        await TaskCombinators();
    }
    
    static async Task TaskCombinators()
    {
        Console.WriteLine("\n=== Task Combinators ===");
        
        // Create multiple tasks
        var task1 = Task.Run(async () => {
            Console.WriteLine("Task 1: Starting...");
            await Task.Delay(1000);
            Console.WriteLine("Task 1: Completed");
            return "Result 1";
        });
        
        var task2 = Task.Run(async () => {
            Console.WriteLine("Task 2: Starting...");
            await Task.Delay(1500);
            Console.WriteLine("Task 2: Completed");
            return "Result 2";
        });
        
        var task3 = Task.Run(async () => {
            Console.WriteLine("Task 3: Starting...");
            await Task.Delay(800);
            Console.WriteLine("Task 3: Completed");
            return "Result 3";
        });
        
        // Task.WhenAll - wait for all tasks
        Console.WriteLine("Waiting for all tasks...");
        var allResults = await Task.WhenAll(task1, task2, task3);
        Console.WriteLine("All tasks completed!");
        foreach (var result in allResults)
        {
            Console.WriteLine($"- {result}");
        }
        
        // Task.WhenAny - wait for first task
        Console.WriteLine("\nWaiting for first task to complete...");
        var firstTask = await Task.WhenAny(task1, task2, task3);
        Console.WriteLine($"First completed: {await firstTask}");
    }
}
```

**Expected Output:**
```
Task Combinators Demo

=== Task Combinators ===
Task 1: Starting...
Task 2: Starting...
Task 3: Starting...
Waiting for all tasks...
Task 3: Completed
Task 1: Completed
Task 2: Completed
All tasks completed!
- Result 1
- Result 2
- Result 3

Waiting for first task to complete...
First completed: Result 3
```

### Task.Run vs async/await

```csharp
using System;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Task.Run vs async/await Demo");
        
        await TaskRunVsAsyncAwait();
    }
    
    static async Task TaskRunVsAsyncAwait()
    {
        Console.WriteLine("\n=== Task.Run vs async/await ===");
        
        // Using Task.Run for CPU-bound work
        Console.WriteLine("CPU-bound work with Task.Run:");
        int cpuResult = await Task.Run(() => {
            Console.WriteLine("CPU work: Calculating...");
            int sum = 0;
            for (int i = 0; i < 1000000; i++)
            {
                sum += i;
            }
            Console.WriteLine("CPU work: Completed");
            return sum;
        });
        Console.WriteLine($"CPU result: {cpuResult}");
        
        // Using async/await for I/O-bound work
        Console.WriteLine("\nI/O-bound work with async/await:");
        string ioResult = await SimulateIOOperation();
        Console.WriteLine($"I/O result: {ioResult}");
    }
    
    static async Task<string> SimulateIOOperation()
    {
        Console.WriteLine("I/O work: Starting...");
        await Task.Delay(1000); // Simulate I/O delay
        Console.WriteLine("I/O work: Completed");
        return "I/O operation completed";
    }
}
```

**Expected Output:**
```
Task.Run vs async/await Demo

=== Task.Run vs async/await ===
CPU-bound work with Task.Run:
CPU work: Calculating...
CPU work: Completed
CPU result: 499999500000

I/O-bound work with async/await:
I/O work: Starting...
I/O work: Completed
I/O result: I/O operation completed
```

## Key Points

1. **Task vs Task<T>**: Use Task for operations that don't return values, Task<T> for operations that do
2. **Task.Run**: Use for CPU-bound work that should run on a background thread
3. **async/await**: Use for I/O-bound operations that can release the thread
4. **TaskCompletionSource**: Use when you need manual control over task completion
5. **Exception Handling**: Always handle exceptions from tasks
6. **Task Combinators**: Use WhenAll/WhenAny for multiple tasks
7. **Performance**: Avoid unnecessary Task.Run for I/O operations
