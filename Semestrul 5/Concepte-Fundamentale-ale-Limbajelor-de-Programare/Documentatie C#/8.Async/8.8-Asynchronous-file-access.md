# 8.8 Asynchronous File Access

## Description

Asynchronous file access is essential for building responsive applications that need to read from or write to files without blocking the main thread. C# provides comprehensive async file I/O methods that allow you to perform file operations efficiently while maintaining application responsiveness. This topic covers reading, writing, copying, and managing files asynchronously.

## Key Concepts

- **Async file I/O methods**: `ReadAllTextAsync`, `WriteAllTextAsync`, `ReadAllBytesAsync`, etc.
- **Stream-based operations**: Using `FileStream` with async methods
- **File monitoring**: Watching for file changes asynchronously
- **Error handling**: Proper exception handling for file operations
- **Performance considerations**: Buffering and memory management

## Code Examples

### Basic File Reading and Writing

```csharp
using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Basic Asynchronous File Access Demo");
        
        await BasicFileOperations();
    }
    
    static async Task BasicFileOperations()
    {
        string fileName = "sample.txt";
        string content = "Hello, Async File World!\nThis is a test file for async operations.\nLine 3 of the file.";
        
        try
        {
            // Write to file asynchronously
            Console.WriteLine("Writing to file...");
            await File.WriteAllTextAsync(fileName, content);
            Console.WriteLine("✓ File written successfully");
            
            // Read from file asynchronously
            Console.WriteLine("Reading from file...");
            string readContent = await File.ReadAllTextAsync(fileName);
            Console.WriteLine("✓ File read successfully");
            Console.WriteLine($"Content:\n{readContent}");
            
            // Append to file asynchronously
            Console.WriteLine("Appending to file...");
            await File.AppendAllTextAsync(fileName, "\nThis line was appended asynchronously.");
            Console.WriteLine("✓ Content appended successfully");
            
            // Read updated content
            string updatedContent = await File.ReadAllTextAsync(fileName);
            Console.WriteLine("Updated content:");
            Console.WriteLine(updatedContent);
        }
        catch (Exception ex)
        {
            Console.WriteLine($"Error: {ex.Message}");
        }
        finally
        {
            // Clean up
            if (File.Exists(fileName))
            {
                File.Delete(fileName);
                Console.WriteLine("✓ Temporary file cleaned up");
            }
        }
    }
}
```

### Stream-Based File Operations

```csharp
using System;
using System.IO;
using System.Text;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Stream-Based File Operations Demo");
        
        await StreamBasedOperations();
    }
    
    static async Task StreamBasedOperations()
    {
        string fileName = "stream_data.txt";
        
        try
        {
            // Write using FileStream
            Console.WriteLine("Writing data using FileStream...");
            using (var fileStream = new FileStream(fileName, FileMode.Create, FileAccess.Write, FileShare.None, bufferSize: 4096, useAsync: true))
            using (var writer = new StreamWriter(fileStream, Encoding.UTF8))
            {
                await writer.WriteLineAsync("Line 1: This is written using FileStream");
                await writer.WriteLineAsync("Line 2: With async operations");
                await writer.WriteLineAsync("Line 3: And proper resource management");
                await writer.FlushAsync();
            }
            Console.WriteLine("✓ Data written using FileStream");
            
            // Read using FileStream
            Console.WriteLine("Reading data using FileStream...");
            using (var fileStream = new FileStream(fileName, FileMode.Open, FileAccess.Read, FileShare.Read, bufferSize: 4096, useAsync: true))
            using (var reader = new StreamReader(fileStream, Encoding.UTF8))
            {
                string line;
                int lineNumber = 1;
                while ((line = await reader.ReadLineAsync()) != null)
                {
                    Console.WriteLine($"  {lineNumber}: {line}");
                    lineNumber++;
                }
            }
            Console.WriteLine("✓ Data read using FileStream");
            
            // Copy file asynchronously
            string copyFileName = "stream_data_copy.txt";
            Console.WriteLine("Copying file asynchronously...");
            using (var sourceStream = new FileStream(fileName, FileMode.Open, FileAccess.Read, FileShare.Read, bufferSize: 4096, useAsync: true))
            using (var destinationStream = new FileStream(copyFileName, FileMode.Create, FileAccess.Write, FileShare.None, bufferSize: 4096, useAsync: true))
            {
                await sourceStream.CopyToAsync(destinationStream);
            }
            Console.WriteLine("✓ File copied successfully");
            
            // Verify copy
            string originalContent = await File.ReadAllTextAsync(fileName);
            string copyContent = await File.ReadAllTextAsync(copyFileName);
            Console.WriteLine($"Copy verification: {originalContent == copyContent}");
        }
        catch (Exception ex)
        {
            Console.WriteLine($"Error: {ex.Message}");
        }
        finally
        {
            // Clean up
            if (File.Exists(fileName)) File.Delete(fileName);
            if (File.Exists("stream_data_copy.txt")) File.Delete("stream_data_copy.txt");
            Console.WriteLine("✓ Files cleaned up");
        }
    }
}
```

```
  2: Line 2: With async operations
```

### Multiple File Operations

```csharp
using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Multiple File Operations Demo");
        
        await MultipleFileOperations();
    }
    
    static async Task MultipleFileOperations()
    {
        var fileNames = new[] { "file1.txt", "file2.txt", "file3.txt", "file4.txt" };
        var fileContents = new[]
        {
            "Content of file 1",
            "Content of file 2",
            "Content of file 3",
            "Content of file 4"
        };
        
        try
        {
            // Write multiple files concurrently
            Console.WriteLine("Writing multiple files concurrently...");
            var writeTasks = fileNames.Zip(fileContents, async (fileName, content) =>
            {
                Console.WriteLine($"  Writing {fileName}...");
                await File.WriteAllTextAsync(fileName, content);
                Console.WriteLine($"  ✓ {fileName} written");
                return fileName;
            }).ToArray();
            
            await Task.WhenAll(writeTasks);
            Console.WriteLine("✓ All files written successfully");
            
            // Read multiple files concurrently
            Console.WriteLine("\nReading multiple files concurrently...");
            var readTasks = fileNames.Select(async fileName =>
            {
                Console.WriteLine($"  Reading {fileName}...");
                string content = await File.ReadAllTextAsync(fileName);
                Console.WriteLine($"  ✓ {fileName} read: {content}");
                return new { FileName = fileName, Content = content };
            }).ToArray();
            
            var results = await Task.WhenAll(readTasks);
            Console.WriteLine("✓ All files read successfully");
            
            // Process all file contents
            Console.WriteLine("\nProcessing all file contents...");
            foreach (var result in results)
            {
                Console.WriteLine($"  {result.FileName}: {result.Content.Length} characters");
            }
            
            // Create a summary file
            Console.WriteLine("\nCreating summary file...");
            string summary = string.Join("\n", results.Select(r => $"{r.FileName}: {r.Content}"));
            await File.WriteAllTextAsync("summary.txt", summary);
            Console.WriteLine("✓ Summary file created");
            
            // Read summary file
            string summaryContent = await File.ReadAllTextAsync("summary.txt");
            Console.WriteLine("\nSummary file content:");
            Console.WriteLine(summaryContent);
        }
        catch (Exception ex)
        {
            Console.WriteLine($"Error: {ex.Message}");
        }
        finally
        {
            // Clean up all files
            var allFiles = fileNames.Concat(new[] { "summary.txt" });
            foreach (var fileName in allFiles)
            {
                if (File.Exists(fileName))
                {
                    File.Delete(fileName);
                    Console.WriteLine($"✓ {fileName} cleaned up");
                }
            }
        }
    }
}
```

### File Monitoring and Change Detection

```csharp
using System;
using System.IO;
using System.Threading;
using System.Threading.Tasks;

class Program
{
    private static FileSystemWatcher _watcher;
    private static CancellationTokenSource _cancellationTokenSource;
    
    static async Task Main(string[] args)
    {
        Console.WriteLine("File Monitoring Demo");
        
        await FileMonitoringDemo();
    }
    
    static async Task FileMonitoringDemo()
    {
        string fileName = "monitored_file.txt";
        _cancellationTokenSource = new CancellationTokenSource();
        
        try
        {
            // Start file monitoring
            Console.WriteLine("Starting file monitoring...");
            var monitoringTask = StartFileMonitoringAsync(fileName, _cancellationTokenSource.Token);
            
            // Create and modify the file
            Console.WriteLine("Creating and modifying file...");
            await File.WriteAllTextAsync(fileName, "Initial content");
            await Task.Delay(1000);
            
            await File.AppendAllTextAsync(fileName, "\nFirst modification");
            await Task.Delay(1000);
            
            await File.WriteAllTextAsync(fileName, "Completely new content");
            await Task.Delay(1000);
            
            // Stop monitoring
            Console.WriteLine("Stopping file monitoring...");
            _cancellationTokenSource.Cancel();
            
            await monitoringTask;
        }
        catch (OperationCanceledException)
        {
            Console.WriteLine("File monitoring stopped.");
        }
        catch (Exception ex)
        {
            Console.WriteLine($"Error: {ex.Message}");
        }
        finally
        {
            _watcher?.Dispose();
            _cancellationTokenSource?.Dispose();
            
            if (File.Exists(fileName))
            {
                File.Delete(fileName);
                Console.WriteLine("✓ Monitored file cleaned up");
            }
        }
    }
    
    static async Task StartFileMonitoringAsync(string fileName, CancellationToken cancellationToken)
    {
        string directory = Path.GetDirectoryName(Path.GetFullPath(fileName));
        string filePattern = Path.GetFileName(fileName);
        
        _watcher = new FileSystemWatcher(directory, filePattern)
        {
            EnableRaisingEvents = true,
            NotifyFilter = NotifyFilters.LastWrite | NotifyFilters.Size
        };
        
        var tcs = new TaskCompletionSource<bool>();
        
        _watcher.Changed += async (sender, e) =>
        {
            Console.WriteLine($"📁 File changed: {e.Name} at {DateTime.Now:HH:mm:ss}");
            
            try
            {
                // Read the file content asynchronously
                string content = await File.ReadAllTextAsync(fileName);
                Console.WriteLine($"   Content preview: {content.Substring(0, Math.Min(50, content.Length))}...");
            }
            catch (Exception ex)
            {
                Console.WriteLine($"   Error reading file: {ex.Message}");
            }
        };
        
        _watcher.Error += (sender, e) =>
        {
            Console.WriteLine($"❌ File watcher error: {e.GetException().Message}");
            tcs.SetException(e.GetException());
        };
        
        cancellationToken.Register(() =>
        {
            _watcher.EnableRaisingEvents = false;
            tcs.SetResult(true);
        });
        
        await tcs.Task;
    }
}
```

### Error Handling and Recovery

```csharp
using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Error Handling and Recovery Demo");
        
        await ErrorHandlingDemo();
    }
    
    static async Task ErrorHandlingDemo()
    {
        string fileName = "test_file.txt";
        string backupFileName = "test_file_backup.txt";
        
        try
        {
            // Create initial file
            Console.WriteLine("Creating initial file...");
            await File.WriteAllTextAsync(fileName, "Initial content");
            Console.WriteLine("✓ File created successfully");
            
            // Demonstrate various error scenarios
            await DemonstrateFileErrors(fileName);
            
            // Demonstrate recovery strategies
            await DemonstrateRecoveryStrategies(fileName, backupFileName);
        }
        catch (Exception ex)
        {
            Console.WriteLine($"Unexpected error: {ex.Message}");
        }
        finally
        {
            // Clean up
            if (File.Exists(fileName)) File.Delete(fileName);
            if (File.Exists(backupFileName)) File.Delete(backupFileName);
            Console.WriteLine("✓ Files cleaned up");
        }
    }
    
    static async Task DemonstrateFileErrors(string fileName)
    {
        Console.WriteLine("\n=== Demonstrating File Errors ===");
        
        // Error 1: File not found
        try
        {
            await File.ReadAllTextAsync("nonexistent_file.txt");
        }
        catch (FileNotFoundException ex)
        {
            Console.WriteLine($"✓ Caught FileNotFoundException: {ex.Message}");
        }
        
        // Error 2: Access denied (simulate by trying to read a directory)
        try
        {
            await File.ReadAllTextAsync(".");
        }
        catch (UnauthorizedAccessException ex)
        {
            Console.WriteLine($"✓ Caught UnauthorizedAccessException: {ex.Message}");
        }
        catch (ArgumentException ex)
        {
            Console.WriteLine($"✓ Caught ArgumentException: {ex.Message}");
        }
        
        // Error 3: Path too long
        try
        {
            string longPath = new string('a', 300) + ".txt";
            await File.WriteAllTextAsync(longPath, "test");
        }
        catch (PathTooLongException ex)
        {
            Console.WriteLine($"✓ Caught PathTooLongException: {ex.Message}");
        }
    }
    
    static async Task DemonstrateRecoveryStrategies(string fileName, string backupFileName)
    {
        Console.WriteLine("\n=== Demonstrating Recovery Strategies ===");
        
        // Strategy 1: Retry with exponential backoff
        await RetryWithBackoff(fileName);
        
        // Strategy 2: Fallback to backup
        await FallbackToBackup(fileName, backupFileName);
        
        // Strategy 3: Graceful degradation
        await GracefulDegradation(fileName);
    }
    
    static async Task RetryWithBackoff(string fileName)
    {
        Console.WriteLine("Retry with exponential backoff:");
        
        int maxRetries = 3;
        int baseDelay = 100;
        
        for (int attempt = 1; attempt <= maxRetries; attempt++)
        {
            try
            {
                // Simulate a potentially failing operation
                if (attempt < maxRetries && new Random().Next(0, 2) == 0)
                {
                    throw new IOException("Simulated I/O error");
                }
                
                string content = await File.ReadAllTextAsync(fileName);
                Console.WriteLine($"  ✓ Success on attempt {attempt}: {content.Length} characters read");
                break;
            }
            catch (IOException ex)
            {
                Console.WriteLine($"  ✗ Attempt {attempt} failed: {ex.Message}");
                
                if (attempt < maxRetries)
                {
                    int delay = baseDelay * (int)Math.Pow(2, attempt - 1);
                    Console.WriteLine($"  ⏳ Waiting {delay}ms before retry...");
                    await Task.Delay(delay);
                }
                else
                {
                    Console.WriteLine("  ❌ All retry attempts failed");
                }
            }
        }
    }
    
    static async Task FallbackToBackup(string fileName, string backupFileName)
    {
        Console.WriteLine("Fallback to backup:");
        
        try
        {
            // Try to read the main file
            string content = await File.ReadAllTextAsync(fileName);
            Console.WriteLine($"  ✓ Read from main file: {content}");
        }
        catch (Exception ex)
        {
            Console.WriteLine($"  ✗ Main file failed: {ex.Message}");
            
            try
            {
                // Create backup if it doesn't exist
                if (!File.Exists(backupFileName))
                {
                    await File.WriteAllTextAsync(backupFileName, "Backup content");
                    Console.WriteLine("  📁 Created backup file");
                }
                
                // Read from backup
                string backupContent = await File.ReadAllTextAsync(backupFileName);
                Console.WriteLine($"  ✓ Read from backup: {backupContent}");
            }
            catch (Exception backupEx)
            {
                Console.WriteLine($"  ❌ Backup also failed: {backupEx.Message}");
            }
        }
    }
    
    static async Task GracefulDegradation(string fileName)
    {
        Console.WriteLine("Graceful degradation:");
        
        try
        {
            // Try to read the full file
            string fullContent = await File.ReadAllTextAsync(fileName);
            Console.WriteLine($"  ✓ Full content read: {fullContent}");
        }
        catch (Exception ex)
        {
            Console.WriteLine($"  ✗ Full read failed: {ex.Message}");
            
            try
            {
                // Try to read just the first few bytes
                using var fileStream = new FileStream(fileName, FileMode.Open, FileAccess.Read, FileShare.Read, bufferSize: 1, useAsync: true);
                byte[] buffer = new byte[10];
                int bytesRead = await fileStream.ReadAsync(buffer, 0, buffer.Length);
                string partialContent = System.Text.Encoding.UTF8.GetString(buffer, 0, bytesRead);
                Console.WriteLine($"  ⚠ Partial content read: {partialContent}");
            }
            catch (Exception partialEx)
            {
                Console.WriteLine($"  ❌ Even partial read failed: {partialEx.Message}");
                Console.WriteLine("  🔄 Using default content");
            }
        }
    }
}
```

## Best Practices

1. **Use async file methods**: Always prefer async versions for I/O operations
2. **Handle exceptions properly**: Wrap file operations in try-catch blocks
3. **Use using statements**: Ensure proper disposal of file streams
4. **Implement retry logic**: For transient failures
5. **Monitor file changes**: Use FileSystemWatcher for real-time updates
6. **Consider performance**: Use appropriate buffer sizes and async operations
7. **Clean up resources**: Always dispose of streams and watchers
8. **Test error scenarios**: Ensure your code handles various failure modes
