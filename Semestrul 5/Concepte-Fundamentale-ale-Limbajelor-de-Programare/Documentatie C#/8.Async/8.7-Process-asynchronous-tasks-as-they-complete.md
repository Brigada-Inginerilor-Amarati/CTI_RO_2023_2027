# 8.7 Process Asynchronous Tasks as They Complete

## Description

When working with multiple asynchronous tasks, you often want to process results as soon as they become available, rather than waiting for all tasks to complete. This pattern improves responsiveness and allows for better resource utilization. C# provides several approaches to achieve this, including `Task.WhenAny()`, `TaskCompletionSource`, and custom task processing patterns.

## Key Concepts

- **Task.WhenAny()**: Returns the first task that completes
- **Processing as completed**: Handle results as they become available
- **Task completion order**: Tasks may complete in different order than started
- **Resource management**: Efficient handling of multiple concurrent operations

## Code Examples

### Basic Task Processing as Completed

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Process Tasks as They Complete Demo");
        
        await ProcessTasksAsCompleted();
    }
    
    static async Task ProcessTasksAsCompleted()
    {
        // Create tasks with different completion times
        var tasks = new[]
        {
            CreateTaskAsync("Task 1", 2000),
            CreateTaskAsync("Task 2", 1000),
            CreateTaskAsync("Task 3", 3000),
            CreateTaskAsync("Task 4", 500),
            CreateTaskAsync("Task 5", 1500)
        };
        
        Console.WriteLine("All tasks started. Processing as they complete...\n");
        
        // Process tasks as they complete
        var remainingTasks = tasks.ToList();
        
        while (remainingTasks.Count > 0)
        {
            var completedTask = await Task.WhenAny(remainingTasks);
            remainingTasks.Remove(completedTask);
            
            string result = await completedTask;
            Console.WriteLine($"✓ {result} completed!");
        }
        
        Console.WriteLine("\nAll tasks have been processed!");
    }
    
    static async Task<string> CreateTaskAsync(string taskName, int delayMs)
    {
        Console.WriteLine($"{taskName}: Started");
        await Task.Delay(delayMs);
        Console.WriteLine($"{taskName}: Finished");
        return $"{taskName} result";
    }
}
```

### Real-World Example: Data Processing Pipeline

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Data Processing Pipeline Demo");
        
        await DataProcessingPipeline();
    }
    
    static async Task DataProcessingPipeline()
    {
        // Simulate different data sources
        var dataSources = new[]
        {
            "Database",
            "API",
            "File System",
            "Cache",
            "Message Queue"
        };
        
        // Start all data collection tasks
        var dataTasks = dataSources.Select(source => 
            CollectDataAsync(source, GetRandomDelay())).ToArray();
        
        Console.WriteLine("Data collection started from all sources...\n");
        
        // Process data as it becomes available
        var remainingTasks = dataTasks.ToList();
        var processedData = new List<DataResult>();
        
        while (remainingTasks.Count > 0)
        {
            var completedTask = await Task.WhenAny(remainingTasks);
            remainingTasks.Remove(completedTask);
            
            var data = await completedTask;
            processedData.Add(data);
            
            Console.WriteLine($"✓ Data received from {data.Source}: {data.Records} records");
            
            // Process the data immediately
            await ProcessDataAsync(data);
        }
        
        Console.WriteLine($"\nAll data processed! Total sources: {processedData.Count}");
        Console.WriteLine($"Total records: {processedData.Sum(d => d.Records)}");
    }
    
    static async Task<DataResult> CollectDataAsync(string source, int delayMs)
    {
        Console.WriteLine($"{source}: Starting data collection...");
        await Task.Delay(delayMs);
        
        var random = new Random();
        int recordCount = random.Next(10, 100);
        
        Console.WriteLine($"{source}: Collected {recordCount} records");
        return new DataResult
        {
            Source = source,
            Records = recordCount,
            Timestamp = DateTime.Now
        };
    }
    
    static async Task ProcessDataAsync(DataResult data)
    {
        Console.WriteLine($"  → Processing {data.Source} data...");
        await Task.Delay(200); // Simulate processing time
        Console.WriteLine($"  ✓ {data.Source} data processed successfully");
    }
    
    static int GetRandomDelay()
    {
        var random = new Random();
        return random.Next(500, 3000);
    }
}

class DataResult
{
    public string Source { get; set; }
    public int Records { get; set; }
    public DateTime Timestamp { get; set; }
}
```

### Advanced Pattern: Task Processing with Priority

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Priority-Based Task Processing Demo");
        
        await PriorityBasedProcessing();
    }
    
    static async Task PriorityBasedProcessing()
    {
        // Create tasks with different priorities
        var tasks = new[]
        {
            new PriorityTask("High Priority Task", 1, 2000),
            new PriorityTask("Medium Priority Task 1", 2, 1500),
            new PriorityTask("Low Priority Task", 3, 1000),
            new PriorityTask("High Priority Task 2", 1, 1800),
            new PriorityTask("Medium Priority Task 2", 2, 1200)
        };
        
        // Start all tasks
        var runningTasks = tasks.Select(t => 
            ExecuteTaskAsync(t)).ToList();
        
        Console.WriteLine("All tasks started. Processing by priority as they complete...\n");
        
        // Process tasks as they complete, but prioritize by priority level
        var completedTasks = new List<PriorityTask>();
        
        while (runningTasks.Count > 0)
        {
            var completedTask = await Task.WhenAny(runningTasks);
            runningTasks.Remove(completedTask);
            
            var taskResult = await completedTask;
            completedTasks.Add(taskResult);
            
            Console.WriteLine($"✓ {taskResult.Name} completed (Priority: {taskResult.Priority})");
            
            // Process high priority tasks immediately
            if (taskResult.Priority == 1)
            {
                await ProcessHighPriorityTask(taskResult);
            }
        }
        
        // Process remaining tasks by priority
        var remainingTasks = completedTasks
            .Where(t => t.Priority > 1)
            .OrderBy(t => t.Priority)
            .ToList();
        
        Console.WriteLine("\nProcessing remaining tasks by priority...");
        foreach (var task in remainingTasks)
        {
            await ProcessTask(task);
        }
    }
    
    static async Task<PriorityTask> ExecuteTaskAsync(PriorityTask task)
    {
        Console.WriteLine($"{task.Name}: Started");
        await Task.Delay(task.DurationMs);
        Console.WriteLine($"{task.Name}: Completed");
        return task;
    }
    
    static async Task ProcessHighPriorityTask(PriorityTask task)
    {
        Console.WriteLine($"  → Processing HIGH PRIORITY: {task.Name}");
        await Task.Delay(100);
        Console.WriteLine($"  ✓ High priority processing completed for {task.Name}");
    }
    
    static async Task ProcessTask(PriorityTask task)
    {
        Console.WriteLine($"  → Processing: {task.Name} (Priority: {task.Priority})");
        await Task.Delay(100);
        Console.WriteLine($"  ✓ Processing completed for {task.Name}");
    }
}

class PriorityTask
{
    public string Name { get; set; }
    public int Priority { get; set; }
    public int DurationMs { get; set; }
    
    public PriorityTask(string name, int priority, int durationMs)
    {
        Name = name;
        Priority = priority;
        DurationMs = durationMs;
    }
}
```

### Error Handling in Task Processing

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Error Handling in Task Processing Demo");
        
        await ErrorHandlingDemo();
    }
    
    static async Task ErrorHandlingDemo()
    {
        // Create tasks with some that will fail
        var tasks = new[]
        {
            CreateTaskAsync("Success Task 1", 1000, false),
            CreateTaskAsync("Failure Task", 1500, true),
            CreateTaskAsync("Success Task 2", 800, false),
            CreateTaskAsync("Another Failure", 1200, true),
            CreateTaskAsync("Success Task 3", 2000, false)
        };
        
        var runningTasks = tasks.ToList();
        var results = new List<TaskResult>();
        
        Console.WriteLine("Processing tasks with error handling...\n");
        
        while (runningTasks.Count > 0)
        {
            var completedTask = await Task.WhenAny(runningTasks);
            runningTasks.Remove(completedTask);
            
            try
            {
                var result = await completedTask;
                results.Add(new TaskResult { Name = result, Success = true });
                Console.WriteLine($"✓ {result} completed successfully");
            }
            catch (Exception ex)
            {
                results.Add(new TaskResult { Name = "Unknown", Success = false, Error = ex.Message });
                Console.WriteLine($"✗ Task failed: {ex.Message}");
            }
        }
        
        // Summary
        Console.WriteLine($"\nProcessing Summary:");
        Console.WriteLine($"Successful: {results.Count(r => r.Success)}");
        Console.WriteLine($"Failed: {results.Count(r => !r.Success)}");
        
        // Retry failed tasks
        var failedTasks = results.Where(r => !r.Success).ToList();
        if (failedTasks.Any())
        {
            Console.WriteLine("\nRetrying failed tasks...");
            await RetryFailedTasks(failedTasks);
        }
    }
    
    static async Task<string> CreateTaskAsync(string name, int delayMs, bool shouldFail)
    {
        Console.WriteLine($"{name}: Started");
        await Task.Delay(delayMs);
        
        if (shouldFail)
        {
            Console.WriteLine($"{name}: Failed");
            throw new InvalidOperationException($"{name} encountered an error");
        }
        
        Console.WriteLine($"{name}: Completed");
        return name;
    }
    
    static async Task RetryFailedTasks(List<TaskResult> failedTasks)
    {
        foreach (var failedTask in failedTasks)
        {
            try
            {
                Console.WriteLine($"Retrying {failedTask.Name}...");
                await Task.Delay(1000); // Simulate retry
                Console.WriteLine($"✓ {failedTask.Name} retry successful");
            }
            catch (Exception ex)
            {
                Console.WriteLine($"✗ {failedTask.Name} retry failed: {ex.Message}");
            }
        }
    }
}

class TaskResult
{
    public string Name { get; set; }
    public bool Success { get; set; }
    public string Error { get; set; }
}
```

## Best Practices

1. **Use Task.WhenAny()**: For processing tasks as they complete
2. **Handle exceptions**: Always wrap task processing in try-catch
3. **Consider priority**: Process high-priority results first
4. **Implement retry logic**: For failed operations
5. **Monitor resource usage**: Don't start too many concurrent tasks
6. **Use appropriate data structures**: Lists or queues for managing tasks
7. **Provide progress feedback**: Keep users informed of completion status
8. **Clean up resources**: Properly dispose of completed tasks
