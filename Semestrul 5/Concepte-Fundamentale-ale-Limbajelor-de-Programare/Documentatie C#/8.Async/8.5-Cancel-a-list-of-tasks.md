# 8.5 Cancel a List of Tasks

## Description

Cancellation is a crucial aspect of asynchronous programming that allows you to stop long-running operations gracefully. When working with multiple tasks, you need to coordinate cancellation across all of them. This topic covers how to cancel a collection of tasks using `CancellationToken` and `CancellationTokenSource`.

## Key Concepts

- **CancellationToken**: A token that can be used to signal cancellation
- **CancellationTokenSource**: Creates and controls cancellation tokens
- **Task cancellation**: How to make tasks cancellable
- **Coordinated cancellation**: Canceling multiple tasks together

## Code Examples

### Basic Task Cancellation

```csharp
using System;
using System.Threading;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Basic Task Cancellation Demo");
        
        await BasicCancellationDemo();
    }
    
    static async Task BasicCancellationDemo()
    {
        // Create a cancellation token source
        using var cts = new CancellationTokenSource();
        
        // Start a cancellable task
        var task = LongRunningTaskAsync(cts.Token);
        
        // Cancel after 3 seconds
        cts.CancelAfter(TimeSpan.FromSeconds(3));
        
        try
        {
            await task;
            Console.WriteLine("Task completed successfully.");
        }
        catch (OperationCanceledException)
        {
            Console.WriteLine("Task was cancelled.");
        }
    }
    
    static async Task LongRunningTaskAsync(CancellationToken cancellationToken)
    {
        Console.WriteLine("Long running task started...");
        
        for (int i = 0; i < 10; i++)
        {
            // Check for cancellation before each iteration
            cancellationToken.ThrowIfCancellationRequested();
            
            Console.WriteLine($"Working... {i + 1}/10");
            await Task.Delay(1000, cancellationToken);
        }
        
        Console.WriteLine("Long running task completed!");
    }
}
```

**Expected Output:**
```
Basic Task Cancellation Demo
Long running task started...
Working... 1/10
Working... 2/10
Working... 3/10
Task was cancelled.
```

### Canceling Multiple Tasks

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Cancel Multiple Tasks Demo");
        
        await CancelMultipleTasksDemo();
    }
    
    static async Task CancelMultipleTasksDemo()
    {
        // Create a cancellation token source
        using var cts = new CancellationTokenSource();
        
        // Create multiple tasks
        var tasks = new List<Task>
        {
            WorkerTaskAsync("Worker 1", 5, cts.Token),
            WorkerTaskAsync("Worker 2", 8, cts.Token),
            WorkerTaskAsync("Worker 3", 3, cts.Token),
            WorkerTaskAsync("Worker 4", 10, cts.Token)
        };
        
        Console.WriteLine("All tasks started. Will cancel in 4 seconds...");
        
        // Cancel after 4 seconds
        cts.CancelAfter(TimeSpan.FromSeconds(4));
        
        try
        {
            // Wait for all tasks to complete or be cancelled
            await Task.WhenAll(tasks);
            Console.WriteLine("All tasks completed successfully.");
        }
        catch (OperationCanceledException)
        {
            Console.WriteLine("Some tasks were cancelled.");
        }
        
        // Check the status of each task
        Console.WriteLine("\nTask Status:");
        for (int i = 0; i < tasks.Count; i++)
        {
            var task = tasks[i];
            Console.WriteLine($"Task {i + 1}: {task.Status}");
        }
    }
    
    static async Task WorkerTaskAsync(string name, int iterations, CancellationToken cancellationToken)
    {
        Console.WriteLine($"{name}: Started");
        
        try
        {
            for (int i = 0; i < iterations; i++)
            {
                cancellationToken.ThrowIfCancellationRequested();
                
                Console.WriteLine($"{name}: Working... {i + 1}/{iterations}");
                await Task.Delay(1000, cancellationToken);
            }
            
            Console.WriteLine($"{name}: Completed successfully!");
        }
        catch (OperationCanceledException)
        {
            Console.WriteLine($"{name}: Was cancelled.");
            throw; // Re-throw to propagate cancellation
        }
    }
}
```

### Canceling Tasks with Different Strategies

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Different Cancellation Strategies Demo");
        
        await DifferentCancellationStrategies();
    }
    
    static async Task DifferentCancellationStrategies()
    {
        // Strategy 1: Cancel all tasks when any task completes
        Console.WriteLine("=== Strategy 1: Cancel all when first completes ===");
        await CancelAllWhenFirstCompletes();
        
        Console.WriteLine("\n=== Strategy 2: Cancel after timeout ===");
        await CancelAfterTimeout();
        
        Console.WriteLine("\n=== Strategy 3: Cancel on user input ===");
        await CancelOnUserInput();
    }
    
    static async Task CancelAllWhenFirstCompletes()
    {
        using var cts = new CancellationTokenSource();
        
        var tasks = new[]
        {
            WorkerTaskAsync("Fast Worker", 2, cts.Token),
            WorkerTaskAsync("Slow Worker 1", 5, cts.Token),
            WorkerTaskAsync("Slow Worker 2", 8, cts.Token)
        };
        
        // Cancel all when the first task completes
        var completedTask = await Task.WhenAny(tasks);
        cts.Cancel();
        
        Console.WriteLine($"First task completed: {completedTask.Status}");
        
        // Wait for all tasks to finish (cancelled or completed)
        try
        {
            await Task.WhenAll(tasks);
        }
        catch (OperationCanceledException)
        {
            Console.WriteLine("Remaining tasks were cancelled.");
        }
    }
    
    static async Task CancelAfterTimeout()
    {
        using var cts = new CancellationTokenSource();
        
        var tasks = new[]
        {
            WorkerTaskAsync("Worker A", 3, cts.Token),
            WorkerTaskAsync("Worker B", 6, cts.Token),
            WorkerTaskAsync("Worker C", 4, cts.Token)
        };
        
        // Cancel after 2.5 seconds
        cts.CancelAfter(TimeSpan.FromSeconds(2.5));
        
        try
        {
            await Task.WhenAll(tasks);
            Console.WriteLine("All tasks completed before timeout.");
        }
        catch (OperationCanceledException)
        {
            Console.WriteLine("Tasks were cancelled due to timeout.");
        }
    }
    
    static async Task CancelOnUserInput()
    {
        using var cts = new CancellationTokenSource();
        
        var tasks = new[]
        {
            WorkerTaskAsync("Background Worker", 10, cts.Token)
        };
        
        // Simulate user input after 2 seconds
        _ = Task.Run(async () =>
        {
            await Task.Delay(2000);
            Console.WriteLine("Simulating user cancellation...");
            cts.Cancel();
        });
        
        try
        {
            await Task.WhenAll(tasks);
            Console.WriteLine("Task completed before user cancellation.");
        }
        catch (OperationCanceledException)
        {
            Console.WriteLine("Task was cancelled by user input.");
        }
    }
    
    static async Task WorkerTaskAsync(string name, int iterations, CancellationToken cancellationToken)
    {
        Console.WriteLine($"{name}: Started");
        
        try
        {
            for (int i = 0; i < iterations; i++)
            {
                cancellationToken.ThrowIfCancellationRequested();
                
                Console.WriteLine($"{name}: Working... {i + 1}/{iterations}");
                await Task.Delay(500, cancellationToken);
            }
            
            Console.WriteLine($"{name}: Completed successfully!");
        }
        catch (OperationCanceledException)
        {
            Console.WriteLine($"{name}: Was cancelled.");
            throw;
        }
    }
}
```

### Advanced Cancellation with Cleanup

```csharp
using System;
using System.Collections.Generic;
using System.Threading;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        Console.WriteLine("Advanced Cancellation with Cleanup Demo");
        
        await AdvancedCancellationDemo();
    }
    
    static async Task AdvancedCancellationDemo()
    {
        using var cts = new CancellationTokenSource();
        
        var tasks = new List<Task>
        {
            ProcessDataWithCleanupAsync("Processor 1", cts.Token),
            ProcessDataWithCleanupAsync("Processor 2", cts.Token),
            ProcessDataWithCleanupAsync("Processor 3", cts.Token)
        };
        
        // Cancel after 3 seconds
        cts.CancelAfter(TimeSpan.FromSeconds(3));
        
        try
        {
            await Task.WhenAll(tasks);
            Console.WriteLine("All processors completed successfully.");
        }
        catch (OperationCanceledException)
        {
            Console.WriteLine("Some processors were cancelled.");
        }
    }
    
    static async Task ProcessDataWithCleanupAsync(string processorName, CancellationToken cancellationToken)
    {
        var resources = new List<string>();
        
        try
        {
            Console.WriteLine($"{processorName}: Starting processing...");
            
            for (int i = 0; i < 10; i++)
            {
                cancellationToken.ThrowIfCancellationRequested();
                
                // Simulate resource allocation
                string resource = $"Resource_{i}";
                resources.Add(resource);
                Console.WriteLine($"{processorName}: Allocated {resource}");
                
                await Task.Delay(500, cancellationToken);
            }
            
            Console.WriteLine($"{processorName}: Processing completed successfully!");
        }
        catch (OperationCanceledException)
        {
            Console.WriteLine($"{processorName}: Cancellation requested, cleaning up...");
            
            // Cleanup allocated resources
            foreach (var resource in resources)
            {
                Console.WriteLine($"{processorName}: Cleaning up {resource}");
                await Task.Delay(100); // Simulate cleanup time
            }
            
            Console.WriteLine($"{processorName}: Cleanup completed.");
            throw; // Re-throw to propagate cancellation
        }
    }
}
```

## Best Practices

1. **Always use CancellationToken**: Make your async methods cancellable
2. **Check cancellation frequently**: Call `ThrowIfCancellationRequested()` in loops
3. **Handle cleanup**: Clean up resources when cancellation occurs
4. **Use using statements**: Properly dispose of CancellationTokenSource
5. **Propagate cancellation**: Re-throw OperationCanceledException to notify callers
6. **Coordinate cancellation**: Use the same token for related tasks
7. **Test cancellation**: Ensure your code handles cancellation gracefully
