# Reference Parameters in C#

## Description

Reference parameters allow methods to directly access and modify the caller's variables. Instead of passing a copy of the value, you pass a reference to the actual variable. C# provides several keywords for reference parameter passing: `ref`, `out`, and `in`.

### Reference Parameter Types:

#### **ref** (Reference Parameter)
- Variable must be initialized before passing
- Method can read and modify the value
- Changes affect the original variable
- Bidirectional parameter (in and out)

#### **out** (Output Parameter)
- Variable doesn't need to be initialized before passing
- Method must assign a value before returning
- Used for returning multiple values
- Unidirectional parameter (out only)

#### **in** (Input Reference Parameter - C# 7.2+)
- Variable is passed by reference but cannot be modified
- Read-only reference
- Performance optimization for large structs
- Prevents defensive copies

## Code Examples

### Example 1: ref Parameters - Basic Usage

```csharp
using System;

class Program
{
    static void Increment(ref int value)
    {
        Console.WriteLine($"  Inside Increment (before): {value}");
        value++;
        Console.WriteLine($"  Inside Increment (after): {value}");
    }

    static void Multiply(ref int value, int multiplier)
    {
        value *= multiplier;
    }

    static void Swap(ref int a, ref int b)
    {
        int temp = a;
        a = b;
        b = temp;
    }

    static void Main()
    {
        int number = 10;
        Console.WriteLine($"Original: {number}");

        Increment(ref number);
        Console.WriteLine($"After Increment: {number}\n");

        Multiply(ref number, 5);
        Console.WriteLine($"After Multiply by 5: {number}\n");

        int x = 100, y = 200;
        Console.WriteLine($"Before swap: x={x}, y={y}");
        Swap(ref x, ref y);
        Console.WriteLine($"After swap: x={x}, y={y}");
    }
}
```

**Output:**
```
Original: 10
  Inside Increment (before): 10
  Inside Increment (after): 11
After Increment: 11

After Multiply by 5: 55

Before swap: x=100, y=200
After swap: x=200, y=100
```

### Example 2: out Parameters - Multiple Return Values

```csharp
using System;

class Program
{
    static void GetMinMax(int[] array, out int min, out int max)
    {
        // Must assign out parameters
        min = array[0];
        max = array[0];

        foreach (int value in array)
        {
            if (value < min) min = value;
            if (value > max) max = value;
        }
    }

    static bool TryDivide(int dividend, int divisor, out double result)
    {
        if (divisor == 0)
        {
            result = 0; // Must assign even on failure
            return false;
        }

        result = (double)dividend / divisor;
        return true;
    }

    static void ParseName(string fullName, out string firstName, out string lastName)
    {
        string[] parts = fullName.Split(' ');

        if (parts.Length >= 2)
        {
            firstName = parts[0];
            lastName = parts[parts.Length - 1];
        }
        else
        {
            firstName = fullName;
            lastName = "";
        }
    }

    static void Main()
    {
        // GetMinMax example
        int[] numbers = { 5, 2, 9, 1, 7, 3 };
        GetMinMax(numbers, out int min, out int max);
        Console.WriteLine($"Array: [{string.Join(", ", numbers)}]");
        Console.WriteLine($"Min: {min}, Max: {max}\n");

        // TryDivide example
        if (TryDivide(10, 3, out double result))
        {
            Console.WriteLine($"10 / 3 = {result:F2}");
        }

        if (!TryDivide(10, 0, out result))
        {
            Console.WriteLine("Division by zero!\n");
        }

        // ParseName example
        ParseName("John Doe", out string first, out string last);
        Console.WriteLine($"First: {first}, Last: {last}");
    }
}
```

**Output:**
```
Array: [5, 2, 9, 1, 7, 3]
Min: 1, Max: 9

10 / 3 = 3.33
Division by zero!

First: John, Last: Doe
```

### Example 3: out Variables with Inline Declaration

```csharp
using System;

class Program
{
    static bool TryParseCoordinate(string input, out int x, out int y)
    {
        string[] parts = input.Split(',');

        if (parts.Length == 2 &&
            int.TryParse(parts[0].Trim(), out x) &&
            int.TryParse(parts[1].Trim(), out y))
        {
            return true;
        }

        x = 0;
        y = 0;
        return false;
    }

    static void Main()
    {
        // Inline out variable declaration (C# 7.0+)
        string input1 = "10, 20";
        if (TryParseCoordinate(input1, out int x1, out int y1))
        {
            Console.WriteLine($"Parsed: ({x1}, {y1})");
        }

        // Using int.TryParse with inline declaration
        string numberStr = "42";
        if (int.TryParse(numberStr, out int number))
        {
            Console.WriteLine($"Parsed number: {number}");
        }

        // Discard unwanted out parameters with _
        if (TryParseCoordinate("5, 15", out int x2, out _))
        {
            Console.WriteLine($"X coordinate: {x2}");
        }

        // Failed parsing
        if (!TryParseCoordinate("invalid", out int x3, out int y3))
        {
            Console.WriteLine($"Parse failed, defaults: ({x3}, {y3})");
        }
    }
}
```

**Output:**
```
Parsed: (10, 20)
Parsed number: 42
X coordinate: 5
Parse failed, defaults: (0, 0)
```

### Example 4: in Parameters - Read-Only References

```csharp
using System;

struct LargeStruct
{
    public int A, B, C, D, E;
    public double X, Y, Z;

    public LargeStruct(int value)
    {
        A = B = C = D = E = value;
        X = Y = Z = value;
    }

    public override string ToString() => $"LargeStruct({A})";
}

class Program
{
    // By value - creates copy (slower)
    static void ProcessByValue(LargeStruct data)
    {
        Console.WriteLine($"  By value: {data}");
    }

    // By in - read-only reference (faster, no copy)
    static void ProcessByIn(in LargeStruct data)
    {
        Console.WriteLine($"  By in: {data}");
        // data.A = 100; // ERROR: Cannot modify in parameter
    }

    // in parameter with computed value
    static double Calculate(in LargeStruct data)
    {
        return data.A + data.B + data.C + data.X + data.Y;
    }

    static void Main()
    {
        LargeStruct bigData = new LargeStruct(42);

        Console.WriteLine("Processing large struct:");
        ProcessByValue(bigData);
        ProcessByIn(in bigData);

        double result = Calculate(in bigData);
        Console.WriteLine($"\nCalculation result: {result}");

        // Can omit 'in' at call site
        ProcessByIn(bigData); // Implicit 'in'
    }
}
```

**Output:**
```
Processing large struct:
  By value: LargeStruct(42)
  By in: LargeStruct(42)

Calculation result: 252

  By in: LargeStruct(42)
```

### Example 5: Comparing ref, out, and in

```csharp
using System;

class Program
{
    // ref: modify existing value
    static void IncrementByRef(ref int value)
    {
        value++;
        Console.WriteLine($"  IncrementByRef result: {value}");
    }

    // out: return new value
    static void InitializeByOut(out int value)
    {
        // Must assign before using
        value = 100;
        Console.WriteLine($"  InitializeByOut result: {value}");
    }

    // in: read-only access
    static void ReadByIn(in int value)
    {
        Console.WriteLine($"  ReadByIn value: {value}");
        // value++; // ERROR: Cannot modify
    }

    static void Main()
    {
        // ref - must initialize first
        int refValue = 10;
        Console.WriteLine($"Before ref: {refValue}");
        IncrementByRef(ref refValue);
        Console.WriteLine($"After ref: {refValue}\n");

        // out - doesn't need initialization
        int outValue; // Not initialized
        Console.WriteLine("Calling InitializeByOut:");
        InitializeByOut(out outValue);
        Console.WriteLine($"After out: {outValue}\n");

        // in - read-only reference
        int inValue = 50;
        Console.WriteLine($"Before in: {inValue}");
        ReadByIn(in inValue);
        Console.WriteLine($"After in: {inValue} (unchanged)");
    }
}
```

**Output:**
```
Before ref: 10
  IncrementByRef result: 11
After ref: 11

Calling InitializeByOut:
  InitializeByOut result: 100
After out: 100

Before in: 50
  ReadByIn value: 50
After in: 50 (unchanged)
```

### Example 6: Reference Parameters with Objects

```csharp
using System;

class Person
{
    public string Name { get; set; }
    public int Age { get; set; }

    public override string ToString() => $"{Name} ({Age})";
}

class Program
{
    // Modify object properties (works without ref)
    static void ModifyPerson(Person p)
    {
        p.Name = "Modified";
        p.Age = 99;
    }

    // Replace entire object (requires ref)
    static void ReplacePerson(ref Person p)
    {
        p = new Person { Name = "Replaced", Age = 50 };
    }

    // Create new object (out)
    static void CreatePerson(out Person p)
    {
        p = new Person { Name = "Created", Age = 25 };
    }

    // Read-only reference (in)
    static void DisplayPerson(in Person p)
    {
        Console.WriteLine($"  Person: {p}");
        // Cannot reassign: p = new Person {...}; // ERROR
    }

    static void Main()
    {
        Person person = new Person { Name = "Alice", Age = 30 };

        Console.WriteLine($"Original: {person}");

        ModifyPerson(person);
        Console.WriteLine($"After ModifyPerson: {person}\n");

        ReplacePerson(ref person);
        Console.WriteLine($"After ReplacePerson: {person}\n");

        CreatePerson(out person);
        Console.WriteLine($"After CreatePerson: {person}\n");

        Console.WriteLine("DisplayPerson:");
        DisplayPerson(in person);
    }
}
```

**Output:**
```
Original: Alice (30)
After ModifyPerson: Modified (99)

After ReplacePerson: Replaced (50)

After CreatePerson: Created (25)

DisplayPerson:
  Person: Created (25)
```

### Example 7: Dictionary TryGet Pattern

```csharp
using System;
using System.Collections.Generic;

class Program
{
    static bool TryGetValue(Dictionary<string, int> dict, string key, out int value)
    {
        return dict.TryGetValue(key, out value);
    }

    static void Main()
    {
        Dictionary<string, int> ages = new Dictionary<string, int>
        {
            { "Alice", 30 },
            { "Bob", 25 },
            { "Charlie", 35 }
        };

        // Using TryGetValue with out parameter
        if (ages.TryGetValue("Alice", out int age))
        {
            Console.WriteLine($"Alice's age: {age}");
        }

        if (ages.TryGetValue("David", out age))
        {
            Console.WriteLine($"David's age: {age}");
        }
        else
        {
            Console.WriteLine("David not found");
        }

        // Inline declaration
        if (ages.TryGetValue("Bob", out int bobAge))
        {
            Console.WriteLine($"Bob's age: {bobAge}");
        }

        // Using discard for existence check only
        if (ages.TryGetValue("Charlie", out _))
        {
            Console.WriteLine("Charlie exists in dictionary");
        }
    }
}
```

**Output:**
```
Alice's age: 30
David not found
Bob's age: 25
Charlie exists in dictionary
```

### Example 8: Complex Example - Matrix Operations

```csharp
using System;

class Program
{
    static void SwapRows(ref int[] row1, ref int[] row2)
    {
        int[] temp = row1;
        row1 = row2;
        row2 = temp;
    }

    static void GetDimensions(int[,] matrix, out int rows, out int cols)
    {
        rows = matrix.GetLength(0);
        cols = matrix.GetLength(1);
    }

    static bool TryGetElement(int[,] matrix, int row, int col, out int value)
    {
        if (row >= 0 && row < matrix.GetLength(0) &&
            col >= 0 && col < matrix.GetLength(1))
        {
            value = matrix[row, col];
            return true;
        }

        value = 0;
        return false;
    }

    static void ModifyElement(ref int element, int newValue)
    {
        element = newValue;
    }

    static void Main()
    {
        int[,] matrix = {
            { 1, 2, 3 },
            { 4, 5, 6 },
            { 7, 8, 9 }
        };

        // Get dimensions
        GetDimensions(matrix, out int rows, out int cols);
        Console.WriteLine($"Matrix dimensions: {rows}x{cols}\n");

        // Display matrix
        Console.WriteLine("Original matrix:");
        for (int i = 0; i < rows; i++)
        {
            for (int j = 0; j < cols; j++)
            {
                Console.Write($"{matrix[i, j],4}");
            }
            Console.WriteLine();
        }

        // Try get element
        if (TryGetElement(matrix, 1, 1, out int element))
        {
            Console.WriteLine($"\nElement at [1,1]: {element}");
        }

        if (!TryGetElement(matrix, 5, 5, out element))
        {
            Console.WriteLine("Element at [5,5]: Out of bounds");
        }

        // Swap row references
        int[] firstRow = { 1, 2, 3 };
        int[] secondRow = { 4, 5, 6 };

        Console.WriteLine($"\nBefore swap:");
        Console.WriteLine($"First: [{string.Join(", ", firstRow)}]");
        Console.WriteLine($"Second: [{string.Join(", ", secondRow)}]");

        SwapRows(ref firstRow, ref secondRow);

        Console.WriteLine($"\nAfter swap:");
        Console.WriteLine($"First: [{string.Join(", ", firstRow)}]");
        Console.WriteLine($"Second: [{string.Join(", ", secondRow)}]");
    }
}
```

**Output:**
```
Matrix dimensions: 3x3

Original matrix:
   1   2   3
   4   5   6
   7   8   9

Element at [1,1]: 5
Element at [5,5]: Out of bounds

Before swap:
First: [1, 2, 3]
Second: [4, 5, 6]

After swap:
First: [4, 5, 6]
Second: [1, 2, 3]
```

## Key Points

### ref Parameters:
- Variable must be initialized before passing
- Method can read and modify the value
- Used for bidirectional data exchange
- Changes affect the original variable

### out Parameters:
- Variable doesn't need initialization
- Method must assign before returning
- Used for returning multiple values
- Common in Try-Parse patterns
- Supports inline declaration (C# 7.0+)

### in Parameters:
- Read-only reference
- Performance optimization (no copying)
- Prevents modifications
- Best for large structs
- Can omit `in` at call site (implicit)

### General Rules:
- `ref` and `in` require initialized variables
- `out` requires assignment in method
- All reference parameters must be matched at call site
- Cannot overload based solely on ref/out/in
- Reference parameters work with both value and reference types
- Use `_` discard to ignore unwanted out parameters
