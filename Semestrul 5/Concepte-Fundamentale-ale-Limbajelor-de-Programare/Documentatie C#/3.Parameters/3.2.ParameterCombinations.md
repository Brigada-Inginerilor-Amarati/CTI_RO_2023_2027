# Combinations of Parameter Type and Argument Mode in C#

## Description

C# provides different parameter passing modes that can be combined with value types and reference types. Understanding how these combinations work is crucial for effective method design and avoiding unexpected behavior.

### Parameter Modes:
- **Value** (default): Pass by value
- **ref**: Pass by reference (must be initialized before passing)
- **out**: Output parameter (must be assigned in method)
- **in**: Read-only reference (C# 7.2+)
- **params**: Variable number of arguments

### Type Categories:
- **Value Types**: int, double, bool, struct, enum
- **Reference Types**: class, string, array, delegate

### Combinations Matrix:
| Type / Mode | Value | ref | out | in |
|-------------|-------|-----|-----|-----|
| **Value Type** | Copy | Reference to variable | Must assign | Read-only ref |
| **Reference Type** | Copy of reference | Reference to reference | Must assign ref | Read-only ref |

## Code Examples

### Example 1: Value Type with Different Modes

```csharp
using System;

class Program
{
    // By value (default)
    static void ModifyByValue(int x)
    {
        Console.WriteLine($"  Inside ModifyByValue (before): {x}");
        x = 100;
        Console.WriteLine($"  Inside ModifyByValue (after): {x}");
    }

    // By reference (ref)
    static void ModifyByRef(ref int x)
    {
        Console.WriteLine($"  Inside ModifyByRef (before): {x}");
        x = 200;
        Console.WriteLine($"  Inside ModifyByRef (after): {x}");
    }

    // Output parameter (out)
    static void ModifyByOut(out int x)
    {
        // Must assign before using
        x = 300;
        Console.WriteLine($"  Inside ModifyByOut: {x}");
    }

    // Read-only reference (in)
    static void ReadByIn(in int x)
    {
        Console.WriteLine($"  Inside ReadByIn: {x}");
        // x = 400; // ERROR: Cannot modify in parameter
    }

    static void Main()
    {
        int value = 10;

        Console.WriteLine($"Original value: {value}");
        Console.WriteLine("\nCalling ModifyByValue:");
        ModifyByValue(value);
        Console.WriteLine($"After ModifyByValue: {value}\n");

        Console.WriteLine("Calling ModifyByRef:");
        ModifyByRef(ref value);
        Console.WriteLine($"After ModifyByRef: {value}\n");

        Console.WriteLine("Calling ModifyByOut:");
        ModifyByOut(out value);
        Console.WriteLine($"After ModifyByOut: {value}\n");

        Console.WriteLine("Calling ReadByIn:");
        ReadByIn(in value);
        Console.WriteLine($"After ReadByIn: {value}");
    }
}
```

**Output:**
```
Original value: 10

Calling ModifyByValue:
  Inside ModifyByValue (before): 10
  Inside ModifyByValue (after): 100
After ModifyByValue: 10

Calling ModifyByRef:
  Inside ModifyByRef (before): 10
  Inside ModifyByRef (after): 200
After ModifyByRef: 200

Calling ModifyByOut:
  Inside ModifyByOut: 300
After ModifyByOut: 300

Calling ReadByIn:
  Inside ReadByIn: 300
After ReadByIn: 300
```

### Example 2: Reference Type with Different Modes

```csharp
using System;

class Person
{
    public string Name { get; set; }
    public int Age { get; set; }

    public override string ToString() => $"{Name} ({Age})";
}

class Program
{
    // By value (reference copy)
    static void ModifyByValue(Person p)
    {
        Console.WriteLine($"  Inside (before): {p}");
        p.Name = "Modified";
        p.Age = 99;
        Console.WriteLine($"  Inside (after): {p}");
    }

    // Reassign by value
    static void ReassignByValue(Person p)
    {
        p = new Person { Name = "New Person", Age = 50 };
        Console.WriteLine($"  Inside ReassignByValue: {p}");
    }

    // By reference
    static void ReassignByRef(ref Person p)
    {
        p = new Person { Name = "Replaced", Age = 75 };
        Console.WriteLine($"  Inside ReassignByRef: {p}");
    }

    // Output parameter
    static void CreateByOut(out Person p)
    {
        p = new Person { Name = "Out Person", Age = 35 };
        Console.WriteLine($"  Inside CreateByOut: {p}");
    }

    static void Main()
    {
        Person person = new Person { Name = "Alice", Age = 30 };

        Console.WriteLine($"Original: {person}");
        Console.WriteLine("\nCalling ModifyByValue:");
        ModifyByValue(person);
        Console.WriteLine($"After ModifyByValue: {person}\n");

        Console.WriteLine("Calling ReassignByValue:");
        ReassignByValue(person);
        Console.WriteLine($"After ReassignByValue: {person}\n");

        Console.WriteLine("Calling ReassignByRef:");
        ReassignByRef(ref person);
        Console.WriteLine($"After ReassignByRef: {person}\n");

        Console.WriteLine("Calling CreateByOut:");
        CreateByOut(out person);
        Console.WriteLine($"After CreateByOut: {person}");
    }
}
```

**Output:**
```
Original: Alice (30)

Calling ModifyByValue:
  Inside (before): Alice (30)
  Inside (after): Modified (99)
After ModifyByValue: Modified (99)

Calling ReassignByValue:
  Inside ReassignByValue: New Person (50)
After ReassignByValue: Modified (99)

Calling ReassignByRef:
  Inside ReassignByRef: Replaced (75)
After ReassignByRef: Replaced (75)

Calling CreateByOut:
  Inside CreateByOut: Out Person (35)
After CreateByOut: Out Person (35)
```

### Example 3: Struct with Different Modes

```csharp
using System;

struct Point
{
    public int X;
    public int Y;

    public Point(int x, int y)
    {
        X = x;
        Y = y;
    }

    public override string ToString() => $"({X}, {Y})";
}

class Program
{
    static void ModifyByValue(Point p)
    {
        p.X = 100;
        p.Y = 200;
        Console.WriteLine($"  Inside ModifyByValue: {p}");
    }

    static void ModifyByRef(ref Point p)
    {
        p.X = 300;
        p.Y = 400;
        Console.WriteLine($"  Inside ModifyByRef: {p}");
    }

    static void CreateByOut(out Point p)
    {
        p = new Point(500, 600);
        Console.WriteLine($"  Inside CreateByOut: {p}");
    }

    static void ReadByIn(in Point p)
    {
        Console.WriteLine($"  Inside ReadByIn: {p}");
        // p.X = 700; // ERROR: Cannot modify in parameter
    }

    static void Main()
    {
        Point point = new Point(10, 20);

        Console.WriteLine($"Original: {point}");
        Console.WriteLine("\nCalling ModifyByValue:");
        ModifyByValue(point);
        Console.WriteLine($"After: {point}\n");

        Console.WriteLine("Calling ModifyByRef:");
        ModifyByRef(ref point);
        Console.WriteLine($"After: {point}\n");

        Console.WriteLine("Calling CreateByOut:");
        CreateByOut(out point);
        Console.WriteLine($"After: {point}\n");

        Console.WriteLine("Calling ReadByIn:");
        ReadByIn(in point);
        Console.WriteLine($"After: {point}");
    }
}
```

**Output:**
```

Calling ModifyByValue:
  Inside ModifyByValue: (100, 200)
After: (10, 20)

Calling ModifyByRef:
  Inside ModifyByRef: (300, 400)
```

### Example 4: Array Parameters with Different Modes

```csharp
        if (arr != null && arr.Length > 0)
        {

    {
        arr = new int[] { 100, 200, 300 };
        Console.WriteLine($"  Inside ReassignArray: [{string.Join(", ", arr)}]");

    {
        int[] numbers = { 1, 2, 3, 4, 5 };
```

### Example 5: Multiple Return Values Using out

```csharp
using System;

class Program
{
    static bool TryParse(string input, out int result, out string error)
    {
        error = null;
        result = 0;

        if (string.IsNullOrWhiteSpace(input))
        {
            error = "Input is empty";
            return false;
        }

        if (int.TryParse(input, out result))
        {
            return true;
        }
        else
        {
            error = "Invalid number format";
            return false;
        }
    }

    static void Divide(double a, double b, out double result, out bool success)
    {
        if (b == 0)
        {
            result = 0;
            success = false;
        }
        else
        {
            result = a / b;
            success = true;
        }
    }

    static void GetMinMax(int[] numbers, out int min, out int max)
    {
        if (numbers == null || numbers.Length == 0)
        {
            min = 0;
            max = 0;
            return;
        }

        min = numbers[0];
        max = numbers[0];

        foreach (int num in numbers)
        {
            if (num < min) min = num;
            if (num > max) max = num;
        }
    }

    static void Main()
    {
        // TryParse example
        if (TryParse("42", out int value, out string error))
        {
            Console.WriteLine($"Parsed successfully: {value}");
        }
        else
        {
            Console.WriteLine($"Parse failed: {error}");
        }

        if (TryParse("abc", out value, out error))
        {
            Console.WriteLine($"Parsed successfully: {value}");
        }
        else
        {
            Console.WriteLine($"Parse failed: {error}");
        }

        Console.WriteLine();

        // Divide example
        Divide(10, 2, out double result, out bool success);
        Console.WriteLine($"10 / 2 = {result}, Success: {success}");

        Divide(10, 0, out result, out success);
        Console.WriteLine($"10 / 0 = {result}, Success: {success}");

        Console.WriteLine();

        // GetMinMax example
        int[] numbers = { 5, 2, 8, 1, 9, 3 };
        GetMinMax(numbers, out int min, out int max);
        Console.WriteLine($"Array: [{string.Join(", ", numbers)}]");
        Console.WriteLine($"Min: {min}, Max: {max}");
    }
}
```

**Output:**
```
Parsed successfully: 42
Parse failed: Invalid number format

10 / 2 = 5, Success: True
10 / 0 = 0, Success: False

Array: [5, 2, 8, 1, 9, 3]
Min: 1, Max: 9
```

### Example 6: in Parameter for Performance

```csharp
using System;

struct LargeStruct
{
    public double A, B, C, D, E, F, G, H, I, J;

    public LargeStruct(double value)
    {
        A = B = C = D = E = F = G = H = I = J = value;
    }

    public double Sum() => A + B + C + D + E + F + G + H + I + J;
}

class Program
{
    // By value - creates a copy (expensive for large structs)
    static double ProcessByValue(LargeStruct data)
    {
        return data.Sum();
    }

    // By reference - no copy (more efficient)
    static double ProcessByRef(ref LargeStruct data)
    {
        return data.Sum();
    }

    // Read-only reference - efficient and safe
    static double ProcessByIn(in LargeStruct data)
    {
        return data.Sum();
        // data.A = 100; // ERROR: Cannot modify
    }

    static void Main()
    {
        LargeStruct data = new LargeStruct(5);

        Console.WriteLine("Processing large struct:");
        Console.WriteLine($"By value: {ProcessByValue(data)}");
        Console.WriteLine($"By ref: {ProcessByRef(ref data)}");
        Console.WriteLine($"By in: {ProcessByIn(in data)}");

        Console.WriteLine("\nNote: 'in' provides read-only access with ref performance");
    }
}
```

**Output:**
```
Processing large struct:
By value: 50
By ref: 50
By in: 50

Note: 'in' provides read-only access with ref performance
```

### Example 7: Swapping with ref

```csharp
using System;

class Program
{
    static void SwapInts(ref int a, ref int b)
    {
        int temp = a;
        a = b;
        b = temp;
    }

    static void SwapStrings(ref string a, ref string b)
    {
        string temp = a;
        a = b;
        b = temp;
    }

    static void SwapObjects<T>(ref T a, ref T b)
    {
        T temp = a;
        a = b;
        b = temp;
    }

    static void Main()
    {
        // Swap integers
        int x = 10, y = 20;
        Console.WriteLine($"Before: x={x}, y={y}");
        SwapInts(ref x, ref y);
        Console.WriteLine($"After: x={x}, y={y}");

        Console.WriteLine();

        // Swap strings
        string s1 = "Hello", s2 = "World";
        Console.WriteLine($"Before: s1={s1}, s2={s2}");
        SwapStrings(ref s1, ref s2);
        Console.WriteLine($"After: s1={s1}, s2={s2}");

        Console.WriteLine();

        // Generic swap
        double d1 = 3.14, d2 = 2.71;
        Console.WriteLine($"Before: d1={d1}, d2={d2}");
        SwapObjects(ref d1, ref d2);
        Console.WriteLine($"After: d1={d1}, d2={d2}");
    }
}
```

**Output:**
```
Before: x=10, y=20
After: x=20, y=10

Before: s1=Hello, s2=World
After: s1=World, s2=Hello

Before: d1=3.14, d2=2.71
After: d1=2.71, d2=3.14
```

### Example 8: Combining Different Parameter Modes

```csharp
using System;

class Program
{
    static void ProcessData(
        int input,                    // By value
        ref int modifyMe,             // By reference
        out int output,               // Output
        in int readOnly,              // Read-only reference
        string message = "Default")   // Optional
    {
        Console.WriteLine($"Input (by value): {input}");
        Console.WriteLine($"ModifyMe (by ref, before): {modifyMe}");
        Console.WriteLine($"ReadOnly (by in): {readOnly}");
        Console.WriteLine($"Message (optional): {message}");

        // Modify ref parameter
        modifyMe *= 2;

        // Must assign out parameter
        output = input + modifyMe + readOnly;

        Console.WriteLine($"ModifyMe (by ref, after): {modifyMe}");
        Console.WriteLine($"Output (assigned): {output}");
    }

    static void Main()
    {
        int inputVal = 10;
        int refVal = 20;
        int readOnlyVal = 30;

        Console.WriteLine("Before method call:");
        Console.WriteLine($"inputVal: {inputVal}");
        Console.WriteLine($"refVal: {refVal}");
        Console.WriteLine($"readOnlyVal: {readOnlyVal}");

        Console.WriteLine("\nCalling ProcessData:");
        ProcessData(inputVal, ref refVal, out int result, in readOnlyVal, "Custom");

        Console.WriteLine("\nAfter method call:");
        Console.WriteLine($"inputVal: {inputVal} (unchanged)");
        Console.WriteLine($"refVal: {refVal} (modified)");
        Console.WriteLine($"readOnlyVal: {readOnlyVal} (unchanged)");
        Console.WriteLine($"result: {result} (new value)");
    }
}
```

**Output:**
```
Before method call:
inputVal: 10
refVal: 20
readOnlyVal: 30

Calling ProcessData:
Input (by value): 10
ModifyMe (by ref, before): 20
ReadOnly (by in): 30
Message (optional): Custom
ModifyMe (by ref, after): 40
Output (assigned): 80

After method call:
inputVal: 10 (unchanged)
refVal: 40 (modified)
readOnlyVal: 30 (unchanged)
result: 80 (new value)
```

## Key Points

- **Value type + by value**: Complete copy; no effect on original
- **Value type + ref**: Direct access to original; modifications affect original
- **Value type + out**: Must assign in method; efficient for returning values
- **Value type + in**: Read-only reference; efficient for large structs

- **Reference type + by value**: Reference copy; can modify object, but not reassign
- **Reference type + ref**: Reference to reference; can reassign the reference itself
- **Reference type + out**: Must assign reference; useful for initialization
- **Reference type + in**: Read-only reference; prevents reassignment

- **ref**: Variable must be initialized before passing
- **out**: Method must assign before returning
- **in**: Provides read-only access with reference performance
- Combine modes appropriately for efficient and clear APIs
