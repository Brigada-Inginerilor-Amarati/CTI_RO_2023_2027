# params Modifier in C#

## Description

The `params` keyword allows a method to accept a variable number of arguments. When you declare a parameter with `params`, callers can pass zero or more arguments of the specified type, or an array of that type. The compiler automatically creates an array to hold the arguments.

### Key Features:
- **Variable Arguments**: Accept any number of parameters
- **Type Safety**: All arguments must be of the specified type
- **Array Conversion**: Arguments are packaged into an array
- **Convenience**: No need to explicitly create arrays at call site
- **Must Be Last**: `params` parameter must be the last in the parameter list
- **Only One**: Can only have one `params` parameter per method

### Syntax:
```csharp
returnType MethodName(params Type[] parameterName)
```

## Code Examples

### Example 1: Basic params Usage

```csharp
using System;

class Program
{
    static int Sum(params int[] numbers)
    {
        int total = 0;
        foreach (int num in numbers)
        {
            total += num;
        }
        return total;
    }

    static void PrintNumbers(params int[] numbers)
    {
        Console.Write("Numbers: ");
        if (numbers.Length == 0)
        {
            Console.WriteLine("(none)");
        }
        else
        {
            Console.WriteLine(string.Join(", ", numbers));
        }
    }

    static void Main()
    {
        // Call with no arguments
        Console.WriteLine($"Sum(): {Sum()}");

        // Call with one argument
        Console.WriteLine($"Sum(5): {Sum(5)}");

        // Call with multiple arguments
        Console.WriteLine($"Sum(1, 2, 3): {Sum(1, 2, 3)}");

        // Call with many arguments
        Console.WriteLine($"Sum(1, 2, 3, 4, 5, 6): {Sum(1, 2, 3, 4, 5, 6)}");

        Console.WriteLine();

        // PrintNumbers examples
        PrintNumbers();
        PrintNumbers(10);
        PrintNumbers(10, 20, 30, 40, 50);

        // Can also pass an array explicitly
        int[] array = { 7, 8, 9 };
        Console.WriteLine($"\nSum(array): {Sum(array)}");
    }
}
```

**Output:**
```
Sum(): 0
Sum(5): 5
Sum(1, 2, 3): 6
Sum(1, 2, 3, 4, 5, 6): 21

Numbers: (none)
Numbers: 10
Numbers: 10, 20, 30, 40, 50

Sum(array): 24
```

### Example 2: params with Different Types

```csharp
using System;

class Program
{
    static void PrintStrings(params string[] strings)
    {
        Console.WriteLine($"Count: {strings.Length}");
        foreach (string str in strings)
        {
            Console.WriteLine($"  - {str}");
        }
    }

    static double Average(params double[] values)
    {
        if (values.Length == 0)
            return 0;

        double sum = 0;
        foreach (double value in values)
        {
            sum += value;
        }
        return sum / values.Length;
    }

    static void PrintObjects(params object[] objects)
    {
        Console.WriteLine("Objects:");
        foreach (object obj in objects)
        {
            Console.WriteLine($"  {obj} (Type: {obj.GetType().Name})");
        }
    }

    static void Main()
    {
        // String params
        Console.WriteLine("PrintStrings:");
        PrintStrings("apple", "banana", "cherry");

        Console.WriteLine();

        // Double params
        double avg = Average(5.5, 7.2, 3.8, 9.1);
        Console.WriteLine($"Average: {avg:F2}");

        Console.WriteLine();

        // Object params (can accept any type)
        PrintObjects(42, "Hello", 3.14, true);
    }
}
```

**Output:**
```
PrintStrings:
Count: 3
  - apple
  - banana
  - cherry

Average: 6.40

Objects:
  42 (Type: Int32)
  Hello (Type: String)
  3.14 (Type: Double)
  True (Type: Boolean)
```

### Example 3: params with Regular Parameters

```csharp
using System;

class Program
{
    // Regular parameter followed by params
    static void LogMessage(string level, params string[] messages)
    {
        Console.WriteLine($"[{level}]");
        foreach (string message in messages)
        {
            Console.WriteLine($"  {message}");
        }
    }

    // Multiple regular parameters with params
    static string FormatList(string prefix, string suffix, params string[] items)
    {
        string result = prefix;
        for (int i = 0; i < items.Length; i++)
        {
            result += items[i];
            if (i < items.Length - 1)
                result += ", ";
        }
        result += suffix;
        return result;
    }

    static int Calculate(int initial, params int[] values)
    {
        int result = initial;
        foreach (int value in values)
        {
            result += value;
        }
        return result;
    }

    static void Main()
    {
        // LogMessage examples
        LogMessage("INFO", "Application started");
        LogMessage("ERROR", "Connection failed", "Retry attempt 1", "Retry attempt 2");

        Console.WriteLine();

        // FormatList examples
        string list1 = FormatList("Items: [", "]", "apple", "banana", "cherry");
        Console.WriteLine(list1);

        string list2 = FormatList("Names: {", "}", "Alice", "Bob", "Charlie", "David");
        Console.WriteLine(list2);

        Console.WriteLine();

        // Calculate examples
        Console.WriteLine($"Calculate(10): {Calculate(10)}");
        Console.WriteLine($"Calculate(10, 5, 3, 2): {Calculate(10, 5, 3, 2)}");
    }
}
```

**Output:**
```
[INFO]
  Application started
[ERROR]
  Connection failed
  Retry attempt 1
  Retry attempt 2

Items: [apple, banana, cherry]
Names: {Alice, Bob, Charlie, David}

Calculate(10): 10
Calculate(10, 5, 3, 2): 20
```

### Example 4: Generic params

```csharp
using System;

class Program
{
    static T[] CreateArray<T>(params T[] elements)
    {
        Console.WriteLine($"Created array of {typeof(T).Name} with {elements.Length} elements");
        return elements;
    }

    static void PrintAll<T>(params T[] items)
    {
        Console.Write($"{typeof(T).Name} items: ");
        Console.WriteLine(string.Join(", ", items));
    }

    static T Max<T>(params T[] values) where T : IComparable<T>
    {
        if (values.Length == 0)
            throw new ArgumentException("At least one value required");

        T max = values[0];
        foreach (T value in values)
        {
            if (value.CompareTo(max) > 0)
                max = value;
        }
        return max;
    }

    static void Main()
    {
        // Generic params with different types
        int[] numbers = CreateArray(1, 2, 3, 4, 5);
        string[] words = CreateArray("hello", "world", "generic");

        Console.WriteLine();

        PrintAll(10, 20, 30, 40);
        PrintAll("Alice", "Bob", "Charlie");
        PrintAll(1.1, 2.2, 3.3);

        Console.WriteLine();

        Console.WriteLine($"Max int: {Max(5, 12, 3, 18, 7)}");
        Console.WriteLine($"Max string: {Max("apple", "zebra", "banana")}");
        Console.WriteLine($"Max double: {Max(3.14, 2.71, 9.81)}");
    }
}
```

**Output:**
```
Created array of Int32 with 5 elements
Created array of String with 3 elements

Int32 items: 10, 20, 30, 40
String items: Alice, Bob, Charlie
Double items: 1.1, 2.2, 3.3

Max int: 18
Max string: zebra
Max double: 9.81
```

### Example 5: String Formatting with params

```csharp
using System;

class Program
{
    static string Format(string template, params object[] args)
    {
        string result = template;
        for (int i = 0; i < args.Length; i++)
        {
            result = result.Replace($"{{{i}}}", args[i].ToString());
        }
        return result;
    }

    static void WriteLine(params object[] values)
    {
        Console.WriteLine(string.Join(" ", values));
    }

    static string Join(string separator, params string[] strings)
    {
        return string.Join(separator, strings);
    }

    static void Main()
    {
        // Custom format method
        string message1 = Format("Hello, {0}! You are {1} years old.", "Alice", 30);
        Console.WriteLine(message1);

        string message2 = Format("Result: {0} + {1} = {2}", 5, 7, 12);
        Console.WriteLine(message2);

        Console.WriteLine();

        // WriteLine with multiple types
        WriteLine("Values:", 42, 3.14, true, "text");

        Console.WriteLine();

        // Join strings
        string path = Join("/", "home", "user", "documents", "file.txt");
        Console.WriteLine($"Path: {path}");

        string csv = Join(", ", "apple", "banana", "cherry", "date");
        Console.WriteLine($"CSV: {csv}");
    }
}
```

**Output:**
```
Hello, Alice! You are 30 years old.
Result: 5 + 7 = 12

Values: 42 3.14 True text

Path: home/user/documents/file.txt
CSV: apple, banana, cherry, date
```

### Example 6: Mathematical Operations

```csharp
using System;

class MathHelper
{
    public static int Add(params int[] numbers)
    {
        int sum = 0;
        foreach (int num in numbers)
            sum += num;
        return sum;
    }

    public static int Multiply(params int[] numbers)
    {
        if (numbers.Length == 0)
            return 0;

        int product = 1;
        foreach (int num in numbers)
            product *= num;
        return product;
    }

    public static double Average(params double[] values)
    {
        if (values.Length == 0)
            return 0;

        double sum = 0;
        foreach (double value in values)
            sum += value;

        return sum / values.Length;
    }

    public static void Statistics(params int[] numbers)
    {
        if (numbers.Length == 0)
        {
            Console.WriteLine("No numbers provided");
            return;
        }

        int sum = 0;
        int min = numbers[0];
        int max = numbers[0];

        foreach (int num in numbers)
        {
            sum += num;
            if (num < min) min = num;
            if (num > max) max = num;
        }

        double avg = (double)sum / numbers.Length;

        Console.WriteLine($"Count: {numbers.Length}");
        Console.WriteLine($"Sum: {sum}");
        Console.WriteLine($"Average: {avg:F2}");
        Console.WriteLine($"Min: {min}");
        Console.WriteLine($"Max: {max}");
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine($"Add(1, 2, 3, 4, 5): {MathHelper.Add(1, 2, 3, 4, 5)}");
        Console.WriteLine($"Multiply(2, 3, 4): {MathHelper.Multiply(2, 3, 4)}");
        Console.WriteLine($"Average(5.5, 7.2, 3.8): {MathHelper.Average(5.5, 7.2, 3.8):F2}");

        Console.WriteLine("\nStatistics:");
        MathHelper.Statistics(5, 12, 3, 18, 7, 9, 15);
    }
}
```

**Output:**
```
Add(1, 2, 3, 4, 5): 15
Multiply(2, 3, 4): 24
Average(5.5, 7.2, 3.8): 5.50

Statistics:
Count: 7
Sum: 69
Average: 9.86
Min: 3
Max: 18
```

### Example 7: Builder Pattern with params

```csharp
using System;
using System.Collections.Generic;

class QueryBuilder
{
    private List<string> conditions = new List<string>();

    public QueryBuilder Where(params string[] conditions)
    {
        this.conditions.AddRange(conditions);
        return this;
    }

    public string Build()
    {
        if (conditions.Count == 0)
            return "SELECT * FROM table";

        return $"SELECT * FROM table WHERE {string.Join(" AND ", conditions)}";
    }
}

class StringBuilder2
{
    private List<string> parts = new List<string>();

    public StringBuilder2 Append(params string[] strings)
    {
        parts.AddRange(strings);
        return this;
    }

    public override string ToString()
    {
        return string.Join("", parts);
    }
}

class Program
{
    static void Main()
    {
        // Query builder
        QueryBuilder query1 = new QueryBuilder()
            .Where("age > 18", "status = 'active'", "role = 'admin'");

        Console.WriteLine(query1.Build());

        QueryBuilder query2 = new QueryBuilder()
            .Where("name LIKE '%John%'");

        Console.WriteLine(query2.Build());

        Console.WriteLine();

        // String builder
        StringBuilder2 sb = new StringBuilder2()
            .Append("Hello", " ", "World", "!")
            .Append(" ", "How", " ", "are", " ", "you?");

        Console.WriteLine(sb.ToString());
    }
}
```

**Output:**
```
SELECT * FROM table WHERE age > 18 AND status = 'active' AND role = 'admin'
SELECT * FROM table WHERE name LIKE '%John%'

Hello World! How are you?
```

### Example 8: Validation and Error Checking

```csharp
using System;
using System.Linq;

class Validator
{
    public static bool AllPositive(params int[] numbers)
    {
        return numbers.All(n => n > 0);
    }

    public static bool AllNotNull(params string[] strings)
    {
        return strings.All(s => s != null);
    }

    public static bool AnyEmpty(params string[] strings)
    {
        return strings.Any(s => string.IsNullOrEmpty(s));
    }

    public static string[] ValidateInputs(params string[] inputs)
    {
        var errors = new System.Collections.Generic.List<string>();

        for (int i = 0; i < inputs.Length; i++)
        {
            if (string.IsNullOrWhiteSpace(inputs[i]))
            {
                errors.Add($"Input {i + 1} is empty or whitespace");
            }
        }

        return errors.ToArray();
    }
}

class Program
{
    static void Main()
    {
        // AllPositive
        Console.WriteLine($"AllPositive(1, 2, 3): {Validator.AllPositive(1, 2, 3)}");
        Console.WriteLine($"AllPositive(1, -2, 3): {Validator.AllPositive(1, -2, 3)}");

        Console.WriteLine();

        // AllNotNull
        Console.WriteLine($"AllNotNull('a', 'b', 'c'): {Validator.AllNotNull("a", "b", "c")}");
        Console.WriteLine($"AllNotNull('a', null, 'c'): {Validator.AllNotNull("a", null, "c")}");

        Console.WriteLine();

        // AnyEmpty
        Console.WriteLine($"AnyEmpty('a', 'b', 'c'): {Validator.AnyEmpty("a", "b", "c")}");
        Console.WriteLine($"AnyEmpty('a', '', 'c'): {Validator.AnyEmpty("a", "", "c")}");

        Console.WriteLine();

        // ValidateInputs
        string[] errors = Validator.ValidateInputs("valid", "", "also valid", "   ");
        if (errors.Length > 0)
        {
            Console.WriteLine("Validation errors:");
            foreach (string error in errors)
            {
                Console.WriteLine($"  - {error}");
            }
        }
    }
}
```

**Output:**
```
AllPositive(1, 2, 3): True
AllPositive(1, -2, 3): False

AllNotNull('a', 'b', 'c'): True
AllNotNull('a', null, 'c'): False

AnyEmpty('a', 'b', 'c'): False
AnyEmpty('a', '', 'c'): True

Validation errors:
  - Input 2 is empty or whitespace
  - Input 4 is empty or whitespace
```

## Key Points

- **params** allows methods to accept variable number of arguments
- Must be the **last parameter** in the parameter list
- Only **one params parameter** per method
- Arguments are packaged into an **array** automatically
- Can be called with **zero or more** arguments
- Can pass an **explicit array** instead of individual arguments
- Works with **any type** including generics
- Provides **type safety** - all arguments must match the parameter type
- Common in utility methods, logging, formatting, and builder patterns
- The `params` keyword is syntactic sugar for array parameters
- Improves API usability by eliminating explicit array creation
- Used extensively in framework methods like `Console.WriteLine()`, `string.Format()`, `string.Concat()`
