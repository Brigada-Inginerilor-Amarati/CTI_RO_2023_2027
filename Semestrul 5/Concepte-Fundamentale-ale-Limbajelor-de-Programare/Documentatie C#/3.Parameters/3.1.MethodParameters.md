# Method Parameters in C#

## Description

Method parameters allow you to pass data into methods. C# supports various parameter passing mechanisms that control how arguments are passed and how they can be modified within the method.

### Parameter Passing Modes:
- **By Value** (default): A copy of the argument is passed
- **By Reference** (`ref`): The actual variable is passed; changes affect the original
- **Output** (`out`): Similar to `ref`, but the method must assign a value
- **Input Reference** (`in`): Read-only reference; cannot be modified
- **Parameter Array** (`params`): Accepts variable number of arguments

### Key Concepts:
- **Value Types**: Passed by copying the value (int, double, struct, etc.)
- **Reference Types**: The reference is copied, but points to the same object (class, string, array, etc.)
- **Named Arguments**: Specify parameter names when calling methods
- **Optional Parameters**: Parameters with default values

## Code Examples

### Example 1: Value Type Parameters (Pass by Value)

```csharp
using System;

class Program
{
    static void ModifyValue(int number)
    {
        Console.WriteLine($"Inside method (before): {number}");
        number = 100;
        Console.WriteLine($"Inside method (after): {number}");
    }

    static void Main()
    {
        int value = 10;
        Console.WriteLine($"Original value: {value}");

        ModifyValue(value);

        Console.WriteLine($"After method call: {value}");
    }
}
```

**Output:**
```
Original value: 10
Inside method (before): 10
Inside method (after): 100
After method call: 10
```

### Example 2: Reference Type Parameters

```csharp
using System;

class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

class Program
{
    static void ModifyPerson(Person p)
    {
        Console.WriteLine($"Inside method (before): {p.Name}, {p.Age}");
        p.Name = "Modified Name";
        p.Age = 100;
        Console.WriteLine($"Inside method (after): {p.Name}, {p.Age}");
    }

    static void ReassignPerson(Person p)
    {
        p = new Person { Name = "New Person", Age = 50 };
        Console.WriteLine($"Inside ReassignPerson: {p.Name}, {p.Age}");
    }

    static void Main()
    {
        Person person = new Person { Name = "Alice", Age = 30 };
        Console.WriteLine($"Original: {person.Name}, {person.Age}");

        ModifyPerson(person);
        Console.WriteLine($"After ModifyPerson: {person.Name}, {person.Age}");

        Console.WriteLine();

        ReassignPerson(person);
        Console.WriteLine($"After ReassignPerson: {person.Name}, {person.Age}");
    }
}
```

**Output:**
```
Original: Alice, 30
Inside method (before): Alice, 30
Inside method (after): Modified Name, 100
After ModifyPerson: Modified Name, 100

Inside ReassignPerson: New Person, 50
After ReassignPerson: Modified Name, 100
```

### Example 3: Multiple Parameters

```csharp
using System;

class Calculator
{
    static int Add(int a, int b)
    {
        return a + b;
    }

    static double CalculateArea(double width, double height)
    {
        return width * height;
    }

    static string FormatPerson(string firstName, string lastName, int age)
    {
        return $"{firstName} {lastName} ({age} years old)";
    }

    static void DisplayInfo(string name, int count, bool isActive)
    {
        Console.WriteLine($"Name: {name}");
        Console.WriteLine($"Count: {count}");
        Console.WriteLine($"Active: {isActive}");
    }
}

class Program
{
    static void Main()
    {
        int sum = Calculator.Add(5, 10);
        Console.WriteLine($"Sum: {sum}");

        double area = Calculator.CalculateArea(5.5, 3.2);
        Console.WriteLine($"Area: {area:F2}");

        string info = Calculator.FormatPerson("John", "Doe", 30);
        Console.WriteLine($"Person: {info}");

        Console.WriteLine();
        Calculator.DisplayInfo("Product", 42, true);
    }
}
```

**Output:**
```
Sum: 15
Area: 17.60
Person: John Doe (30 years old)

Name: Product
Count: 42
Active: True
```

### Example 4: Optional Parameters and Default Values

```csharp
using System;

class Program
{
    static void Greet(string name, string greeting = "Hello", bool formal = false)
    {
        if (formal)
        {
            Console.WriteLine($"{greeting}, Mr./Ms. {name}");
        }
        else
        {
            Console.WriteLine($"{greeting}, {name}!");
        }
    }

    static double CalculatePrice(double basePrice, double discount = 0, double tax = 0.1)
    {
        double afterDiscount = basePrice * (1 - discount);
        double afterTax = afterDiscount * (1 + tax);
        return afterTax;
    }

    static void Main()
    {
        // Using all defaults
        Greet("Alice");

        // Overriding greeting
        Greet("Bob", "Hi");

        // Overriding both
        Greet("Charlie", "Good morning", true);

        Console.WriteLine();

        // Calculate prices with different parameters
        Console.WriteLine($"Price (base): ${CalculatePrice(100):F2}");
        Console.WriteLine($"Price (with discount): ${CalculatePrice(100, 0.2):F2}");
        Console.WriteLine($"Price (with discount and tax): ${CalculatePrice(100, 0.2, 0.15):F2}");
    }
}
```

**Output:**
```
Hello, Alice!
Hi, Bob!
Good morning, Mr./Ms. Charlie

Price (base): $110.00
Price (with discount): $88.00
Price (with discount and tax): $92.00
```

### Example 5: Named Arguments

```csharp
using System;

class Program
{
    static void CreateAccount(string username, string email, int age, bool premium = false)
    {
        Console.WriteLine($"Username: {username}");
        Console.WriteLine($"Email: {email}");
        Console.WriteLine($"Age: {age}");
        Console.WriteLine($"Premium: {premium}");
        Console.WriteLine();
    }

    static void Main()
    {
        // Positional arguments
        CreateAccount("alice123", "alice@example.com", 25, true);

        // Named arguments (any order)
        CreateAccount(
            email: "bob@example.com",
            username: "bob456",
            age: 30
        );

        // Mix of positional and named
        CreateAccount("charlie", email: "charlie@example.com", age: 28);

        // Named arguments with optional parameter
        CreateAccount(
            username: "david",
            age: 35,
            email: "david@example.com",
            premium: true
        );
    }
}
```

**Output:**
```
Username: alice123
Email: alice@example.com
Age: 25
Premium: True

Username: bob456
Email: bob@example.com
Age: 30
Premium: False

Username: charlie
Email: charlie@example.com
Age: 28
Premium: False

Username: david
Email: david@example.com
Age: 35
Premium: True
```

### Example 6: Method Overloading

```csharp
using System;

class Calculator
{
    // Integer addition
    public static int Add(int a, int b)
    {
        Console.WriteLine($"Calling Add(int, int)");
        return a + b;
    }

    // Double addition
    public static double Add(double a, double b)
    {
        Console.WriteLine($"Calling Add(double, double)");
        return a + b;
    }

    // Three parameters
    public static int Add(int a, int b, int c)
    {
        Console.WriteLine($"Calling Add(int, int, int)");
        return a + b + c;
    }

    // String concatenation
    public static string Add(string a, string b)
    {
        Console.WriteLine($"Calling Add(string, string)");
        return a + b;
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine($"Result: {Calculator.Add(5, 10)}\n");

        Console.WriteLine($"Result: {Calculator.Add(5.5, 3.2)}\n");

        Console.WriteLine($"Result: {Calculator.Add(1, 2, 3)}\n");

        Console.WriteLine($"Result: {Calculator.Add("Hello", "World")}\n");
    }
}
```

**Output:**
```
Calling Add(int, int)
Result: 15

Calling Add(double, double)
Result: 8.7

Calling Add(int, int, int)
Result: 6

Calling Add(string, string)
Result: HelloWorld
```

### Example 7: Array Parameters

```csharp
using System;

class Program
{
    static void PrintArray(int[] numbers)
    {
        Console.Write("Array: [");
        for (int i = 0; i < numbers.Length; i++)
        {
            Console.Write(numbers[i]);
            if (i < numbers.Length - 1)
                Console.Write(", ");
        }
        Console.WriteLine("]");
    }

    static void ModifyArray(int[] numbers)
    {
        for (int i = 0; i < numbers.Length; i++)
        {
            numbers[i] *= 2;
        }
    }

    static int SumArray(int[] numbers)
    {
        int sum = 0;
        foreach (int num in numbers)
        {
            sum += num;
        }
        return sum;
    }

    static void Main()
    {
        int[] numbers = { 1, 2, 3, 4, 5 };

        Console.WriteLine("Original array:");
        PrintArray(numbers);

        int sum = SumArray(numbers);
        Console.WriteLine($"Sum: {sum}");

        Console.WriteLine("\nModifying array...");
        ModifyArray(numbers);

        Console.WriteLine("Modified array:");
        PrintArray(numbers);
    }
}
```

**Output:**
```
Original array:
Array: [1, 2, 3, 4, 5]
Sum: 15

Modifying array...
Modified array:
Array: [2, 4, 6, 8, 10]
```

### Example 8: Null and Nullable Parameters

```csharp
using System;

class Program
{
    static void ProcessString(string text)
    {
        if (text == null)
        {
            Console.WriteLine("String is null");
        }
        else if (text.Length == 0)
        {
            Console.WriteLine("String is empty");
        }
        else
        {
            Console.WriteLine($"String: '{text}' (Length: {text.Length})");
        }
    }

    static void ProcessNumber(int? number)
    {
        if (number.HasValue)
        {
            Console.WriteLine($"Number: {number.Value}");
        }
        else
        {
            Console.WriteLine("Number is null");
        }
    }

    static void DisplayPerson(string name = "Unknown", int? age = null)
    {
        Console.Write($"Name: {name}");
        if (age.HasValue)
        {
            Console.WriteLine($", Age: {age.Value}");
        }
        else
        {
            Console.WriteLine(", Age: Not specified");
        }
    }

    static void Main()
    {
        ProcessString("Hello");
        ProcessString("");
        ProcessString(null);

        Console.WriteLine();

        ProcessNumber(42);
        ProcessNumber(null);

        Console.WriteLine();

        DisplayPerson("Alice", 30);
        DisplayPerson("Bob");
        DisplayPerson();
    }
}
```

**Output:**
```
String: 'Hello' (Length: 5)
String is empty
String is null

Number: 42
Number is null

Name: Alice, Age: 30
Name: Bob, Age: Not specified
Name: Unknown, Age: Not specified
```

### Example 9: Expression-Bodied Methods

```csharp
using System;

class Calculator
{
    // Expression-bodied method
    static int Square(int x) => x * x;

    static double CircleArea(double radius) => Math.PI * radius * radius;

    static string GetGreeting(string name) => $"Hello, {name}!";

    static bool IsEven(int number) => number % 2 == 0;

    static int Max(int a, int b) => a > b ? a : b;
}

class Program
{
    static void Main()
    {
        Console.WriteLine($"Square(5): {Calculator.Square(5)}");

        Console.WriteLine($"Circle area (r=3): {Calculator.CircleArea(3):F2}");

        Console.WriteLine(Calculator.GetGreeting("Alice"));

        Console.WriteLine($"Is 7 even? {Calculator.IsEven(7)}");
        Console.WriteLine($"Is 8 even? {Calculator.IsEven(8)}");

        Console.WriteLine($"Max(15, 23): {Calculator.Max(15, 23)}");
    }
}
```

**Output:**
```
Square(5): 25
Circle area (r=3): 28.27
Hello, Alice!
Is 7 even? False
Is 8 even? True
Max(15, 23): 23
```

## Key Points

- **Default behavior**: Value types are copied; reference types pass a reference copy
- **Value types** (int, double, struct): Changes inside the method don't affect the original
- **Reference types** (class, array, string): Can modify object properties, but reassignment doesn't affect the original
- **Optional parameters**: Must appear after required parameters
- **Default values**: Must be compile-time constants
- **Named arguments**: Allow any order and improve readability
- **Method overloading**: Same name, different parameters
- Arrays are reference types; modifications affect the original
- Nullable types allow null values for value types
- Expression-bodied methods provide concise syntax for simple methods
