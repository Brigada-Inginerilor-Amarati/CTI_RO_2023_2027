# 1.3 Main Function and Command Line Arguments

## Description

The `Main` method is the entry point of a C# application. When a program starts, the `Main` method is the first method that gets invoked. It can accept command-line arguments, allowing users to pass data to the program when it starts.

**Key Points:**
- **Entry point**: Every C# application must have exactly one `Main` method
- **Signature variations**: Main can have different valid signatures
- **Return types**: Can return `void` or `int`
- **Parameters**: Can accept `string[] args` for command-line arguments
- **Static method**: Must be declared as `static`

**Valid Main Method Signatures:**
```csharp
static void Main()
static void Main(string[] args)
static int Main()
static int Main(string[] args)
static async Task Main()
static async Task Main(string[] args)
static async Task<int> Main()
static async Task<int> Main(string[] args)
```

## Code Examples

### Basic Main Method (No Arguments)

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello from Main method!");
        Console.WriteLine("No command-line arguments accepted.");
    }
}
```

**Output:**
```
Hello from Main method!
No command-line arguments accepted.
```

### Main with Command-Line Arguments

```csharp
using System;

class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine($"Number of arguments: {args.Length}");

        if (args.Length > 0)
        {
            Console.WriteLine("\nArguments received:");
            for (int i = 0; i < args.Length; i++)
            {
                Console.WriteLine($"Argument {i}: {args[i]}");
            }
        }
        else
        {
            Console.WriteLine("No arguments provided.");
        }
    }
}
```

**Run Command:**
```bash
dotnet run Alice 25 Developer
```

**Output:**
```
Number of arguments: 3

Arguments received:
Argument 0: Alice
Argument 1: 25
Argument 2: Developer
```

### Main with Return Value

```csharp
using System;

class Program
{
    static int Main(string[] args)
    {
        Console.WriteLine("Processing command...");

        if (args.Length == 0)
        {
            Console.WriteLine("Error: No arguments provided!");
            return 1; // Return error code
        }

        Console.WriteLine($"Success! Processed: {args[0]}");
        return 0; // Return success code
    }
}
```

**Run Command (with error):**
```bash
dotnet run
```

**Output:**
```
Processing command...
Error: No arguments provided!
```
**Exit code: 1**

**Run Command (success):**
```bash
dotnet run MyFile.txt
```

**Output:**
```
Processing command...
Success! Processed: MyFile.txt
```
**Exit code: 0**

### Processing Different Argument Types

```csharp
using System;

class Calculator
{
    static void Main(string[] args)
    {
        if (args.Length != 3)
        {
            Console.WriteLine("Usage: Calculator <number1> <operation> <number2>");
            Console.WriteLine("Example: Calculator 10 + 5");
            return;
        }

        try
        {
            double num1 = double.Parse(args[0]);
            string operation = args[1];
            double num2 = double.Parse(args[2]);
            double result = 0;

            switch (operation)
            {
                case "+":
                    result = num1 + num2;
                    break;
                case "-":
                    result = num1 - num2;
                    break;
                case "*":
                    result = num1 * num2;
                    break;
                case "/":
                    if (num2 == 0)
                    {
                        Console.WriteLine("Error: Division by zero!");
                        return;
                    }
                    result = num1 / num2;
                    break;
                default:
                    Console.WriteLine($"Unknown operation: {operation}");
                    return;
            }

            Console.WriteLine($"{num1} {operation} {num2} = {result}");
        }
        catch (FormatException)
        {
            Console.WriteLine("Error: Invalid number format!");
        }
    }
}
```

**Run Command:**
```bash
dotnet run 15 + 7
```

**Output:**
```
15 + 7 = 22
```

### Named Arguments and Options

```csharp
using System;

class FileProcessor
{
    static void Main(string[] args)
    {
        string filename = "";
        bool verbose = false;
        string outputPath = "output.txt";

        // Parse arguments
        for (int i = 0; i < args.Length; i++)
        {
            switch (args[i])
            {
                case "--file":
                case "-f":
                    if (i + 1 < args.Length)
                    {
                        filename = args[++i];
                    }
                    break;

                case "--verbose":
                case "-v":
                    verbose = true;
                    break;

                case "--output":
                case "-o":
                    if (i + 1 < args.Length)
                    {
                        outputPath = args[++i];
                    }
                    break;

                case "--help":
                case "-h":
                    ShowHelp();
                    return;
            }
        }

        if (string.IsNullOrEmpty(filename))
        {
            Console.WriteLine("Error: No file specified!");
            Console.WriteLine("Use --help for usage information.");
            return;
        }

        Console.WriteLine($"Processing file: {filename}");
        Console.WriteLine($"Output path: {outputPath}");

        if (verbose)
        {
            Console.WriteLine("Verbose mode: ON");
            Console.WriteLine("Starting detailed processing...");
        }
    }

    static void ShowHelp()
    {
        Console.WriteLine("FileProcessor - Command Line Options");
        Console.WriteLine("====================================");
        Console.WriteLine("--file, -f <filename>    Input file to process");
        Console.WriteLine("--output, -o <path>      Output file path");
        Console.WriteLine("--verbose, -v            Enable verbose output");
        Console.WriteLine("--help, -h               Show this help message");
    }
}
```

**Run Command:**
```bash
dotnet run --file data.txt --verbose --output results.txt
```

**Output:**
```
Processing file: data.txt
Output path: results.txt
Verbose mode: ON
Starting detailed processing...
```

### Using Environment.GetCommandLineArgs()

```csharp
using System;

class Program
{
    static void Main(string[] args)
    {
        // Get all command-line arguments including program name
        string[] allArgs = Environment.GetCommandLineArgs();

        Console.WriteLine("All command-line arguments:");
        Console.WriteLine($"Program: {allArgs[0]}");
        Console.WriteLine();

        Console.WriteLine("Additional arguments:");
        for (int i = 1; i < allArgs.Length; i++)
        {
            Console.WriteLine($"  [{i}]: {allArgs[i]}");
        }

        Console.WriteLine();
        Console.WriteLine("Using Main args parameter:");
        foreach (string arg in args)
        {
            Console.WriteLine($"  - {arg}");
        }
    }
}
```

**Run Command:**
```bash
dotnet run arg1 arg2 arg3
```

**Output:**
```
All command-line arguments:
Program: /path/to/Program.dll

Additional arguments:
  [1]: arg1
  [2]: arg2
  [3]: arg3

Using Main args parameter:
  - arg1
  - arg2
  - arg3
```

### Async Main Method

```csharp
using System;
using System.Net.Http;
using System.Threading.Tasks;

class Program
{
    static async Task<int> Main(string[] args)
    {
        Console.WriteLine("Starting async operation...");

        if (args.Length == 0)
        {
            Console.WriteLine("Please provide a URL");
            return 1;
        }

        string url = args[0];

        try
        {
            using (HttpClient client = new HttpClient())
            {
                Console.WriteLine($"Fetching: {url}");
                string content = await client.GetStringAsync(url);
                Console.WriteLine($"Content length: {content.Length} characters");
                return 0;
            }
        }
        catch (Exception ex)
        {
            Console.WriteLine($"Error: {ex.Message}");
            return 1;
        }
    }
}
```

**Run Command:**
```bash
dotnet run https://example.com
```

**Output:**
```
Starting async operation...
Fetching: https://example.com
Content length: 1256 characters
```

## Best Practices

1. **Validate arguments** - Always check argument count and format
2. **Provide help** - Include a --help option to show usage
3. **Handle errors gracefully** - Use try-catch for parsing and provide clear error messages
4. **Return meaningful exit codes** - 0 for success, non-zero for errors
5. **Use options consistently** - Follow conventions like --option or -o
