# 1.1 A Tour of C#

## Description

C# (pronounced "C Sharp") is a modern, object-oriented, and type-safe programming language developed by Microsoft. It runs on the .NET platform and enables developers to build various types of applications including web, mobile, desktop, games, and cloud-based services.

Key features of C#:
- **Object-oriented**: Everything is an object with properties and methods
- **Type-safe**: Prevents type errors at compile time
- **Garbage collected**: Automatic memory management
- **Cross-platform**: Runs on Windows, Linux, and macOS via .NET
- **Rich standard library**: Extensive built-in functionality

## Code Examples

### Hello World - The Classic First Program

```csharp
using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
```

**Output:**
```
Hello, World!
```

### Variables and Data Types

```csharp
using System;

class DataTypesDemo
{
    static void Main()
    {
        // Integer types
        int age = 25;
        long population = 7800000000L;

        // Floating-point types
        double price = 19.99;
        float temperature = 36.6f;

        // Other types
        char grade = 'A';
        bool isStudent = true;
        string name = "Alice";

        Console.WriteLine($"Name: {name}");
        Console.WriteLine($"Age: {age}");
        Console.WriteLine($"Grade: {grade}");
        Console.WriteLine($"Is Student: {isStudent}");
        Console.WriteLine($"Price: ${price}");
    }
}
```

**Output:**
```
Name: Alice
Age: 25
Grade: A
Is Student: True
Price: $19.99
```

### Classes and Objects

```csharp
using System;

class Person
{
    // Properties
    public string Name { get; set; }
    public int Age { get; set; }

    // Constructor
    public Person(string name, int age)
    {
        Name = name;
        Age = age;
    }

    // Method
    public void Introduce()
    {
        Console.WriteLine($"Hi, I'm {Name} and I'm {Age} years old.");
    }
}

class Program
{
    static void Main()
    {
        Person person1 = new Person("John", 30);
        Person person2 = new Person("Emma", 28);

        person1.Introduce();
        person2.Introduce();
    }
}
```

**Output:**
```
Hi, I'm John and I'm 30 years old.
Hi, I'm Emma and I'm 28 years old.
```

### Control Flow

```csharp
using System;

class ControlFlowDemo
{
    static void Main()
    {
        // If-else statement
        int score = 85;

        if (score >= 90)
        {
            Console.WriteLine("Grade: A");
        }
        else if (score >= 80)
        {
            Console.WriteLine("Grade: B");
        }
        else
        {
            Console.WriteLine("Grade: C");
        }

        // For loop
        Console.WriteLine("\nCounting to 5:");
        for (int i = 1; i <= 5; i++)
        {
            Console.WriteLine(i);
        }

        // Foreach loop
        Console.WriteLine("\nFruits:");
        string[] fruits = { "Apple", "Banana", "Orange" };
        foreach (string fruit in fruits)
        {
            Console.WriteLine(fruit);
        }
    }
}
```

**Output:**
```
Grade: B

Counting to 5:
1
2
3
4
5

Fruits:
Apple
Banana
Orange
```

### Methods and Return Values

```csharp
using System;

class Calculator
{
    static int Add(int a, int b)
    {
        return a + b;
    }

    static double Multiply(double a, double b)
    {
        return a * b;
    }

    static void PrintResult(string operation, double result)
    {
        Console.WriteLine($"{operation}: {result}");
    }

    static void Main()
    {
        int sum = Add(10, 5);
        double product = Multiply(4.5, 2.0);

        PrintResult("10 + 5", sum);
        PrintResult("4.5 * 2.0", product);
    }
}
```

**Output:**
```
10 + 5: 15
4.5 * 2.0: 9
```

### Arrays and Collections

```csharp
using System;
using System.Collections.Generic;

class CollectionsDemo
{
    static void Main()
    {
        // Array
        int[] numbers = { 1, 2, 3, 4, 5 };
        Console.WriteLine($"First number: {numbers[0]}");
        Console.WriteLine($"Array length: {numbers.Length}");

        // List
        List<string> cities = new List<string>();
        cities.Add("New York");
        cities.Add("London");
        cities.Add("Tokyo");

        Console.WriteLine("\nCities:");
        foreach (string city in cities)
        {
            Console.WriteLine($"- {city}");
        }
    }
}
```

**Output:**
```
First number: 1
Array length: 5

Cities:
- New York
- London
- Tokyo
```
