# 7.1 Events

## Description

**Events** in C# are a special type of delegate that provides a way for a class to notify other classes when something interesting happens. Events are built on top of delegates and provide a secure, controlled way to implement the observer pattern.

### Key Characteristics:
- **Publisher-Subscriber Pattern**: One class publishes events, others subscribe
- **Type Safety**: Events are strongly typed and checked at compile time
- **Encapsulation**: External classes cannot directly invoke events
- **Multicast**: Multiple subscribers can listen to the same event
- **Thread Safety**: Built-in thread safety for event operations

### Event vs Delegate:
- **Events** are encapsulated delegates with restricted access
- **Delegates** can be invoked from outside the class
- **Events** can only be invoked from within the declaring class
- **Events** provide better encapsulation and security

## Code Examples

### Example 1: Basic Event Declaration and Usage

```csharp
using System;

// Event publisher class
class Button
{
    // Declare an event
    public event EventHandler Clicked;

    // Method to raise the event
    protected virtual void OnClicked()
    {
        Clicked?.Invoke(this, EventArgs.Empty);
    }

    // Simulate button click
    public void SimulateClick()
    {
        Console.WriteLine("Button clicked!");
        OnClicked();
    }
}

// Event subscriber class
class EventHandler
{
    public void OnButtonClicked(object sender, EventArgs e)
    {
        Console.WriteLine("Button click event handled!");
    }
}

class Program
{
    static void Main()
    {
        Button button = new Button();
        EventHandler handler = new EventHandler();

        // Subscribe to the event
        button.Clicked += handler.OnButtonClicked;

        // Simulate button clicks
        button.SimulateClick();
        button.SimulateClick();

        // Unsubscribe from the event
        button.Clicked -= handler.OnButtonClicked;

        // This click won't be handled
        button.SimulateClick();
    }
}
```

**Output:**
```
Button clicked!
Button click event handled!
Button clicked!
Button click event handled!
Button clicked!
```

### Example 2: Custom Event Arguments

```csharp
using System;

// Custom event arguments
class TemperatureChangedEventArgs : EventArgs
{
    public double OldTemperature { get; }
    public double NewTemperature { get; }
    public DateTime Timestamp { get; }

    public TemperatureChangedEventArgs(double oldTemp, double newTemp)
    {
        OldTemperature = oldTemp;
        NewTemperature = newTemp;
        Timestamp = DateTime.Now;
    }
}

// Event publisher
class TemperatureSensor
{
    private double _temperature;

    public double Temperature
    {
        get => _temperature;
        set
        {
            if (_temperature != value)
            {
                double oldTemp = _temperature;
                _temperature = value;
                OnTemperatureChanged(oldTemp, value);
            }
        }
    }

    // Event declaration with custom arguments
    public event EventHandler<TemperatureChangedEventArgs> TemperatureChanged;

    protected virtual void OnTemperatureChanged(double oldTemp, double newTemp)
    {
        TemperatureChanged?.Invoke(this, new TemperatureChangedEventArgs(oldTemp, newTemp));
    }
}

// Event subscriber
class TemperatureMonitor
{
    public void OnTemperatureChanged(object sender, TemperatureChangedEventArgs e)
    {
        Console.WriteLine($"Temperature changed from {e.OldTemperature}°C to {e.NewTemperature}°C");
        Console.WriteLine($"Change detected at: {e.Timestamp:HH:mm:ss}");

        if (e.NewTemperature > 30)
        {
            Console.WriteLine("WARNING: High temperature detected!");
        }
        else if (e.NewTemperature < 0)
        {
            Console.WriteLine("WARNING: Freezing temperature detected!");
        }
    }
}

class Program
{
    static void Main()
    {
        TemperatureSensor sensor = new TemperatureSensor();
        TemperatureMonitor monitor = new TemperatureMonitor();

        // Subscribe to the event
        sensor.TemperatureChanged += monitor.OnTemperatureChanged;

        // Change temperature (this will trigger events)
        sensor.Temperature = 25.5;
        sensor.Temperature = 32.1;
        sensor.Temperature = -5.0;
        sensor.Temperature = 20.0;

        // This won't trigger an event (same temperature)
        sensor.Temperature = 20.0;
    }
}
```

**Output:**
```
Temperature changed from 0°C to 25.5°C
Change detected at: 14:30:25
Temperature changed from 25.5°C to 32.1°C
Change detected at: 14:30:25
WARNING: High temperature detected!
Temperature changed from 32.1°C to -5°C
Change detected at: 14:30:25
WARNING: Freezing temperature detected!
Temperature changed from -5°C to 20°C
Change detected at: 14:30:25
```

### Example 3: Multiple Event Subscribers

```csharp
using System;
using System.Collections.Generic;

// Event arguments for user actions
class UserActionEventArgs : EventArgs
{
    public string Action { get; }
    public string UserId { get; }
    public DateTime Timestamp { get; }

    public UserActionEventArgs(string action, string userId)
    {
        Action = action;
        UserId = userId;
        Timestamp = DateTime.Now;
    }
}

// Event publisher
class UserManager
{
    public event EventHandler<UserActionEventArgs> UserLoggedIn;
    public event EventHandler<UserActionEventArgs> UserLoggedOut;
    public event EventHandler<UserActionEventArgs> UserActionPerformed;

    protected virtual void OnUserLoggedIn(string userId)
    {
        UserLoggedIn?.Invoke(this, new UserActionEventArgs("Login", userId));
    }

    protected virtual void OnUserLoggedOut(string userId)
    {
        UserLoggedOut?.Invoke(this, new UserActionEventArgs("Logout", userId));
    }

    protected virtual void OnUserActionPerformed(string action, string userId)
    {
        UserActionPerformed?.Invoke(this, new UserActionEventArgs(action, userId));
    }

    public void Login(string userId)
    {
        Console.WriteLine($"User {userId} logged in");
        OnUserLoggedIn(userId);
    }

    public void Logout(string userId)
    {
        Console.WriteLine($"User {userId} logged out");
        OnUserLoggedOut(userId);
    }

    public void PerformAction(string action, string userId)
    {
        Console.WriteLine($"User {userId} performed action: {action}");
        OnUserActionPerformed(action, userId);
    }
}

// Event subscribers
class Logger
{
    public void OnUserAction(object sender, UserActionEventArgs e)
    {
        Console.WriteLine($"  [LOG] {e.Timestamp:HH:mm:ss} - User {e.UserId}: {e.Action}");
    }
}

class SecurityMonitor
{
    public void OnUserLogin(object sender, UserActionEventArgs e)
    {
        Console.WriteLine($"  [SECURITY] Login detected for user: {e.UserId}");
    }

    public void OnUserAction(object sender, UserActionEventArgs e)
    {
        if (e.Action.Contains("admin") || e.Action.Contains("delete"))
        {
            Console.WriteLine($"  [SECURITY] Sensitive action detected: {e.Action} by {e.UserId}");
        }
    }
}

class AnalyticsTracker
{
    private Dictionary<string, int> actionCounts = new Dictionary<string, int>();

    public void OnUserAction(object sender, UserActionEventArgs e)
    {
        if (!actionCounts.ContainsKey(e.Action))
        {
            actionCounts[e.Action] = 0;
        }
        actionCounts[e.Action]++;

        Console.WriteLine($"  [ANALYTICS] Action '{e.Action}' count: {actionCounts[e.Action]}");
    }
}

class Program
{
    static void Main()
    {
        UserManager userManager = new UserManager();
        Logger logger = new Logger();
        SecurityMonitor security = new SecurityMonitor();
        AnalyticsTracker analytics = new AnalyticsTracker();

        // Subscribe multiple handlers to events
        userManager.UserLoggedIn += security.OnUserLogin;
        userManager.UserActionPerformed += logger.OnUserAction;
        userManager.UserActionPerformed += security.OnUserAction;
        userManager.UserActionPerformed += analytics.OnUserAction;

        // Simulate user activities
        userManager.Login("alice");
        userManager.PerformAction("view_profile", "alice");
        userManager.PerformAction("admin_access", "alice");
        userManager.PerformAction("delete_file", "alice");
        userManager.Logout("alice");

        Console.WriteLine("\n" + new string('-', 40));

        userManager.Login("bob");
        userManager.PerformAction("view_profile", "bob");
        userManager.PerformAction("edit_document", "bob");
        userManager.Logout("bob");
    }
}
```

**Output:**
```
User alice logged in
  [SECURITY] Login detected for user: alice
User alice performed action: view_profile
  [LOG] 14:30:25 - User alice: view_profile
  [ANALYTICS] Action 'view_profile' count: 1
User alice performed action: admin_access
  [LOG] 14:30:25 - User alice: admin_access
  [SECURITY] Sensitive action detected: admin_access by alice
  [ANALYTICS] Action 'admin_access' count: 1
User alice performed action: delete_file
  [LOG] 14:30:25 - User alice: delete_file
  [SECURITY] Sensitive action detected: delete_file by alice
  [ANALYTICS] Action 'delete_file' count: 1
User alice logged out

----------------------------------------
User bob logged in
  [SECURITY] Login detected for user: bob
User bob performed action: view_profile
  [LOG] 14:30:25 - User bob: view_profile
  [ANALYTICS] Action 'view_profile' count: 2
User bob performed action: edit_document
  [LOG] 14:30:30 - User bob: edit_document
  [ANALYTICS] Action 'edit_document' count: 1
User bob logged out
```

### Example 4: Event with Generic Type Parameters

```csharp
using System;

// Generic event arguments
class DataChangedEventArgs<T> : EventArgs
{
    public T OldValue { get; }
    public T NewValue { get; }
    public string PropertyName { get; }

    public DataChangedEventArgs(T oldValue, T newValue, string propertyName)
    {
        OldValue = oldValue;
        NewValue = newValue;
        PropertyName = propertyName;
    }
}

// Generic event publisher
class DataStore<T>
{
    private T _value;

    public T Value
    {
        get => _value;
        set
        {
            if (!Equals(_value, value))
            {
                T oldValue = _value;
                _value = value;
                OnValueChanged(oldValue, value);
            }
        }
    }

    // Generic event
    public event EventHandler<DataChangedEventArgs<T>> ValueChanged;

    protected virtual void OnValueChanged(T oldValue, T newValue)
    {
        ValueChanged?.Invoke(this, new DataChangedEventArgs<T>(oldValue, newValue, "Value"));
    }
}

// Event subscribers
class DataLogger<T>
{
    public void OnValueChanged(object sender, DataChangedEventArgs<T> e)
    {
        Console.WriteLine($"  [LOG] {e.PropertyName} changed from '{e.OldValue}' to '{e.NewValue}'");
    }
}

class DataValidator<T> where T : IComparable<T>
{
    public void OnValueChanged(object sender, DataChangedEventArgs<T> e)
    {
        if (e.NewValue.CompareTo(e.OldValue) > 0)
        {
            Console.WriteLine($"  [VALIDATOR] Value increased");
        }
        else if (e.NewValue.CompareTo(e.OldValue) < 0)
        {
            Console.WriteLine($"  [VALIDATOR] Value decreased");
        }
    }
}

class Program
{
    static void Main()
    {
        // Integer data store
        var intStore = new DataStore<int>();
        var intLogger = new DataLogger<int>();
        var intValidator = new DataValidator<int>();

        intStore.ValueChanged += intLogger.OnValueChanged;
        intStore.ValueChanged += intValidator.OnValueChanged;

        Console.WriteLine("Integer Data Store:");
        intStore.Value = 10;
        intStore.Value = 15;
        intStore.Value = 5;

        Console.WriteLine("\n" + new string('-', 30));

        // String data store
        var stringStore = new DataStore<string>();
        var stringLogger = new DataLogger<string>();

        stringStore.ValueChanged += stringLogger.OnValueChanged;

        Console.WriteLine("String Data Store:");
        stringStore.Value = "Hello";
        stringStore.Value = "World";
        stringStore.Value = "C# Events";
    }
}
```

**Output:**
```
Integer Data Store:
  [LOG] Value changed from '0' to '10'
  [VALIDATOR] Value increased
  [LOG] Value changed from '10' to '15'
  [VALIDATOR] Value increased
  [LOG] Value changed from '15' to '5'
  [VALIDATOR] Value decreased

------------------------------
String Data Store:
  [LOG] Value changed from '' to 'Hello'
  [LOG] Value changed from 'Hello' to 'World'
  [LOG] Value changed from 'World' to 'C# Events'
```

### Example 5: Event Access Control and Security

```csharp
using System;

// Event arguments with security context
class SecureEventArgs : EventArgs
{
    public string Message { get; }
    public string UserId { get; }
    public string SecurityLevel { get; }

    public SecureEventArgs(string message, string userId, string securityLevel)
    {
        Message = message;
        UserId = userId;
        SecurityLevel = securityLevel;
    }
}

// Secure event publisher
class SecureEventPublisher
{
    // Public event that anyone can subscribe to
    public event EventHandler<SecureEventArgs> PublicEvent;

    // Internal event only for same assembly
    internal event EventHandler<SecureEventArgs> InternalEvent;

    // Protected event for derived classes
    protected event EventHandler<SecureEventArgs> ProtectedEvent;

    // Private event for this class only
    private event EventHandler<SecureEventArgs> PrivateEvent;

    public void TriggerPublicEvent(string message, string userId, string securityLevel)
    {
        Console.WriteLine($"Triggering public event: {message}");
        PublicEvent?.Invoke(this, new SecureEventArgs(message, userId, securityLevel));
    }

    internal void TriggerInternalEvent(string message, string userId, string securityLevel)
    {
        Console.WriteLine($"Triggering internal event: {message}");
        InternalEvent?.Invoke(this, new SecureEventArgs(message, userId, securityLevel));
    }

    protected void TriggerProtectedEvent(string message, string userId, string securityLevel)
    {
        Console.WriteLine($"Triggering protected event: {message}");
        ProtectedEvent?.Invoke(this, new SecureEventArgs(message, userId, securityLevel));
    }

    private void TriggerPrivateEvent(string message, string userId, string securityLevel)
    {
        Console.WriteLine($"Triggering private event: {message}");
        PrivateEvent?.Invoke(this, new SecureEventArgs(message, userId, securityLevel));
    }

    public void TriggerAllEvents(string message, string userId, string securityLevel)
    {
        TriggerPublicEvent(message, userId, securityLevel);
        TriggerInternalEvent(message, userId, securityLevel);
        TriggerProtectedEvent(message, userId, securityLevel);
        TriggerPrivateEvent(message, userId, securityLevel);
    }
}

// Event subscriber with security filtering
class SecureEventHandler
{
    public void OnSecureEvent(object sender, SecureEventArgs e)
    {
        Console.WriteLine($"  [SECURE] User: {e.UserId}, Level: {e.SecurityLevel}, Message: {e.Message}");
        
        // Filter based on security level
        if (e.SecurityLevel == "HIGH")
        {
            Console.WriteLine($"    [ALERT] High security event detected!");
        }
    }
}

class Program
{
    static void Main()
    {
        SecureEventPublisher publisher = new SecureEventPublisher();
        SecureEventHandler handler = new SecureEventHandler();

        // Subscribe to public event
        publisher.PublicEvent += handler.OnSecureEvent;

        // Trigger events
        publisher.TriggerPublicEvent("User login", "alice", "LOW");
        publisher.TriggerPublicEvent("Admin access", "bob", "HIGH");
        publisher.TriggerPublicEvent("Data export", "charlie", "MEDIUM");

        Console.WriteLine("\n" + new string('-', 40));

        // Demonstrate access control
        Console.WriteLine("Event Access Control:");
        Console.WriteLine("- Public events: Can be subscribed from anywhere");
        Console.WriteLine("- Internal events: Only from same assembly");
        Console.WriteLine("- Protected events: Only from derived classes");
        Console.WriteLine("- Private events: Only from same class");
    }
}
```

**Output:**
```
Triggering public event: User login
  [SECURE] User: alice, Level: LOW, Message: User login
Triggering public event: Admin access
  [SECURE] User: bob, Level: HIGH, Message: Admin access
    [ALERT] High security event detected!
Triggering public event: Data export
  [SECURE] User: charlie, Level: MEDIUM, Message: Data export

----------------------------------------
Event Access Control:
- Public events: Can be subscribed from anywhere
- Internal events: Only from same assembly
- Protected events: Only from derived classes
- Private events: Only from same class
```

## Key Points

### Event Declaration
- **Syntax**: `public event EventHandler EventName;`
- **Access modifiers**: public, internal, protected, private
- **Generic events**: `public event EventHandler<T> EventName;`

### Event Raising
- **Null check**: Always use `?.` operator
- **Protected virtual**: Allow derived classes to override
- **Event arguments**: Pass meaningful data to subscribers

### Event Subscription
- **Subscribe**: `publisher.EventName += handler.Method;`
- **Unsubscribe**: `publisher.EventName -= handler.Method;`
- **Multiple subscribers**: Events support multicast

### Best Practices
- **Event naming**: Use past tense (e.g., `Clicked`, `Changed`)
- **Event arguments**: Inherit from `EventArgs` or `EventArgs<T>`
- **Null safety**: Always check for null before invoking
- **Thread safety**: Events are thread-safe for subscription/unsubscription
- **Memory management**: Unsubscribe to prevent memory leaks
