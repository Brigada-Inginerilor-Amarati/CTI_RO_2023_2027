# 7.3 Handle and Raise Events

## Description

This section covers the essential patterns for handling and raising events in C#. Understanding how to properly handle events and raise them is crucial for building robust, event-driven applications.

### Key Concepts:
- **Event Handling**: Responding to events with appropriate logic
- **Event Raising**: Triggering events from within classes
- **Event Arguments**: Passing meaningful data to event handlers
- **Exception Handling**: Managing exceptions in event handlers
- **Thread Safety**: Ensuring events work correctly in multi-threaded environments

### Common Patterns:
- **Protected Virtual Methods**: Allow derived classes to override event raising
- **Null Checking**: Always check for null before raising events
- **Exception Handling**: Implement robust error handling in event handlers
- **Async Event Handling**: Handle events asynchronously when appropriate

## Code Examples

### Example 1: Basic Event Handling and Raising

```csharp
using System;

// Event arguments
class OrderEventArgs : EventArgs
{
    public string OrderId { get; }
    public decimal Amount { get; }
    public DateTime OrderDate { get; }

    public OrderEventArgs(string orderId, decimal amount)
    {
        OrderId = orderId;
        Amount = amount;
        OrderDate = DateTime.Now;
    }
}

// Event publisher
class OrderProcessor
{
    // Event declarations
    public event EventHandler<OrderEventArgs> OrderCreated;
    public event EventHandler<OrderEventArgs> OrderProcessed;
    public event EventHandler<OrderEventArgs> OrderCancelled;

    private string _currentOrderId;
    private decimal _currentAmount;

    public void CreateOrder(string orderId, decimal amount)
    {
        _currentOrderId = orderId;
        _currentAmount = amount;

        Console.WriteLine($"Creating order: {orderId} for ${amount}");
        
        // Raise the OrderCreated event
        OnOrderCreated(new OrderEventArgs(orderId, amount));
    }

    public void ProcessOrder()
    {
        if (string.IsNullOrEmpty(_currentOrderId))
        {
            Console.WriteLine("No order to process");
            return;
        }

        Console.WriteLine($"Processing order: {_currentOrderId}");
        
        // Simulate processing
        System.Threading.Thread.Sleep(100);
        
        // Raise the OrderProcessed event
        OnOrderProcessed(new OrderEventArgs(_currentOrderId, _currentAmount));
    }

    public void CancelOrder()
    {
        if (string.IsNullOrEmpty(_currentOrderId))
        {
            Console.WriteLine("No order to cancel");
            return;
        }

        Console.WriteLine($"Cancelling order: {_currentOrderId}");
        
        // Raise the OrderCancelled event
        OnOrderCancelled(new OrderEventArgs(_currentOrderId, _currentAmount));
        
        _currentOrderId = null;
        _currentAmount = 0;
    }

    // Protected virtual methods for raising events
    protected virtual void OnOrderCreated(OrderEventArgs e)
    {
        OrderCreated?.Invoke(this, e);
    }

    protected virtual void OnOrderProcessed(OrderEventArgs e)
    {
        OrderProcessed?.Invoke(this, e);
    }

    protected virtual void OnOrderCancelled(OrderEventArgs e)
    {
        OrderCancelled?.Invoke(this, e);
    }
}

// Event handlers
class OrderLogger
{
    public void OnOrderCreated(object sender, OrderEventArgs e)
    {
        Console.WriteLine($"  [LOG] Order created: {e.OrderId} at {e.OrderDate:HH:mm:ss}");
    }

    public void OnOrderProcessed(object sender, OrderEventArgs e)
    {
        Console.WriteLine($"  [LOG] Order processed: {e.OrderId} for ${e.Amount}");
    }

    public void OnOrderCancelled(object sender, OrderEventArgs e)
    {
        Console.WriteLine($"  [LOG] Order cancelled: {e.OrderId}");
    }
}

class InventoryManager
{
    public void OnOrderCreated(object sender, OrderEventArgs e)
    {
        Console.WriteLine($"  [INVENTORY] Reserving items for order: {e.OrderId}");
    }

    public void OnOrderProcessed(object sender, OrderEventArgs e)
    {
        Console.WriteLine($"  [INVENTORY] Items shipped for order: {e.OrderId}");
    }

    public void OnOrderCancelled(object sender, OrderEventArgs e)
    {
        Console.WriteLine($"  [INVENTORY] Items released for cancelled order: {e.OrderId}");
    }
}

class Program
{
    static void Main()
    {
        OrderProcessor processor = new OrderProcessor();
        OrderLogger logger = new OrderLogger();
        InventoryManager inventory = new InventoryManager();

        // Subscribe to events
        processor.OrderCreated += logger.OnOrderCreated;
        processor.OrderCreated += inventory.OnOrderCreated;
        processor.OrderProcessed += logger.OnOrderProcessed;
        processor.OrderProcessed += inventory.OnOrderProcessed;
        processor.OrderCancelled += logger.OnOrderCancelled;
        processor.OrderCancelled += inventory.OnOrderCancelled;

        // Process orders
        processor.CreateOrder("ORD001", 99.99m);
        processor.ProcessOrder();

        Console.WriteLine();

        processor.CreateOrder("ORD002", 149.50m);
        processor.CancelOrder();
    }
}
```

**Output:**
```

  [LOG] Order created: ORD001 at 14:30:25
  [INVENTORY] Reserving items for order: ORD001
Processing order: ORD001
  [LOG] Order processed: ORD001 for $99.99
  [INVENTORY] Items shipped for order: ORD001

Creating order: ORD002 for $149.50
  [LOG] Order created: ORD002 at 14:30:25
  [INVENTORY] Reserving items for order: ORD002
Cancelling order: ORD002
```

### Example 2: Exception Handling in Event Handlers

```csharp
using System;

// Event arguments
class DataProcessingEventArgs : EventArgs
{
    public string Data { get; }
    public bool Success { get; set; }
    public string ErrorMessage { get; set; }

    public DataProcessingEventArgs(string data)
    {
        Data = data;
        Success = true;
    }
}

// Event publisher with exception handling
class DataProcessor
{
    public event EventHandler<DataProcessingEventArgs> DataProcessed;
    public event EventHandler<DataProcessingEventArgs> ProcessingError;

    public void ProcessData(string data)
    {
        var args = new DataProcessingEventArgs(data);
        
        try
        {
            Console.WriteLine($"Processing data: {data}");
            
            // Simulate processing
            if (data.Contains("error"))
            {
                throw new InvalidOperationException("Simulated processing error");
            }
            
            // Raise success event
            OnDataProcessed(args);
        }
        catch (Exception ex)
        {
            args.Success = false;
            args.ErrorMessage = ex.Message;
            
            // Raise error event
            OnProcessingError(args);
        }
    }

    protected virtual void OnDataProcessed(DataProcessingEventArgs e)
    {
        DataProcessed?.Invoke(this, e);
    }

    protected virtual void OnProcessingError(DataProcessingEventArgs e)
    {
        ProcessingError?.Invoke(this, e);
    }
}

// Event handlers with different error handling strategies
class SafeEventHandler
{
    public void OnDataProcessed(object sender, DataProcessingEventArgs e)
    {
        try
        {
            Console.WriteLine($"  [SAFE] Data processed successfully: {e.Data}");
        }
        catch (Exception ex)
        {
            Console.WriteLine($"  [SAFE] Error in handler: {ex.Message}");
        }
    }

    public void OnProcessingError(object sender, DataProcessingEventArgs e)
    {
        try
        {
            Console.WriteLine($"  [SAFE] Processing error: {e.ErrorMessage}");
        }
        catch (Exception ex)
        {
            Console.WriteLine($"  [SAFE] Error in error handler: {ex.Message}");
        }
    }
}

class UnsafeEventHandler
{
    public void OnDataProcessed(object sender, DataProcessingEventArgs e)
    {
        Console.WriteLine($"  [UNSAFE] Data processed: {e.Data}");
        
        // This will throw an exception
        if (e.Data.Contains("crash"))
        {
            throw new InvalidOperationException("Handler crashed!");
        }
    }

    public void OnProcessingError(object sender, DataProcessingEventArgs e)
    {
        Console.WriteLine($"  [UNSAFE] Error: {e.ErrorMessage}");
    }
}

class Program
{
    static void Main()
    {
        DataProcessor processor = new DataProcessor();
        SafeEventHandler safeHandler = new SafeEventHandler();
        UnsafeEventHandler unsafeHandler = new UnsafeEventHandler();

        // Subscribe handlers
        processor.DataProcessed += safeHandler.OnDataProcessed;
        processor.DataProcessed += unsafeHandler.OnDataProcessed;
        processor.ProcessingError += safeHandler.OnProcessingError;
        processor.ProcessingError += unsafeHandler.OnProcessingError;

        Console.WriteLine("=== Safe Processing ===");
        processor.ProcessData("normal data");

        Console.WriteLine("\n=== Processing with Error ===");
        processor.ProcessData("data with error");

        Console.WriteLine("\n=== Handler Exception ===");
        processor.ProcessData("data with crash");
    }
}
```

**Output:**
```
=== Safe Processing ===
Processing data: normal data
  [SAFE] Data processed successfully: normal data
  [UNSAFE] Data processed: normal data

=== Processing with Error ===
Processing data: data with error
  [SAFE] Processing error: Simulated processing error
  [UNSAFE] Error: Simulated processing error

=== Handler Exception ===
Processing data: data with crash
  [SAFE] Data processed successfully: data with crash
  [UNSAFE] Data processed: data with crash
```

### Example 3: Async Event Handling

```csharp
using System;
using System.Threading.Tasks;

// Event arguments for async operations
class FileOperationEventArgs : EventArgs
{
    public string FileName { get; }
    public long FileSize { get; }
    public bool Success { get; set; }
    public string ErrorMessage { get; set; }

    public FileOperationEventArgs(string fileName, long fileSize)
    {
        FileName = fileName;
        FileSize = fileSize;
        Success = true;
    }
}

// Event publisher
class FileProcessor
{
    public event EventHandler<FileOperationEventArgs> FileProcessed;
    public event EventHandler<FileOperationEventArgs> ProcessingCompleted;

    public async Task ProcessFileAsync(string fileName, long fileSize)
    {
        var args = new FileOperationEventArgs(fileName, fileSize);
        
        try
        {
            Console.WriteLine($"Processing file: {fileName} ({fileSize} bytes)");
            
            // Simulate async processing
            await Task.Delay(1000);
            
            // Raise event
            OnFileProcessed(args);
        }
        catch (Exception ex)
        {
            args.Success = false;
            args.ErrorMessage = ex.Message;
            OnFileProcessed(args);
        }
        finally
        {
            OnProcessingCompleted(args);
        }
    }

    protected virtual void OnFileProcessed(FileOperationEventArgs e)
    {
        FileProcessed?.Invoke(this, e);
    }

    protected virtual void OnProcessingCompleted(FileOperationEventArgs e)
    {
        ProcessingCompleted?.Invoke(this, e);
    }
}

// Async event handlers
class AsyncFileHandler
{
    public async void OnFileProcessed(object sender, FileOperationEventArgs e)
    {
        try
        {
            Console.WriteLine($"  [ASYNC] Processing file: {e.FileName}");
            
            // Simulate async work
            await Task.Delay(500);
            
            if (e.Success)
            {
                Console.WriteLine($"  [ASYNC] File processed successfully: {e.FileName}");
            }
            else
            {
                Console.WriteLine($"  [ASYNC] File processing failed: {e.ErrorMessage}");
            }
        }
        catch (Exception ex)
        {
            Console.WriteLine($"  [ASYNC] Error in handler: {ex.Message}");
        }
    }

    public async void OnProcessingCompleted(object sender, FileOperationEventArgs e)
    {
        try
        {
            Console.WriteLine($"  [ASYNC] Processing completed for: {e.FileName}");
            
            // Simulate cleanup work
            await Task.Delay(200);
            
            Console.WriteLine($"  [ASYNC] Cleanup completed for: {e.FileName}");
        }
        catch (Exception ex)
        {
            Console.WriteLine($"  [ASYNC] Error in completion handler: {ex.Message}");
        }
    }
}

class Program
{
    static async Task Main()
    {
        FileProcessor processor = new FileProcessor();
        AsyncFileHandler handler = new AsyncFileHandler();

        // Subscribe to events
        processor.FileProcessed += handler.OnFileProcessed;
        processor.ProcessingCompleted += handler.OnProcessingCompleted;

        Console.WriteLine("=== Async File Processing ===");
        
        // Process files asynchronously
        var tasks = new[]
        {
            processor.ProcessFileAsync("document1.pdf", 1024000),
            processor.ProcessFileAsync("image1.jpg", 2048000),
            processor.ProcessFileAsync("video1.mp4", 10485760)
        };

        await Task.WhenAll(tasks);
        
        Console.WriteLine("\nAll files processed!");
    }
}
```

### Example 4: Thread-Safe Event Handling

```csharp
using System;
using System.Threading;
using System.Threading.Tasks;

// Thread-safe event publisher
class ThreadSafeEventPublisher
{
    private readonly object _lockObject = new object();
    private EventHandler<string> _dataChanged;

    public event EventHandler<string> DataChanged
    {
        add
        {
            lock (_lockObject)
            {
                _dataChanged += value;
            }
        }
        remove
        {
            lock (_lockObject)
            {
                _dataChanged -= value;
            }
        }
    }

    public void ChangeData(string newData)
    {
        Console.WriteLine($"Changing data to: {newData}");
        
        // Raise event in a thread-safe manner
        EventHandler<string> handlers;
        lock (_lockObject)
        {
            handlers = _dataChanged;
        }
        
        handlers?.Invoke(this, newData);
    }
}

// Thread-safe event handler
class ThreadSafeEventHandler
{
    private readonly string _name;
    private readonly object _lockObject = new object();

    public ThreadSafeEventHandler(string name)
    {
        _name = name;
    }

    public void OnDataChanged(object sender, string data)
    {
        lock (_lockObject)
        {
            Console.WriteLine($"  [{_name}] Data changed to: {data} (Thread: {Thread.CurrentThread.ManagedThreadId})");
        }
    }
}

class Program
{
    static void Main()
    {
        ThreadSafeEventPublisher publisher = new ThreadSafeEventPublisher();
        ThreadSafeEventHandler handler1 = new ThreadSafeEventHandler("Handler1");
        ThreadSafeEventHandler handler2 = new ThreadSafeEventHandler("Handler2");

        // Subscribe to events
        publisher.DataChanged += handler1.OnDataChanged;
        publisher.DataChanged += handler2.OnDataChanged;

        Console.WriteLine("=== Thread-Safe Event Handling ===");
        
        // Test single-threaded
        publisher.ChangeData("Data1");
        publisher.ChangeData("Data2");

        Console.WriteLine("\n=== Multi-Threaded Event Handling ===");
        
        // Test multi-threaded
        var tasks = new Task[5];
        for (int i = 0; i < 5; i++)
        {
            int index = i;
            tasks[i] = Task.Run(() =>
            {
                publisher.ChangeData($"ThreadData{index}");
            });
        }

        Task.WaitAll(tasks);
        
        Console.WriteLine("\nAll tasks completed!");
    }
}
```

### Example 5: Event Handling with Validation

```csharp
using System;

// Event arguments with validation
class ValidationEventArgs : EventArgs
{
    public string Data { get; }
    public bool IsValid { get; set; }
    public string ValidationMessage { get; set; }

    public ValidationEventArgs(string data)
    {
        Data = data;
        IsValid = true;
        ValidationMessage = "Valid";
    }
}

// Event publisher with validation
class DataValidator
{
    public event EventHandler<ValidationEventArgs> DataValidated;
    public event EventHandler<ValidationEventArgs> ValidationFailed;

    public void ValidateData(string data)
    {
        var args = new ValidationEventArgs(data);
        
        Console.WriteLine($"Validating data: {data}");
        
        // Perform validation
        if (string.IsNullOrEmpty(data))
        {
            args.IsValid = false;
            args.ValidationMessage = "Data cannot be empty";
        }
        else if (data.Length < 3)
        {
            args.IsValid = false;
            args.ValidationMessage = "Data must be at least 3 characters";
        }
        else if (data.Contains("invalid"))
        {
            args.IsValid = false;
            args.ValidationMessage = "Data contains invalid content";
        }

        // Raise appropriate event
        if (args.IsValid)
        {
            OnDataValidated(args);
        }
        else
        {
            OnValidationFailed(args);
        }
    }

    protected virtual void OnDataValidated(ValidationEventArgs e)
    {
        DataValidated?.Invoke(this, e);
    }

    protected virtual void OnValidationFailed(ValidationEventArgs e)
    {
        ValidationFailed?.Invoke(this, e);
    }
}

// Event handlers
class ValidationLogger
{
    public void OnDataValidated(object sender, ValidationEventArgs e)
    {
        Console.WriteLine($"  [LOG] Validation passed: {e.Data} - {e.ValidationMessage}");
    }

    public void OnValidationFailed(object sender, ValidationEventArgs e)
    {
        Console.WriteLine($"  [LOG] Validation failed: {e.Data} - {e.ValidationMessage}");
    }
}

class ValidationNotifier
{
    public void OnDataValidated(object sender, ValidationEventArgs e)
    {
        Console.WriteLine($"  [NOTIFY] Data is valid: {e.Data}");
    }

    public void OnValidationFailed(object sender, ValidationEventArgs e)
    {
        Console.WriteLine($"  [NOTIFY] Data is invalid: {e.Data} - {e.ValidationMessage}");
    }
}

class Program
{
    static void Main()
    {
        DataValidator validator = new DataValidator();
        ValidationLogger logger = new ValidationLogger();
        ValidationNotifier notifier = new ValidationNotifier();

        // Subscribe to events
        validator.DataValidated += logger.OnDataValidated;
        validator.DataValidated += notifier.OnDataValidated;
        validator.ValidationFailed += logger.OnValidationFailed;
        validator.ValidationFailed += notifier.OnValidationFailed;

        Console.WriteLine("=== Data Validation ===");
        
        // Test various data
        validator.ValidateData("valid data");
        validator.ValidateData("ab"); // Too short
        validator.ValidateData(""); // Empty
        validator.ValidateData("invalid data"); // Contains "invalid"
        validator.ValidateData("good data");
    }
}
```

```
  [LOG] Validation failed: ab - Data must be at least 3 characters
  [NOTIFY] Data is invalid: ab - Data must be at least 3 characters

  [NOTIFY] Data is invalid:  - Data cannot be empty
```

## Key Points

### Event Raising Patterns
- **Protected Virtual**: Use `protected virtual` methods for raising events
- **Null Checking**: Always check for null before invoking events
- **Event Arguments**: Pass meaningful data through event arguments
- **Exception Handling**: Handle exceptions in event raising

### Event Handling Patterns
- **Safe Handling**: Implement try-catch in event handlers
- **Async Handling**: Use async/await for long-running operations
- **Thread Safety**: Ensure thread-safe event handling
- **Validation**: Implement validation in event handlers

### Best Practices
- **Exception Safety**: Don't let exceptions in handlers break the publisher
- **Performance**: Consider performance implications of many handlers
- **Memory Management**: Properly unsubscribe to prevent memory leaks
- **Documentation**: Document event behavior and expected arguments
- **Testing**: Test event handling thoroughly
