# 7.5 Implementing Interface Events

## Description

Events can be declared in interfaces and implemented by classes that implement those interfaces. This provides a contract for event-driven behavior and enables polymorphism with events. Understanding how to implement interface events is crucial for building flexible, event-driven architectures.

### Key Concepts:
- **Interface Events**: Events declared in interfaces
- **Event Implementation**: How classes implement interface events
- **Explicit Implementation**: Implementing interface events explicitly
- **Event Polymorphism**: Using events through interface references
- **Event Contracts**: Defining event behavior through interfaces

### Common Patterns:
- **Interface Events**: Declare events in interfaces
- **Explicit Implementation**: Use explicit interface implementation for events
- **Event Polymorphism**: Subscribe to events through interface references
- **Multiple Interfaces**: Implement events from multiple interfaces

## Code Examples

### Example 1: Basic Interface Event Implementation

```csharp
using System;

// Interface with events
interface INotifyPropertyChanged
{
    event EventHandler<string> PropertyChanged;
}

interface INotifyDataErrorInfo
{
    event EventHandler<string> DataError;
}

// Class implementing interface events
class Person : INotifyPropertyChanged, INotifyDataErrorInfo
{
    private string _name;
    private int _age;

    // Implement interface events
    public event EventHandler<string> PropertyChanged;
    public event EventHandler<string> DataError;

    public string Name
    {
        get => _name;
        set
        {
            if (_name != value)
            {
                _name = value;
                OnPropertyChanged(nameof(Name));
            }
        }
    }

    public int Age
    {
        get => _age;
        set
        {
            if (value < 0 || value > 150)
            {
                OnDataError($"Invalid age: {value}");
                return;
            }

            if (_age != value)
            {
                _age = value;
                OnPropertyChanged(nameof(Age));
            }
        }
    }

    protected virtual void OnPropertyChanged(string propertyName)
    {
        PropertyChanged?.Invoke(this, propertyName);
    }

    protected virtual void OnDataError(string errorMessage)
    {
        DataError?.Invoke(this, errorMessage);
    }
}

// Event handlers
class PropertyChangeHandler
{
    public void OnPropertyChanged(object sender, string propertyName)
    {
        Console.WriteLine($"  [PROPERTY] {propertyName} changed");
    }

    public void OnDataError(object sender, string errorMessage)
    {
        Console.WriteLine($"  [ERROR] {errorMessage}");
    }
}

class Program
{
    static void Main()
    {
        Person person = new Person();
        PropertyChangeHandler handler = new PropertyChangeHandler();

        // Subscribe to events through interface references
        INotifyPropertyChanged propertyNotifier = person;
        INotifyDataErrorInfo errorNotifier = person;

        propertyNotifier.PropertyChanged += handler.OnPropertyChanged;
        errorNotifier.DataError += handler.OnDataError;

        Console.WriteLine("=== Property Changes ===");
        
        // Test property changes
        person.Name = "Alice";
        person.Age = 25;
        person.Age = 30;

        Console.WriteLine("\n=== Data Validation ===");
        
        // Test data validation
        person.Age = -5; // Invalid age
        person.Age = 200; // Invalid age
        person.Age = 35; // Valid age
    }
}
```

**Output:**
```
=== Property Changes ===
  [PROPERTY] Name changed
  [PROPERTY] Age changed
  [PROPERTY] Age changed

=== Data Validation ===
  [ERROR] Invalid age: -5
  [ERROR] Invalid age: 200
  [PROPERTY] Age changed
```

### Example 2: Explicit Interface Event Implementation

```csharp
using System;

// Interface with events
interface IObservable
{
    event EventHandler<string> DataChanged;
    event EventHandler<string> ErrorOccurred;
}

interface ILoggable
{
    event EventHandler<string> LogMessage;
}

// Class with explicit interface implementation
class DataService : IObservable, ILoggable
{
    private string _data;
    private readonly object _lockObject = new object();

    // Explicit interface implementation
    event EventHandler<string> IObservable.DataChanged
    {
        add
        {
            lock (_lockObject)
            {
                _dataChanged += value;
            }
        }
        remove
        {
            lock (_lockObject)
            {
                _dataChanged -= value;
            }
        }
    }

    event EventHandler<string> IObservable.ErrorOccurred
    {
        add
        {
            lock (_lockObject)
            {
                _errorOccurred += value;
            }
        }
        remove
        {
            lock (_lockObject)
            {
                _errorOccurred -= value;
            }
        }
    }

    event EventHandler<string> ILoggable.LogMessage
    {
        add
        {
            lock (_lockObject)
            {
                _logMessage += value;
            }
        }
        remove
        {
            lock (_lockObject)
            {
                _logMessage -= value;
            }
        }
    }

    // Private event fields
    private EventHandler<string> _dataChanged;
    private EventHandler<string> _errorOccurred;
    private EventHandler<string> _logMessage;

    public string Data
    {
        get => _data;
        set
        {
            if (_data != value)
            {
                _data = value;
                OnDataChanged(value);
            }
        }
    }

    public void ProcessData(string data)
    {
        try
        {
            OnLogMessage($"Processing data: {data}");
            
            if (string.IsNullOrEmpty(data))
            {
                throw new ArgumentException("Data cannot be empty");
            }

            Data = data;
            OnLogMessage($"Data processed successfully: {data}");
        }
        catch (Exception ex)
        {
            OnErrorOccurred($"Error processing data: {ex.Message}");
        }
    }

    private void OnDataChanged(string data)
    {
        _dataChanged?.Invoke(this, data);
    }

    private void OnErrorOccurred(string error)
    {
        _errorOccurred?.Invoke(this, error);
    }

    private void OnLogMessage(string message)
    {
        _logMessage?.Invoke(this, message);
    }
}

// Event handlers
class DataObserver
{
    public void OnDataChanged(object sender, string data)
    {
        Console.WriteLine($"  [OBSERVER] Data changed: {data}");
    }

    public void OnErrorOccurred(object sender, string error)
    {
        Console.WriteLine($"  [OBSERVER] Error: {error}");
    }

    public void OnLogMessage(object sender, string message)
    {
        Console.WriteLine($"  [OBSERVER] Log: {message}");
    }
}

class Program
{
    static void Main()
    {
        DataService service = new DataService();
        DataObserver observer = new DataObserver();

        // Subscribe through interface references
        IObservable observable = service;
        ILoggable loggable = service;

        observable.DataChanged += observer.OnDataChanged;
        observable.ErrorOccurred += observer.OnErrorOccurred;
        loggable.LogMessage += observer.OnLogMessage;

        Console.WriteLine("=== Data Processing ===");
        
        // Test data processing
        service.ProcessData("Valid data");
        service.ProcessData(""); // Empty data
        service.ProcessData("Another valid data");

        Console.WriteLine("\n=== Direct Property Access ===");
        
        // Test direct property access
        service.Data = "Direct assignment";
    }
}
```

**Output:**
```
=== Data Processing ===
  [OBSERVER] Log: Processing data: Valid data
  [OBSERVER] Data changed: Valid data
  [OBSERVER] Log: Data processed successfully: Valid data
  [OBSERVER] Log: Processing data: 
  [OBSERVER] Error: Error processing data: Data cannot be empty
  [OBSERVER] Log: Processing data: Another valid data
  [OBSERVER] Data changed: Another valid data
  [OBSERVER] Log: Data processed successfully: Another valid data

=== Direct Property Access ===
  [OBSERVER] Data changed: Direct assignment
```

### Example 3: Event Polymorphism with Interfaces

```csharp
using System;
using System.Collections.Generic;

// Interface with events
interface IShape
{
    event EventHandler<string> ShapeMoved;
    event EventHandler<double> AreaCalculated;
    
    void Move(string direction);
    double CalculateArea();
}

// Circle implementing IShape
class Circle : IShape
{
    private double _radius;
    private double _x, _y;

    public event EventHandler<string> ShapeMoved;
    public event EventHandler<double> AreaCalculated;

    public Circle(double radius)
    {
        _radius = radius;
        _x = 0;
        _y = 0;
    }

    public void Move(string direction)
    {
        Console.WriteLine($"Circle moving {direction}");
        OnShapeMoved(direction);
        
        // Update position
        switch (direction.ToLower())
        {
            case "up":
                _y += 1;
                break;
            case "down":
                _y -= 1;
                break;
            case "left":
                _x -= 1;
                break;
            case "right":
                _x += 1;
                break;
        }
    }

    public double CalculateArea()
    {
        double area = Math.PI * _radius * _radius;
        OnAreaCalculated(area);
        return area;
    }

    protected virtual void OnShapeMoved(string direction)
    {
        ShapeMoved?.Invoke(this, direction);
    }

    protected virtual void OnAreaCalculated(double area)
    {
        AreaCalculated?.Invoke(this, area);
    }
}

// Rectangle implementing IShape
class Rectangle : IShape
{
    private double _width, _height;
    private double _x, _y;

    public event EventHandler<string> ShapeMoved;
    public event EventHandler<double> AreaCalculated;

    public Rectangle(double width, double height)
    {
        _width = width;
        _height = height;
        _x = 0;
        _y = 0;
    }

    public void Move(string direction)
    {
        Console.WriteLine($"Rectangle moving {direction}");
        OnShapeMoved(direction);
        
        // Update position
        switch (direction.ToLower())
        {
            case "up":
                _y += 1;
                break;
            case "down":
                _y -= 1;
                break;
            case "left":
                _x -= 1;
                break;
            case "right":
                _x += 1;
                break;
        }
    }

    public double CalculateArea()
    {
        double area = _width * _height;
        OnAreaCalculated(area);
        return area;
    }

    protected virtual void OnShapeMoved(string direction)
    {
        ShapeMoved?.Invoke(this, direction);
    }

    protected virtual void OnAreaCalculated(double area)
    {
        AreaCalculated?.Invoke(this, area);
    }
}

// Event handler
class ShapeObserver
{
    public void OnShapeMoved(object sender, string direction)
    {
        Console.WriteLine($"  [OBSERVER] Shape moved: {direction}");
    }

    public void OnAreaCalculated(object sender, double area)
    {
        Console.WriteLine($"  [OBSERVER] Area calculated: {area:F2}");
    }
}

class Program
{
    static void Main()
    {
        // Create shapes
        Circle circle = new Circle(5.0);
        Rectangle rectangle = new Rectangle(4.0, 6.0);

        // Create observer
        ShapeObserver observer = new ShapeObserver();

        // Subscribe to events through interface references
        IShape circleShape = circle;
        IShape rectangleShape = rectangle;

        circleShape.ShapeMoved += observer.OnShapeMoved;
        circleShape.AreaCalculated += observer.OnAreaCalculated;
        rectangleShape.ShapeMoved += observer.OnShapeMoved;
        rectangleShape.AreaCalculated += observer.OnAreaCalculated;

        Console.WriteLine("=== Circle Operations ===");
        
        // Test circle operations
        circleShape.CalculateArea();
        circleShape.Move("right");
        circleShape.Move("up");

        Console.WriteLine("\n=== Rectangle Operations ===");
        
        // Test rectangle operations
        rectangleShape.CalculateArea();
        rectangleShape.Move("left");
        rectangleShape.Move("down");

        Console.WriteLine("\n=== Polymorphic Operations ===");
        
        // Test polymorphic operations
        List<IShape> shapes = new List<IShape> { circle, rectangle };
        
        foreach (var shape in shapes)
        {
            shape.CalculateArea();
            shape.Move("forward");
        }
    }
}
```

**Output:**
```
=== Circle Operations ===
  [OBSERVER] Area calculated: 78.54
Circle moving right
  [OBSERVER] Shape moved: right
Circle moving up
  [OBSERVER] Shape moved: up

=== Rectangle Operations ===
  [OBSERVER] Area calculated: 24.00
Rectangle moving left
  [OBSERVER] Shape moved: left
Rectangle moving down
  [OBSERVER] Shape moved: down

=== Polymorphic Operations ===
  [OBSERVER] Area calculated: 78.54
Circle moving forward
  [OBSERVER] Shape moved: forward
  [OBSERVER] Area calculated: 24.00
Rectangle moving forward
  [OBSERVER] Shape moved: forward
```

### Example 4: Multiple Interface Events

```csharp
using System;

// Multiple interfaces with events
interface IDataProvider
{
    event EventHandler<string> DataReceived;
    event EventHandler<string> DataError;
}

interface IDataProcessor
{
    event EventHandler<string> DataProcessed;
    event EventHandler<string> ProcessingError;
}

interface IDataStorage
{
    event EventHandler<string> DataStored;
    event EventHandler<string> StorageError;
}

// Class implementing multiple interface events
class DataManager : IDataProvider, IDataProcessor, IDataStorage
{
    private string _data;

    // IDataProvider events
    public event EventHandler<string> DataReceived;
    public event EventHandler<string> DataError;

    // IDataProcessor events
    public event EventHandler<string> DataProcessed;
    public event EventHandler<string> ProcessingError;

    // IDataStorage events
    public event EventHandler<string> DataStored;
    public event EventHandler<string> StorageError;

    public void ReceiveData(string data)
    {
        Console.WriteLine($"Receiving data: {data}");
        
        if (string.IsNullOrEmpty(data))
        {
            OnDataError("Data cannot be empty");
            return;
        }

        _data = data;
        OnDataReceived(data);
    }

    public void ProcessData()
    {
        if (string.IsNullOrEmpty(_data))
        {
            OnProcessingError("No data to process");
            return;
        }

        Console.WriteLine($"Processing data: {_data}");
        
        try
        {
            // Simulate processing
            string processedData = _data.ToUpper();
            OnDataProcessed(processedData);
        }
        catch (Exception ex)
        {
            OnProcessingError($"Processing failed: {ex.Message}");
        }
    }

    public void StoreData()
    {
        if (string.IsNullOrEmpty(_data))
        {
            OnStorageError("No data to store");
            return;
        }

        Console.WriteLine($"Storing data: {_data}");
        
        try
        {
            // Simulate storage
            OnDataStored(_data);
        }
        catch (Exception ex)
        {
            OnStorageError($"Storage failed: {ex.Message}");
        }
    }

    // Event raising methods
    protected virtual void OnDataReceived(string data)
    {
        DataReceived?.Invoke(this, data);
    }

    protected virtual void OnDataError(string error)
    {
        DataError?.Invoke(this, error);
    }

    protected virtual void OnDataProcessed(string data)
    {
        DataProcessed?.Invoke(this, data);
    }

    protected virtual void OnProcessingError(string error)
    {
        ProcessingError?.Invoke(this, error);
    }

    protected virtual void OnDataStored(string data)
    {
        DataStored?.Invoke(this, data);
    }

    protected virtual void OnStorageError(string error)
    {
        StorageError?.Invoke(this, error);
    }
}

// Event handlers
class DataEventHandler
{
    public void OnDataReceived(object sender, string data)
    {
        Console.WriteLine($"  [PROVIDER] Data received: {data}");
    }

    public void OnDataError(object sender, string error)
    {
        Console.WriteLine($"  [PROVIDER] Error: {error}");
    }

    public void OnDataProcessed(object sender, string data)
    {
        Console.WriteLine($"  [PROCESSOR] Data processed: {data}");
    }

    public void OnProcessingError(object sender, string error)
    {
        Console.WriteLine($"  [PROCESSOR] Error: {error}");
    }

    public void OnDataStored(object sender, string data)
    {
        Console.WriteLine($"  [STORAGE] Data stored: {data}");
    }

    public void OnStorageError(object sender, string error)
    {
        Console.WriteLine($"  [STORAGE] Error: {error}");
    }
}

class Program
{
    static void Main()
    {
        DataManager manager = new DataManager();
        DataEventHandler handler = new DataEventHandler();

        // Subscribe to events through interface references
        IDataProvider provider = manager;
        IDataProcessor processor = manager;
        IDataStorage storage = manager;

        provider.DataReceived += handler.OnDataReceived;
        provider.DataError += handler.OnDataError;
        processor.DataProcessed += handler.OnDataProcessed;
        processor.ProcessingError += handler.OnProcessingError;
        storage.DataStored += handler.OnDataStored;
        storage.StorageError += handler.OnStorageError;

        Console.WriteLine("=== Data Flow ===");
        
        // Test data flow
        manager.ReceiveData("Hello World");
        manager.ProcessData();
        manager.StoreData();

        Console.WriteLine("\n=== Error Handling ===");
        
        // Test error handling
        manager.ReceiveData(""); // Empty data
        manager.ProcessData();
        manager.StoreData();
    }
}
```

**Output:**
```
=== Data Flow ===
Receiving data: Hello World
  [PROVIDER] Data received: Hello World
Processing data: Hello World
  [PROCESSOR] Data processed: HELLO WORLD
Storing data: Hello World
  [STORAGE] Data stored: Hello World

=== Error Handling ===
Receiving data: 
  [PROVIDER] Error: Data cannot be empty
  [PROCESSOR] Error: No data to process
  [STORAGE] Error: No data to store
```

### Example 5: Interface Event Inheritance

```csharp
using System;

// Base interface with events
interface IBase
{
    event EventHandler<string> BaseEvent;
}

// Derived interface with additional events
interface IDerived : IBase
{
    event EventHandler<string> DerivedEvent;
}

// Another interface
interface IAdditional
{
    event EventHandler<string> AdditionalEvent;
}

// Class implementing multiple interfaces
class MultiInterfaceClass : IDerived, IAdditional
{
    // Implement IBase events
    public event EventHandler<string> BaseEvent;

    // Implement IDerived events
    public event EventHandler<string> DerivedEvent;

    // Implement IAdditional events
    public event EventHandler<string> AdditionalEvent;

    public void TriggerBaseEvent(string message)
    {
        Console.WriteLine($"Triggering base event: {message}");
        OnBaseEvent(message);
    }

    public void TriggerDerivedEvent(string message)
    {
        Console.WriteLine($"Triggering derived event: {message}");
        OnDerivedEvent(message);
    }

    public void TriggerAdditionalEvent(string message)
    {
        Console.WriteLine($"Triggering additional event: {message}");
        OnAdditionalEvent(message);
    }

    protected virtual void OnBaseEvent(string message)
    {
        BaseEvent?.Invoke(this, message);
    }

    protected virtual void OnDerivedEvent(string message)
    {
        DerivedEvent?.Invoke(this, message);
    }

    protected virtual void OnAdditionalEvent(string message)
    {
        AdditionalEvent?.Invoke(this, message);
    }
}

// Event handler
class InterfaceEventHandler
{
    public void OnBaseEvent(object sender, string message)
    {
        Console.WriteLine($"  [BASE] {message}");
    }

    public void OnDerivedEvent(object sender, string message)
    {
        Console.WriteLine($"  [DERIVED] {message}");
    }

    public void OnAdditionalEvent(object sender, string message)
    {
        Console.WriteLine($"  [ADDITIONAL] {message}");
    }
}

class Program
{
    static void Main()
    {
        MultiInterfaceClass obj = new MultiInterfaceClass();
        InterfaceEventHandler handler = new InterfaceEventHandler();

        // Subscribe through different interface references
        IBase baseInterface = obj;
        IDerived derivedInterface = obj;
        IAdditional additionalInterface = obj;

        baseInterface.BaseEvent += handler.OnBaseEvent;
        derivedInterface.DerivedEvent += handler.OnDerivedEvent;
        additionalInterface.AdditionalEvent += handler.OnAdditionalEvent;

        Console.WriteLine("=== Interface Event Inheritance ===");
        
        // Trigger events
        obj.TriggerBaseEvent("Base message");
        obj.TriggerDerivedEvent("Derived message");
        obj.TriggerAdditionalEvent("Additional message");

        Console.WriteLine("\n=== Interface Polymorphism ===");
        
        // Test interface polymorphism
        IBase baseRef = obj;
        IDerived derivedRef = obj;
        IAdditional additionalRef = obj;

        baseRef.BaseEvent += handler.OnBaseEvent;
        derivedRef.DerivedEvent += handler.OnDerivedEvent;
        additionalRef.AdditionalEvent += handler.OnAdditionalEvent;

        obj.TriggerBaseEvent("Polymorphic base message");
        obj.TriggerDerivedEvent("Polymorphic derived message");
        obj.TriggerAdditionalEvent("Polymorphic additional message");
    }
}
```

**Output:**
```
=== Interface Event Inheritance ===
Triggering base event: Base message
  [BASE] Base message
Triggering derived event: Derived message
  [DERIVED] Derived message
Triggering additional event: Additional message
  [ADDITIONAL] Additional message

=== Interface Polymorphism ===
Triggering base event: Polymorphic base message
  [BASE] Polymorphic base message
  [BASE] Polymorphic base message
Triggering derived event: Polymorphic derived message
  [DERIVED] Polymorphic derived message
  [DERIVED] Polymorphic derived message
Triggering additional event: Polymorphic additional message
  [ADDITIONAL] Polymorphic additional message
  [ADDITIONAL] Polymorphic additional message
```

## Key Points

### Interface Event Declaration
- **Event Syntax**: `event EventHandler<T> EventName;`
- **Access Modifiers**: Events in interfaces are implicitly public
- **Generic Events**: Use generic event handlers for type safety

### Event Implementation
- **Implicit Implementation**: Direct implementation of interface events
- **Explicit Implementation**: Use explicit interface implementation for complex scenarios
- **Event Fields**: Use private event fields for explicit implementation

### Event Polymorphism
- **Interface References**: Subscribe to events through interface references
- **Multiple Interfaces**: Implement events from multiple interfaces
- **Interface Inheritance**: Events are inherited through interface inheritance

### Best Practices
- **Event Contracts**: Define clear event contracts in interfaces
- **Explicit Implementation**: Use explicit implementation for complex event scenarios
- **Thread Safety**: Implement thread-safe event subscription/unsubscription
- **Documentation**: Document event behavior in interfaces
- **Testing**: Test event behavior through interface references
