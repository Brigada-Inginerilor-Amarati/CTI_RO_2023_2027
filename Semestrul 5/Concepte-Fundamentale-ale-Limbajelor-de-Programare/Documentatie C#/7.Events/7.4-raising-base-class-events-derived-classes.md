# 7.4 Raising Base Class Events in Derived Classes

## Description

When working with inheritance in C#, derived classes often need to raise events that are declared in their base classes. This section covers the proper patterns for raising base class events from derived classes while maintaining encapsulation and following best practices.

### Key Concepts:
- **Protected Events**: Events declared as protected in base classes
- **Virtual Methods**: Overrideable methods for raising events
- **Event Inheritance**: How events work with inheritance
- **Encapsulation**: Maintaining proper access control
- **Event Raising Patterns**: Best practices for derived classes

### Common Patterns:
- **Protected Virtual Methods**: Override event raising methods in derived classes
- **Base Class Events**: Raise base class events from derived classes
- **Event Chaining**: Chain events from derived to base classes
- **Access Control**: Proper access modifiers for inherited events

## Code Examples

### Example 1: Basic Event Inheritance Pattern

```csharp
using System;

// Base class with events
class Animal
{
    // Protected events that derived classes can raise
    protected event EventHandler<string> AnimalMoved;
    protected event EventHandler<string> AnimalAte;
    protected event EventHandler<string> AnimalSlept;

    // Virtual methods for raising events
    protected virtual void OnAnimalMoved(string movement)
    {
        AnimalMoved?.Invoke(this, movement);
    }

    protected virtual void OnAnimalAte(string food)
    {
        AnimalAte?.Invoke(this, food);
    }

    protected virtual void OnAnimalSlept(string location)
    {
        AnimalSlept?.Invoke(this, location);
    }

    // Public methods that derived classes can override
    public virtual void Move(string direction)
    {
        Console.WriteLine($"Animal moves {direction}");
        OnAnimalMoved(direction);
    }

    public virtual void Eat(string food)
    {
        Console.WriteLine($"Animal eats {food}");
        OnAnimalAte(food);
    }

    public virtual void Sleep(string location)
    {
        Console.WriteLine($"Animal sleeps at {location}");
        OnAnimalSlept(location);
    }
}

// Derived class that raises base class events
class Dog : Animal
{
    private string _name;

    public Dog(string name)
    {
        _name = name;
    }

    // Override methods to add specific behavior
    public override void Move(string direction)
    {
        Console.WriteLine($"{_name} the dog runs {direction}");
        OnAnimalMoved($"running {direction}"); // Raise base class event
    }

    public override void Eat(string food)
    {
        Console.WriteLine($"{_name} the dog devours {food}");
        OnAnimalAte(food); // Raise base class event
    }

    public override void Sleep(string location)
    {
        Console.WriteLine($"{_name} the dog sleeps at {location}");
        OnAnimalSlept(location); // Raise base class event
    }

    // Additional methods that raise base class events
    public void Bark()
    {
        Console.WriteLine($"{_name} barks loudly");
        OnAnimalMoved("barking"); // Raise base class event
    }
}

// Event handler
class AnimalObserver
{
    public void OnAnimalMoved(object sender, string movement)
    {
        Console.WriteLine($"  [OBSERVER] Animal moved: {movement}");
    }

    public void OnAnimalAte(object sender, string food)
    {
        Console.WriteLine($"  [OBSERVER] Animal ate: {food}");
    }

    public void OnAnimalSlept(object sender, string location)
    {
        Console.WriteLine($"  [OBSERVER] Animal slept at: {location}");
    }
}

class Program
{
    static void Main()
    {
        Dog dog = new Dog("Buddy");
        AnimalObserver observer = new AnimalObserver();

        // Subscribe to events (inherited from base class)
        dog.AnimalMoved += observer.OnAnimalMoved;
        dog.AnimalAte += observer.OnAnimalAte;
        dog.AnimalSlept += observer.OnAnimalSlept;

        Console.WriteLine("=== Dog Behavior ===");
        
        // Call methods that raise base class events
        dog.Move("forward");
        dog.Eat("bone");
        dog.Sleep("dog bed");
        dog.Bark();
    }
}
```

**Output:**
```
=== Dog Behavior ===
Buddy the dog runs forward
  [OBSERVER] Animal moved: running forward
Buddy the dog devours bone
  [OBSERVER] Animal ate: bone
Buddy the dog sleeps at dog bed
  [OBSERVER] Animal slept at: dog bed
Buddy barks loudly
  [OBSERVER] Animal moved: barking
```

### Example 2: Event Chaining from Derived to Base

```csharp
using System;

// Base class with events
class Vehicle
{
    protected event EventHandler<string> EngineStarted;
    protected event EventHandler<string> EngineStopped;
    protected event EventHandler<string> VehicleMoved;

    protected virtual void OnEngineStarted(string engineType)
    {
        EngineStarted?.Invoke(this, engineType);
    }

    protected virtual void OnEngineStopped(string engineType)
    {
        EngineStopped?.Invoke(this, engineType);
    }

    protected virtual void OnVehicleMoved(string movement)
    {
        VehicleMoved?.Invoke(this, movement);
    }

    public virtual void StartEngine()
    {
        Console.WriteLine("Vehicle engine starting...");
        OnEngineStarted("generic");
    }

    public virtual void StopEngine()
    {
        Console.WriteLine("Vehicle engine stopping...");
        OnEngineStopped("generic");
    }

    public virtual void Move(string direction)
    {
        Console.WriteLine($"Vehicle moving {direction}");
        OnVehicleMoved(direction);
    }
}

// Derived class with additional events
class Car : Vehicle
{
    private string _model;
    private bool _engineRunning;

    public Car(string model)
    {
        _model = model;
    }

    // Additional events specific to Car
    public event EventHandler<string> DoorOpened;
    public event EventHandler<string> DoorClosed;

    protected virtual void OnDoorOpened(string door)
    {
        DoorOpened?.Invoke(this, door);
    }

    protected virtual void OnDoorClosed(string door)
    {
        DoorClosed?.Invoke(this, door);
    }

    // Override base class methods
    public override void StartEngine()
    {
        Console.WriteLine($"{_model} car engine starting...");
        _engineRunning = true;
        OnEngineStarted("gasoline"); // Raise base class event
    }

    public override void StopEngine()
    {
        Console.WriteLine($"{_model} car engine stopping...");
        _engineRunning = false;
        OnEngineStopped("gasoline"); // Raise base class event
    }

    public override void Move(string direction)
    {
        if (!_engineRunning)
        {
            Console.WriteLine("Cannot move - engine not running");
            return;
        }

        Console.WriteLine($"{_model} car moving {direction}");
        OnVehicleMoved(direction); // Raise base class event
    }

    // Additional methods that raise both base and derived events
    public void OpenDoor(string door)
    {
        Console.WriteLine($"{_model} car {door} door opened");
        OnDoorOpened(door); // Raise derived class event
        OnVehicleMoved($"opening {door} door"); // Raise base class event
    }

    public void CloseDoor(string door)
    {
        Console.WriteLine($"{_model} car {door} door closed");
        OnDoorClosed(door); // Raise derived class event
        OnVehicleMoved($"closing {door} door"); // Raise base class event
    }
}

// Event handlers
class VehicleMonitor
{
    public void OnEngineStarted(object sender, string engineType)
    {
        Console.WriteLine($"  [MONITOR] Engine started: {engineType}");
    }

    public void OnEngineStopped(object sender, string engineType)
    {
        Console.WriteLine($"  [MONITOR] Engine stopped: {engineType}");
    }

    public void OnVehicleMoved(object sender, string movement)
    {
        Console.WriteLine($"  [MONITOR] Vehicle moved: {movement}");
    }

    public void OnDoorOpened(object sender, string door)
    {
        Console.WriteLine($"  [MONITOR] Door opened: {door}");
    }

    public void OnDoorClosed(object sender, string door)
    {
        Console.WriteLine($"  [MONITOR] Door closed: {door}");
    }
}

class Program
{
    static void Main()
    {
        Car car = new Car("Toyota Camry");
        VehicleMonitor monitor = new VehicleMonitor();

        // Subscribe to both base and derived events
        car.EngineStarted += monitor.OnEngineStarted;
        car.EngineStopped += monitor.OnEngineStopped;
        car.VehicleMoved += monitor.OnVehicleMoved;
        car.DoorOpened += monitor.OnDoorOpened;
        car.DoorClosed += monitor.OnDoorClosed;

        Console.WriteLine("=== Car Operations ===");
        
        // Test base class events
        car.StartEngine();
        car.Move("forward");
        car.StopEngine();

        Console.WriteLine("\n=== Door Operations ===");
        
        // Test derived class events
        car.OpenDoor("driver's");
        car.CloseDoor("driver's");
        car.OpenDoor("passenger's");
        car.CloseDoor("passenger's");
    }
}
```

**Output:**
```

  [MONITOR] Vehicle moved: opening driver's door
```

### Example 3: Abstract Base Class with Events

```csharp
using System;

// Abstract base class with events
abstract class Shape
{
    protected event EventHandler<string> ShapeCreated;
    protected event EventHandler<double> AreaCalculated;
    protected event EventHandler<string> ShapeMoved;

    protected virtual void OnShapeCreated(string shapeType)
    {
        ShapeCreated?.Invoke(this, shapeType);
    }

    protected virtual void OnAreaCalculated(double area)
    {
        AreaCalculated?.Invoke(this, area);
    }

    protected virtual void OnShapeMoved(string movement)
    {
        ShapeMoved?.Invoke(this, movement);
    }

    // Abstract methods that derived classes must implement
    public abstract double CalculateArea();
    public abstract void Move(string direction);

    // Virtual method that can be overridden
    public virtual void DisplayInfo()
    {
        Console.WriteLine("Shape information displayed");
    }
}

// Concrete derived class
class Circle : Shape
{
    private double _radius;
    private double _x, _y;

    public Circle(double radius, double x = 0, double y = 0)
    {
        _radius = radius;
        _x = x;
        _y = y;
        OnShapeCreated("Circle"); // Raise base class event
    }

    public override double CalculateArea()
    {
        double area = Math.PI * _radius * _radius;
        OnAreaCalculated(area); // Raise base class event
        return area;
    }

    public override void Move(string direction)
    {
        Console.WriteLine($"Circle moving {direction}");
        OnShapeMoved(direction); // Raise base class event
        
        // Update position (simplified)
        switch (direction.ToLower())
        {
            case "up":
                _y += 1;
                break;
            case "down":
                _y -= 1;
                break;
            case "left":
                _x -= 1;
                break;
            case "right":
                _x += 1;
                break;
        }
    }

    public override void DisplayInfo()
    {
        Console.WriteLine($"Circle: radius={_radius}, position=({_x}, {_y})");
        base.DisplayInfo(); // Call base class method
    }
}

// Another concrete derived class
class Rectangle : Shape
{
    private double _width, _height;
    private double _x, _y;

    public Rectangle(double width, double height, double x = 0, double y = 0)
    {
        _width = width;
        _height = height;
        _x = x;
        _y = y;
        OnShapeCreated("Rectangle"); // Raise base class event
    }

    public override double CalculateArea()
    {
        double area = _width * _height;
        OnAreaCalculated(area); // Raise base class event
        return area;
    }

    public override void Move(string direction)
    {
        Console.WriteLine($"Rectangle moving {direction}");
        OnShapeMoved(direction); // Raise base class event
        
        // Update position (simplified)
        switch (direction.ToLower())
        {
            case "up":
                _y += 1;
                break;
            case "down":
                _y -= 1;
                break;
            case "left":
                _x -= 1;
                break;
            case "right":
                _x += 1;
                break;
        }
    }

    public override void DisplayInfo()
    {
        Console.WriteLine($"Rectangle: width={_width}, height={_height}, position=({_x}, {_y})");
        base.DisplayInfo(); // Call base class method
    }
}

// Event handler
class ShapeObserver
{
    public void OnShapeCreated(object sender, string shapeType)
    {
        Console.WriteLine($"  [OBSERVER] Shape created: {shapeType}");
    }

    public void OnAreaCalculated(object sender, double area)
    {
        Console.WriteLine($"  [OBSERVER] Area calculated: {area:F2}");
    }

    public void OnShapeMoved(object sender, string movement)
    {
        Console.WriteLine($"  [OBSERVER] Shape moved: {movement}");
    }
}

class Program
{
    static void Main()
    {
        ShapeObserver observer = new ShapeObserver();

        Console.WriteLine("=== Circle Operations ===");
        
        // Create circle and subscribe to events
        Circle circle = new Circle(5.0, 0, 0);
        circle.ShapeCreated += observer.OnShapeCreated;
        circle.AreaCalculated += observer.OnAreaCalculated;
        circle.ShapeMoved += observer.OnShapeMoved;

        circle.DisplayInfo();
        double circleArea = circle.CalculateArea();

        Console.WriteLine("\n=== Rectangle Operations ===");
        
        // Create rectangle and subscribe to events

        rectangle.ShapeCreated += observer.OnShapeCreated;
        rectangle.AreaCalculated += observer.OnAreaCalculated;
        rectangle.ShapeMoved += observer.OnShapeMoved;

        rectangle.DisplayInfo();
        double rectangleArea = rectangle.CalculateArea();
        rectangle.Move("left");
        rectangle.Move("down");
    }
}
```

### Example 4: Event Overriding and Hiding

```csharp
using System;

// Base class with events
class BaseClass
{
    protected event EventHandler<string> BaseEvent;
    protected event EventHandler<string> VirtualEvent;

    protected virtual void OnBaseEvent(string message)
    {
        BaseEvent?.Invoke(this, message);
    }

    protected virtual void OnVirtualEvent(string message)
    {
        VirtualEvent?.Invoke(this, message);
    }

    public virtual void TriggerEvents()
    {
        Console.WriteLine("Base class triggering events");
        OnBaseEvent("Base event message");
        OnVirtualEvent("Virtual event message");
    }
}

// Derived class that overrides event raising
class DerivedClass : BaseClass
{
    // Override the virtual method for raising events
    protected override void OnBaseEvent(string message)
    {
        Console.WriteLine($"Derived class handling base event: {message}");
        base.OnBaseEvent(message); // Call base class method
    }

    protected override void OnVirtualEvent(string message)
    {
        Console.WriteLine($"Derived class handling virtual event: {message}");
        base.OnVirtualEvent(message); // Call base class method
    }

    // Override the public method
    public override void TriggerEvents()
    {
        Console.WriteLine("Derived class triggering events");
        base.TriggerEvents(); // Call base class method
    }

    // Additional method that raises base class events
    public void TriggerDerivedEvents()
    {
        Console.WriteLine("Derived class triggering its own events");
        OnBaseEvent("Derived event message");
        OnVirtualEvent("Derived virtual event message");
    }
}

// Event handler
class EventHandler
{
    public void OnBaseEvent(object sender, string message)
    {
        Console.WriteLine($"  [HANDLER] Base event received: {message}");
    }

    public void OnVirtualEvent(object sender, string message)
    {
        Console.WriteLine($"  [HANDLER] Virtual event received: {message}");
    }
}

class Program
{
    static void Main()
    {
        EventHandler handler = new EventHandler();

        Console.WriteLine("=== Base Class Events ===");
        
        BaseClass baseObj = new BaseClass();
        baseObj.BaseEvent += handler.OnBaseEvent;
        baseObj.VirtualEvent += handler.OnVirtualEvent;
        baseObj.TriggerEvents();

        Console.WriteLine("\n=== Derived Class Events ===");
        
        DerivedClass derivedObj = new DerivedClass();
        derivedObj.BaseEvent += handler.OnBaseEvent;
        derivedObj.VirtualEvent += handler.OnVirtualEvent;
        derivedObj.TriggerEvents();

        Console.WriteLine("\n=== Derived Class Own Events ===");
        
        derivedObj.TriggerDerivedEvents();
    }
}
```

**Output:**
```

  [HANDLER] Base event received: Base event message
  [HANDLER] Virtual event received: Virtual event message

=== Derived Class Events ===

Base class triggering events
Derived class handling base event: Base event message
  [HANDLER] Base event received: Base event message
Derived class handling virtual event: Virtual event message
  [HANDLER] Virtual event received: Virtual event message
```

### Example 5: Event Access Control in Inheritance

```csharp
using System;

// Base class with different access levels for events
class BaseClass
{
    // Public event - accessible from anywhere
    public event EventHandler<string> PublicEvent;

    // Protected event - accessible from derived classes
    protected event EventHandler<string> ProtectedEvent;

    // Internal event - accessible from same assembly
    internal event EventHandler<string> InternalEvent;

    // Private event - accessible only from this class
    private event EventHandler<string> PrivateEvent;

    // Methods to raise events
    public void RaisePublicEvent(string message)
    {
        Console.WriteLine($"Raising public event: {message}");
        PublicEvent?.Invoke(this, message);
    }

    protected void RaiseProtectedEvent(string message)
    {
        Console.WriteLine($"Raising protected event: {message}");
        ProtectedEvent?.Invoke(this, message);
    }

    internal void RaiseInternalEvent(string message)
    {
        Console.WriteLine($"Raising internal event: {message}");
        InternalEvent?.Invoke(this, message);
    }

    private void RaisePrivateEvent(string message)
    {
        Console.WriteLine($"Raising private event: {message}");
        PrivateEvent?.Invoke(this, message);
    }

    // Public method that raises all events
    public void RaiseAllEvents(string message)
    {
        RaisePublicEvent(message);
        RaiseProtectedEvent(message);
        RaiseInternalEvent(message);
        RaisePrivateEvent(message);
    }
}

// Derived class
class DerivedClass : BaseClass
{
    // Additional events
    public event EventHandler<string> DerivedEvent;

    // Method that raises both base and derived events
    public void RaiseDerivedEvents(string message)
    {
        Console.WriteLine($"Raising derived events: {message}");
        
        // Can raise public and protected events from base class
        RaisePublicEvent(message);
        RaiseProtectedEvent(message);
        
        // Can raise internal events if in same assembly
        RaiseInternalEvent(message);
        
        // Cannot raise private events from base class
        // RaisePrivateEvent(message); // This would cause compilation error
        
        // Raise derived class event
        DerivedEvent?.Invoke(this, message);
    }

    // Override to add additional behavior
    public new void RaiseAllEvents(string message)
    {
        Console.WriteLine("Derived class raising all events");
        base.RaiseAllEvents(message);
        DerivedEvent?.Invoke(this, message);
    }
}

// Event handler
class EventHandler
{
    public void OnPublicEvent(object sender, string message)
    {
        Console.WriteLine($"  [HANDLER] Public event: {message}");
    }

    public void OnProtectedEvent(object sender, string message)
    {
        Console.WriteLine($"  [HANDLER] Protected event: {message}");
    }

    public void OnInternalEvent(object sender, string message)
    {
        Console.WriteLine($"  [HANDLER] Internal event: {message}");
    }

    public void OnDerivedEvent(object sender, string message)
    {
        Console.WriteLine($"  [HANDLER] Derived event: {message}");
    }
}

class Program
{
    static void Main()
    {
        EventHandler handler = new EventHandler();

        Console.WriteLine("=== Base Class Event Access ===");
        
        BaseClass baseObj = new BaseClass();
        baseObj.PublicEvent += handler.OnPublicEvent;
        baseObj.RaiseAllEvents("Base message");

        Console.WriteLine("\n=== Derived Class Event Access ===");
        
        DerivedClass derivedObj = new DerivedClass();
        derivedObj.PublicEvent += handler.OnPublicEvent;
        derivedObj.DerivedEvent += handler.OnDerivedEvent;
        derivedObj.RaiseDerivedEvents("Derived message");

        Console.WriteLine("\n=== Derived Class All Events ===");
        
        derivedObj.RaiseAllEvents("All events message");

        Console.WriteLine("\n=== Event Access Summary ===");
        Console.WriteLine("Public events: Accessible from anywhere");
        Console.WriteLine("Protected events: Accessible from derived classes");
        Console.WriteLine("Internal events: Accessible from same assembly");
        Console.WriteLine("Private events: Accessible only from declaring class");
    }
}
```

## Key Points

### Event Inheritance Patterns
- **Protected Events**: Use protected events for derived classes to raise
- **Virtual Methods**: Override virtual methods for custom event raising behavior
- **Base Class Calls**: Use `base.MethodName()` to call base class implementations
- **Event Chaining**: Chain events from derived to base classes

### Access Control
- **Public Events**: Accessible from anywhere
- **Protected Events**: Accessible from derived classes
- **Internal Events**: Accessible from same assembly
- **Private Events**: Accessible only from declaring class

### Best Practices
- **Encapsulation**: Maintain proper access control for events
- **Virtual Methods**: Use virtual methods for event raising
- **Base Class Calls**: Always call base class methods when appropriate
- **Event Naming**: Use consistent naming conventions
- **Documentation**: Document event behavior in derived classes
