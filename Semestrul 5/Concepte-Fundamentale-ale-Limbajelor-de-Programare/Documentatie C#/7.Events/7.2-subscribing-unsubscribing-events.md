# 7.2 Subscribing to and Unsubscribing from Events

## Description

Event subscription management is a crucial aspect of working with events in C#. Proper subscription and unsubscription patterns ensure that your application responds correctly to events while avoiding memory leaks and unexpected behavior.

### Key Concepts:
- **Subscription**: Adding event handlers using `+=` operator
- **Unsubscription**: Removing event handlers using `-=` operator
- **Memory Management**: Proper unsubscription prevents memory leaks
- **Lifecycle Management**: Subscribe/unsubscribe at appropriate times
- **Multiple Subscribers**: Events can have multiple handlers

### Common Patterns:
- **Constructor/Destructor**: Subscribe in constructor, unsubscribe in destructor
- **Using Statement**: Automatic cleanup with IDisposable
- **Weak References**: Prevent memory leaks with weak event patterns
- **Conditional Subscription**: Subscribe/unsubscribe based on conditions

## Code Examples

### Example 1: Basic Subscription and Unsubscription

```csharp
using System;

// Event publisher
class Timer
{
    public event EventHandler<TimerEventArgs> Tick;
    public event EventHandler<TimerEventArgs> Elapsed;

    private int _count;
    private int _maxCount;

    public Timer(int maxCount)
    {
        _maxCount = maxCount;
        _count = 0;
    }

    public void Start()
    {
        Console.WriteLine("Timer started...");
        
        for (int i = 0; i < _maxCount; i++)
        {
            _count++;
            OnTick();
            
            if (_count >= _maxCount)
            {
                OnElapsed();
            }
        }
    }

    protected virtual void OnTick()
    {
        Tick?.Invoke(this, new TimerEventArgs(_count));
    }

    protected virtual void OnElapsed()
    {
        Elapsed?.Invoke(this, new TimerEventArgs(_count));
    }
}

// Event arguments
class TimerEventArgs : EventArgs
{
    public int Count { get; }

    public TimerEventArgs(int count)
    {
        Count = count;
    }
}

// Event subscriber
class TimerSubscriber
{
    private string _name;

    public TimerSubscriber(string name)
    {
        _name = name;
    }

    public void OnTick(object sender, TimerEventArgs e)
    {
        Console.WriteLine($"  [{_name}] Tick: {e.Count}");
    }

    public void OnElapsed(object sender, TimerEventArgs e)
    {
        Console.WriteLine($"  [{_name}] Timer elapsed at count: {e.Count}");
    }
}

class Program
{
    static void Main()
    {
        Timer timer = new Timer(5);
        TimerSubscriber subscriber1 = new TimerSubscriber("Subscriber1");
        TimerSubscriber subscriber2 = new TimerSubscriber("Subscriber2");

        Console.WriteLine("=== Basic Subscription ===");
        
        // Subscribe to events
        timer.Tick += subscriber1.OnTick;
        timer.Elapsed += subscriber1.OnElapsed;
        
        timer.Start();

        Console.WriteLine("\n=== Multiple Subscribers ===");
        
        // Reset timer
        timer = new Timer(3);
        
        // Subscribe multiple handlers
        timer.Tick += subscriber1.OnTick;
        timer.Tick += subscriber2.OnTick;
        timer.Elapsed += subscriber1.OnElapsed;
        timer.Elapsed += subscriber2.OnElapsed;
        
        timer.Start();

        Console.WriteLine("\n=== After Unsubscription ===");
        
        // Reset timer
        timer = new Timer(3);
        
        // Subscribe
        timer.Tick += subscriber1.OnTick;
        timer.Tick += subscriber2.OnTick;
        
        // Unsubscribe one handler
        timer.Tick -= subscriber1.OnTick;
        
        timer.Start();
    }
}
```

### Example 2: Constructor/Destructor Pattern

```csharp
using System;

// Event publisher
class DataProcessor
{
    public event EventHandler<string> DataProcessed;
    public event EventHandler<string> ErrorOccurred;

    public void ProcessData(string data)
    {
        try
        {
            Console.WriteLine($"Processing: {data}");
            DataProcessed?.Invoke(this, $"Processed: {data}");
        }
        catch (Exception ex)
        {
            ErrorOccurred?.Invoke(this, $"Error processing {data}: {ex.Message}");
        }
    }
}

// Event subscriber with proper lifecycle management
class DataLogger : IDisposable
{
    private DataProcessor _processor;
    private string _logName;

    public DataLogger(DataProcessor processor, string logName)
    {
        _processor = processor;
        _logName = logName;
        
        // Subscribe in constructor
        _processor.DataProcessed += OnDataProcessed;
        _processor.ErrorOccurred += OnErrorOccurred;
        
        Console.WriteLine($"DataLogger '{_logName}' subscribed to events");
    }

    private void OnDataProcessed(object sender, string message)
    {
        Console.WriteLine($"  [{_logName}] LOG: {message}");
    }

    private void OnErrorOccurred(object sender, string message)
    {
        Console.WriteLine($"  [{_logName}] ERROR: {message}");
    }

    // Unsubscribe in Dispose method
    public void Dispose()
    {
        if (_processor != null)
        {
            _processor.DataProcessed -= OnDataProcessed;
            _processor.ErrorOccurred -= OnErrorOccurred;
            _processor = null;
            
            Console.WriteLine($"DataLogger '{_logName}' unsubscribed from events");
        }
    }
}

class Program
{
    static void Main()
    {
        DataProcessor processor = new DataProcessor();

        Console.WriteLine("=== Using Statement Pattern ===");
        
        // Using statement ensures proper disposal
        using (var logger1 = new DataLogger(processor, "Logger1"))
        using (var logger2 = new DataLogger(processor, "Logger2"))
        {
            processor.ProcessData("Data1");
            processor.ProcessData("Data2");
        }

        Console.WriteLine("\n=== After Disposal ===");
        
        // These events won't be handled
        processor.ProcessData("Data3");
    }
}
```

**Output:**
```

  [Logger1] LOG: Processed: Data1
  [Logger2] LOG: Processed: Data1
Processing: Data2
  [Logger1] LOG: Processed: Data2
  [Logger2] LOG: Processed: Data2

DataLogger 'Logger1' unsubscribed from events

=== After Disposal ===
Processing: Data3
```

### Example 3: Conditional Subscription

```csharp
using System;

// Event publisher
class FileWatcher
{
    public event EventHandler<string> FileCreated;
    public event EventHandler<string> FileModified;
    public event EventHandler<string> FileDeleted;

    public void SimulateFileCreated(string fileName)
    {
        Console.WriteLine($"File created: {fileName}");
        FileCreated?.Invoke(this, fileName);
    }

    public void SimulateFileModified(string fileName)
    {
        Console.WriteLine($"File modified: {fileName}");
        FileModified?.Invoke(this, fileName);
    }

    public void SimulateFileDeleted(string fileName)
    {
        Console.WriteLine($"File deleted: {fileName}");
        FileDeleted?.Invoke(this, fileName);
    }
}

// Conditional event subscriber
class ConditionalFileHandler
{
    private FileWatcher _watcher;
    private bool _isEnabled;
    private string[] _allowedExtensions;

    public bool IsEnabled
    {
        get => _isEnabled;
        set
        {
            _isEnabled = value;
            UpdateSubscriptions();
        }
    }

    public string[] AllowedExtensions
    {
        get => _allowedExtensions;
        set
        {
            _allowedExtensions = value;
            UpdateSubscriptions();
        }
    }

    public ConditionalFileHandler(FileWatcher watcher)
    {
        _watcher = watcher;
        _isEnabled = false;
        _allowedExtensions = new string[0];
    }

    private void UpdateSubscriptions()
    {
        // Unsubscribe from all events first
        _watcher.FileCreated -= OnFileCreated;
        _watcher.FileModified -= OnFileModified;
        _watcher.FileDeleted -= OnFileDeleted;

        // Subscribe only if enabled
        if (_isEnabled)
        {
            _watcher.FileCreated += OnFileCreated;
            _watcher.FileModified += OnFileModified;
            _watcher.FileDeleted += OnFileDeleted;
            Console.WriteLine("File handler enabled and subscribed");
        }
        else
        {
            Console.WriteLine("File handler disabled and unsubscribed");
        }
    }

    private void OnFileCreated(object sender, string fileName)
    {
        if (IsFileAllowed(fileName))
        {
            Console.WriteLine($"  [HANDLER] Created: {fileName}");
        }
    }

    private void OnFileModified(object sender, string fileName)
    {
        if (IsFileAllowed(fileName))
        {
            Console.WriteLine($"  [HANDLER] Modified: {fileName}");
        }
    }

    private void OnFileDeleted(object sender, string fileName)
    {
        if (IsFileAllowed(fileName))
        {
            Console.WriteLine($"  [HANDLER] Deleted: {fileName}");
        }
    }

    private bool IsFileAllowed(string fileName)
    {
        if (_allowedExtensions.Length == 0) return true;

        foreach (string ext in _allowedExtensions)
        {
            if (fileName.EndsWith(ext, StringComparison.OrdinalIgnoreCase))
            {
                return true;
            }
        }
        return false;
    }
}

class Program
{
    static void Main()
    {
        FileWatcher watcher = new FileWatcher();
        ConditionalFileHandler handler = new ConditionalFileHandler(watcher);

        Console.WriteLine("=== Handler Disabled ===");
        watcher.SimulateFileCreated("document.txt");
        watcher.SimulateFileModified("image.jpg");
        watcher.SimulateFileDeleted("data.csv");

        Console.WriteLine("\n=== Handler Enabled (All Files) ===");
        handler.IsEnabled = true;
        watcher.SimulateFileCreated("document.txt");
        watcher.SimulateFileModified("image.jpg");
        watcher.SimulateFileDeleted("data.csv");

        Console.WriteLine("\n=== Handler Enabled (Only .txt files) ===");
        handler.AllowedExtensions = new[] { ".txt" };
        watcher.SimulateFileCreated("document.txt");
        watcher.SimulateFileModified("image.jpg");
        watcher.SimulateFileDeleted("data.csv");

        Console.WriteLine("\n=== Handler Disabled Again ===");
        handler.IsEnabled = false;
        watcher.SimulateFileCreated("document.txt");
    }
}
```

**Output:**

```
=== Handler Enabled (All Files) ===

  [HANDLER] Created: document.txt

  [HANDLER] Modified: image.jpg
```

### Example 4: Weak Event Pattern

```csharp
using System;
using System.Runtime.CompilerServices;
using System.Collections.Generic;

// Weak event pattern implementation
class WeakEventManager
{
    private static readonly Dictionary<string, List<WeakReference>> _eventHandlers = 
        new Dictionary<string, List<WeakReference>>();

    public static void AddHandler<T>(string eventName, EventHandler<T> handler) where T : EventArgs
    {
        if (!_eventHandlers.ContainsKey(eventName))
        {
            _eventHandlers[eventName] = new List<WeakReference>();
        }

        _eventHandlers[eventName].Add(new WeakReference(handler));
    }

    public static void RemoveHandler<T>(string eventName, EventHandler<T> handler) where T : EventArgs
    {
        if (_eventHandlers.ContainsKey(eventName))
        {
            var handlers = _eventHandlers[eventName];
            for (int i = handlers.Count - 1; i >= 0; i--)
            {
                if (handlers[i].Target == handler || !handlers[i].IsAlive)
                {
                    handlers.RemoveAt(i);
                }
            }
        }
    }

    public static void RaiseEvent<T>(string eventName, object sender, T args) where T : EventArgs
    {
        if (_eventHandlers.ContainsKey(eventName))
        {
            var handlers = _eventHandlers[eventName];
            for (int i = handlers.Count - 1; i >= 0; i--)
            {
                if (handlers[i].IsAlive)
                {
                    var handler = handlers[i].Target as EventHandler<T>;
                    handler?.Invoke(sender, args);
                }
                else
                {
                    handlers.RemoveAt(i);
                }
            }
        }
    }
}

// Event publisher using weak events
class WeakEventPublisher
{
    public void RaiseCustomEvent(string message)
    {
        Console.WriteLine($"Raising event: {message}");
        WeakEventManager.RaiseEvent("CustomEvent", this, new CustomEventArgs(message));
    }
}

// Event arguments
class CustomEventArgs : EventArgs
{
    public string Message { get; }

    public CustomEventArgs(string message)
    {
        Message = message;
    }
}

// Event subscriber
class WeakEventSubscriber
{
    private string _name;

    public WeakEventSubscriber(string name)
    {
        _name = name;
    }

    public void OnCustomEvent(object sender, CustomEventArgs e)
    {
        Console.WriteLine($"  [{_name}] Received: {e.Message}");
    }

    public void Subscribe(WeakEventPublisher publisher)
    {
        WeakEventManager.AddHandler("CustomEvent", OnCustomEvent);
        Console.WriteLine($"Subscriber '{_name}' subscribed");
    }

    public void Unsubscribe()
    {
        WeakEventManager.RemoveHandler("CustomEvent", OnCustomEvent);
        Console.WriteLine($"Subscriber '{_name}' unsubscribed");
    }
}

class Program
{
    static void Main()
    {
        WeakEventPublisher publisher = new WeakEventPublisher();
        WeakEventSubscriber subscriber1 = new WeakEventSubscriber("Subscriber1");
        WeakEventSubscriber subscriber2 = new WeakEventSubscriber("Subscriber2");

        Console.WriteLine("=== Weak Event Pattern ===");
        
        // Subscribe using weak references
        subscriber1.Subscribe(publisher);
        subscriber2.Subscribe(publisher);

        // Raise events
        publisher.RaiseCustomEvent("Message 1");
        publisher.RaiseCustomEvent("Message 2");

        Console.WriteLine("\n=== After Unsubscription ===");
        
        // Unsubscribe one subscriber
        subscriber1.Unsubscribe();
        
        publisher.RaiseCustomEvent("Message 3");

        Console.WriteLine("\n=== Garbage Collection Test ===");
        
        // Create a local subscriber that will be garbage collected
        {
            var localSubscriber = new WeakEventSubscriber("LocalSubscriber");
            localSubscriber.Subscribe(publisher);
            publisher.RaiseCustomEvent("Message 4");
        }

        // Force garbage collection
        GC.Collect();
        GC.WaitForPendingFinalizers();
        
        publisher.RaiseCustomEvent("Message 5");
    }
}
```

### Example 5: Event Subscription Management

```csharp
using System;
using System.Collections.Generic;

// Event subscription manager
class EventSubscriptionManager
{
    private Dictionary<string, List<Delegate>> _subscriptions = new Dictionary<string, List<Delegate>>();

    public void Subscribe<T>(string eventName, EventHandler<T> handler) where T : EventArgs
    {
        if (!_subscriptions.ContainsKey(eventName))
        {
            _subscriptions[eventName] = new List<Delegate>();
        }

        _subscriptions[eventName].Add(handler);
        Console.WriteLine($"Subscribed to '{eventName}': {handler.Method.Name}");
    }

    public void Unsubscribe<T>(string eventName, EventHandler<T> handler) where T : EventArgs
    {
        if (_subscriptions.ContainsKey(eventName))
        {
            _subscriptions[eventName].Remove(handler);
            Console.WriteLine($"Unsubscribed from '{eventName}': {handler.Method.Name}");
        }
    }

    public void UnsubscribeAll(string eventName)
    {
        if (_subscriptions.ContainsKey(eventName))
        {
            int count = _subscriptions[eventName].Count;
            _subscriptions[eventName].Clear();
            Console.WriteLine($"Unsubscribed all ({count}) handlers from '{eventName}'");
        }
    }

    public void UnsubscribeAll()
    {
        int totalHandlers = 0;
        foreach (var kvp in _subscriptions)
        {
            totalHandlers += kvp.Value.Count;
        }
        
        _subscriptions.Clear();
        Console.WriteLine($"Unsubscribed all ({totalHandlers}) handlers from all events");
    }

    public int GetSubscriptionCount(string eventName)
    {
        return _subscriptions.ContainsKey(eventName) ? _subscriptions[eventName].Count : 0;
    }

    public void ListSubscriptions()
    {
        Console.WriteLine("\nCurrent subscriptions:");
        foreach (var kvp in _subscriptions)
        {
            Console.WriteLine($"  {kvp.Key}: {kvp.Value.Count} handlers");
            foreach (var handler in kvp.Value)
            {
                Console.WriteLine($"    - {handler.Method.Name}");
            }
        }
    }
}
```

## Key Points

### Subscription Patterns
- **Constructor/Destructor**: Subscribe in constructor, unsubscribe in destructor
- **Using Statement**: Automatic cleanup with IDisposable
- **Conditional**: Subscribe/unsubscribe based on conditions
- **Weak References**: Prevent memory leaks with weak event patterns

### Memory Management
- **Always Unsubscribe**: Prevent memory leaks by unsubscribing
- **Weak Events**: Use weak references for automatic cleanup
- **IDisposable**: Implement proper disposal patterns
- **Garbage Collection**: Weak references allow GC to clean up

### Best Practices
- **Null Safety**: Always check for null before subscribing/unsubscribing
- **Lifecycle Management**: Subscribe/unsubscribe at appropriate times
- **Error Handling**: Handle exceptions in event handlers
- **Performance**: Consider performance implications of many subscribers
- **Thread Safety**: Be aware of thread safety in event operations
